# note-crafter + hxc-obsidian-vault 合并

## 变更摘要

**日期**：2026-05-01
**变更**：将 hxc-obsidian-vault 技能合并入 note-crafter，删除独立的 hxc-obsidian-vault SKILL.md。
**触发**：每次创建笔记需加载 2 个 skill（note-crafter + hxc-obsidian-vault），各约 400 行，大量内容重叠。

## 合并策略

note-crafter 为主骨架（6 步创建流程），hxc-obsidian-vault 的规范内容附录化：

| hxc-obsidian-vault 内容 | 合并后位置 |
|--------------------------|-----------|
| Vault 基本信息 | 顶部「Vault 基本信息」表 |
| MOC 归属判断 | Step 2 内联（去重） |
| Frontmatter 模板 | Step 3a（去重） |
| 顶层目录设计 | 「Vault 顶层目录设计」节 |
| 归属判断规则 | Step 2e 速查表（去重） |
| 日记与周期笔记规范 | 「日记与周期笔记规范」节 |
| 标签分类体系 | 「标签分类体系」节 |
| 特殊规则（SVG/附件/MOC/大文件） | 「特殊规则」节 |
| 批量操作安全协议 | 「运维操作」节 |
| 默认禁止操作 | 「运维操作」节 |
| 存量笔记修复 / 审计 / 索引重建 | 「运维操作」节 |
| 关联文件 | 合并入各节引用 |

## 执行策略更新

| 项目 | 合并前 | 合并后 |
|------|--------|--------|

| 目标工具调用 | 5 次 | 4 次 |
