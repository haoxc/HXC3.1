## 内容
在工程、建筑、IT 研发等多专业协作（Cross-disciplinary Collaboration）的场景中，除了“提资”，还有一系列带有“黑话”性质的专业术语。
以下是常见的同类术语及其对应的英文表述：

## 2. 叠图 (Overlay / Clash Detection)
- 含义：把各专业的图纸叠在一起看，找出矛盾点。
- 英文：Overlay (平面叠图) 或 Clash Detection (三维碰撞检查，常见于 BIM)。
- 职场场景：发现空调管道和消防喷淋撞在一起了，需要“叠图”调整。
## 3. 反提 (Reverse Input / Feedback)
- 含义：下游专业根据自己的设计结果，反馈给上游专业，要求其修改原始输入。
- 英文：Design Feedback 或 Reverse Feedback。
- 职场场景：机电组算完负荷后，“反提”给建筑组，要求增加机房面积。
## 4. 甩项 (Left-over items / Descoping)
- 含义：在工程验收或交付时，因为某些原因暂时无法完成，先强行交付，把未完成部分“甩”掉。
- 英文：Carried-over items 或 Out of scope。
- 职场场景：为了赶进度，先把主楼验收了，室外园林由于季节原因先“甩项”。
## 5. 封版 (Freezing / Version Lock)

- 含义：确定某个阶段的设计或代码不再修改，作为下一阶段的底稿。
- 英文：Design Freeze 或 Version Lock。
- 职场场景：图纸今天下午两点“封版”，之后再改要收变更费。
## 6. 变更 / 洽商 (Change Order / Variation)
- 含义：甲方或现场临时要求改动，导致成本或设计变动。
- 英文：Change Order (CO) 或 Variation Order (VO)。
- 职场场景：现场发现土质不对，需要做设计“变更”。

## 7. 见证取样 (Witness Testing / Sampling)
- 含义：在监理或第三方监督下，对材料进行抽检。
- 英文：Witness Sampling 或 Lab Testing。
