## 内容
| 特性   | npm                       | npx                  |
| ---- | ------------------------- | -------------------- |
| 全称   | Node Package Manager      | Node Package Execute |
| 本质   | 包管理器                      | 包执行工具（npm 5.2+内置）    |
| 主要用途 | 安装、管理、更新软件包               | 临时执行包中的命令            |
| 安装位置 | 持久化安装到 node_modules 或全局路径 | 临时安装，执行后自动清理         |
| 典型命令 | `npm install <包名>`        | `npx <包名> <参数>`      |
| 适用场景 | 需要长期使用的依赖包                | 偶尔使用的临时工具（脚手架等）      |
|      |                           |                      |
