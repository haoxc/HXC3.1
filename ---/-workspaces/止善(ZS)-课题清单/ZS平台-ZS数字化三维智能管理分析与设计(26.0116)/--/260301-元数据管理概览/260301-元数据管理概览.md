- 概念

		- 元数据 (MetaData): 元数据是“关于数据的数据”，它描述了信息的结构、约束和逻辑;

### 框架标准
- 元数据管理标准框架
	1. **[[ISO-IEC 11179]]**：国际标准化组织制定的《元数据注册标准》，是构建元数据登记系统的`核心指导`。
	2. **CWM (Common Warehouse Metamodel)**：公共仓库元模型，由 OMG 组织制定，旨在解决不同数据仓库工具之间`元数据交换的问题`。
	3. **DAMA DMBOK2**：国际数据管理协会(DAMA) 将元数据管理定义为 11 个数据管理职能之一，提供了成熟的治理流程和组织架构建议。
	4. **Dublin Core (都柏林核心集)**：一种跨行业的轻量级标准，包含 15 个核心元素（如标题、创作者、日期），常用于互联网资源的发现与描述。

### 主流技术
三、 主流技术实现
在大数据生态中，元数据管理已从简单的“数据字典”演变为自动化的治理平台：
- **Apache Atlas**：Hadoop 生态中最成熟的开源治理平台，支持分类、血缘追踪及与 Hive、Kafka 的深度集成。
- **DataHub**：最初由 LinkedIn 开发，是目前非常流行的下一代元数据管理平台，支持多租户和现代化的搜索体验。
- **云原生服务**：如 AWS Glue Data Catalog、Google Cloud Data Catalog 等，提供托管式的元数据发现与存储服务。
