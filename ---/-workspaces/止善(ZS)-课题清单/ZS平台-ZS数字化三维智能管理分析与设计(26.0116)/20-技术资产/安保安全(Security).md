## 内容
Security (信息/安保安全) —— “防止人伤害事物”
- **本质 (Essence)**：**对抗性的、动态演化的、外部视角的。**
- **核心逻辑**：通过加密、认证、防火墙等手段，保护系统资产免受来自**人类攻击者**的窃取、破坏、篡改或非法访问。
- **关注点**：**威胁与攻击 (Threats / Attacks)**。
- **例子**：数据库的访问控制、防黑客渗透、物理门禁系统、防间谍协议。

### 维度对比 (Tech Style Analysis)

| 维度       | Safety (功能安全)         | Security (安防安全)            |
| -------- | --------------------- | -------------------------- |
| **威胁来源** | **自然/随机** (重力、磨损、疏忽)  | **人为/刻意** (黑客、间谍、窃贼)       |
| **视点**   | **由内向外** (系统不伤人)      | **由外向内** (人不伤系统)           |
| **侧重点**  | **可靠性 (Reliability)** | **完整性 (Integrity)**        |
| **判定逻辑** | **故障树分析** (FTA)       | **威胁建模** (Threat Modeling) |
