# 重启后操作清单

## 重启原因

清除 macOS 26.3 PGLite WASM 缓存状态，解除 gbrain MCP server 启动阻塞。

## 重启后操作

### 1. 验证 WASM bug 是否解除

```bash
cd ~/gbrain && bun run src/cli.ts doctor
```

如果是 "filesystem checks only"，说明仍未解除。

### 2. 如果 WASM 解除，启动 gbrain MCP server

```bash
cd ~/gbrain
OPENAI_API_KEY="sk-litellm-31ad16c241727f7003928ea4b6b7a03f0aa9" \
OPENAI_BASE_URL="http://localhost:4001/v1" \
bun run src/cli.ts serve &
```

### 3. Hermes MCP 自动重连

Hermes 会在检测到 server 可用后自动重连。通过 `mcp_gbrain_get_stats` 验证。

### 4. 测试完整流程

- `mcp_gbrain_put_page` 导入 `CLAUDE.md`
- `mcp_gbrain_query "笔记风格 标题 规则"` 验证搜索

### 5. 记录结果到本目录

所有命令和输出追加到 `测试执行日志.md`。
