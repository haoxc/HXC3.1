## 笔记：AI Skill (技能包)

## 1. 定义与核心
- 定义：一种为 AI Agent（如 Claude）设计的`模块化能力`插件。
- 本质：结构化的 Prompt（提示词）指令集，通常以 `SKILL.md` 文件形式存在。
- 作用：赋予 AI 特定领域的“`专家经验”`和“`操作流程`”。

## 2. 关键对比：Skill vs. MCP

| 特性  | MCP (Model Context Protocol) | Skill (技能包)        |
| --- | ---------------------------- | ------------------ |
| 类比  | 像 AI 的器官/工具（手和眼）             | 像 AI 的大脑/经验（知识和逻辑） |
| 功能  | `连接`外部数据、读取文件、调用 API         | 规定任务流程、专业标准、回复风格   |
| 目标  | 解决“`能不能`”连接的问题               | 解决“`怎么做`”得更好的问题    |

## 3. 核心优势
- 低门槛：使用 Markdown 编写，无需深度编程。
- 省空间：仅在需要时加载，节省 Context Window（上下文窗口）。
- 可复用：一次定义，跨项目、跨团队共享。

## 4. 常见应用
- [[代码审查]]：固化团队的 Code Review 标准。
- [[内容创作]]：设定特定的排版格式、语气和选题逻辑。
- [[自动化流]]：定义从数据提取到报告生成的完整 Slogan。
