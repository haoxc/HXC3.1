## 内容
> 地址: [插件市场(ClawHub)](https://clawhub.ai/)

1. 安装clawhub包(全局)
```shell
	npm i -g clawhub
```
2. 在插件市场搜索,并获取`技能名称`,比如上网(tavily web search)
   ![[e628d0b0855ec9c71490569655a6fa87_MD5.jpg|607]]
```shell
	clawhub install  "tavily-search"
```

> [!Warning] 注意
>  `技能名`前后引号包裹,避免歧义
