## 内容
- **[[模型(Model)]]**：是**逻辑的抽象**，解决 " 是什么、为什么 " 的底层规律（如：履约能力的逻辑闭环）。
- **框架 (Framework)**：是**结构的支撑**，提供 " 怎么做、放哪里 " 的骨架支点（如：评估维度的四大板块）。
- **体系 (System)**：是**完整的生态**，由多个模型和框架协同运作形成的完整闭环（如：公司整体的履约管理制度）。

### 对比总结

| 词汇     | 比喻              | 表达的重点       |
| ------ | --------------- | ----------- |
| **模型** | 建筑的**设计图**      | 内部逻辑与原理(机理) |
| **框架** | 建筑的**钢筋骨架**     | 划分领域与范围(构成) |
| **体系** | 整座**大楼**（含水电物业） | 完整功能与运作     |
|        |                 |             |

> [!一句话总结] 
> **模型**提供核心逻辑，**框架**搭建运作结构，**体系**集成全部要素。
