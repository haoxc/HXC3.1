## 内容
视觉思维先驱 **Dave Gray**（《[Gamestorming](https://gamestorming.com/)》与《Liminal Thinking》作者）开发的**视觉框架（Visual Frameworks）**，是==一套旨在将抽象思维、复杂问题和团队假设转化为可见、可共享模式的“认知工具箱”==

### 核心定义
视觉框架被描述为**思维的积木**。它们是简单的`图形草图`，代表了人类脑中已有的原型或心理模型（如“摆锤”、“太阳系”或“季节”）。其目的是： 
- **外化思维**：将脑中的模糊想法“放在桌面上”，供团队共同审视。
- **重构问题**：通过不同的图形隐喻（Metaphors）改变人们看待困境的角度。
- **模式识别**：利用格式塔心理学原理（如简化、组织和闭合），让复杂系统变得直观易懂。

### 2. 主要形式：卡片组 (Card Deck) 

Dave Gray 推出了一套包含 **100 个框架**的实体卡片组，旨在作为解决问题的催化剂： 
- **构成**：每张卡片正面是一个简单的 #手绘草图 ，背面是标题和标签，作为一个“语义手柄”便于检索。
- **数字化应用**：除了实体卡，该系统也被集成到了 Mural、Excalidraw 等协作工具中，用于在线工作坊。 

### 3. 应用场景

- **解决“僵局” (Getting Unstuck)**：当团队在某个问题上停滞不前时，通过随机抽取或有意识选择一个框架（如“极地探索”或“漏斗”），引导出新的讨论方向。
- **战略设计**：将不同的框架像乐高一样组合，构建完整的战略叙事或产品路线图。
- **建立共识 (Alignment)**：在高度复杂或政治敏感的组织环境中，通过共同绘图来发现隐藏的动态。 

### 4. 代表性框架示例

| 框架名称                   | 象征意义       | 应用            |
| ---------------------- | ---------- | ------------- |
| **太阳系 (Solar System)** | 核心与卫星的关系   | 客户中心化或生态系统建模  |
| **摆锤 (Pendulum)**      | 在两个极端之间摆动  | 分析组织文化或政策的变化  |
| **季节 (Seasons)**       | 周期性的循环与更替  | 理解产品生命周期或团队动态 |
| **管道路线图 (Tube Map)**   | 抽象化复杂的连接路径 | 简化多阶段的任务流程或计划 |
