## 内容
### 1. [[语义(Semantic)|语义]] (Semantics) —— **"是什么"**

- **核心：** 研究语言的**意义**。
- **关注点：** 词汇的本义、引申义，以及句子如何通过逻辑组合表达明确的信息。它关注的是语言与客观世界或思维概念之间的指代关系。
- **判定标准：** 真/假、逻辑成立/不成立。
- **例子：** "太阳从西边升起"在**语义**上是清晰的，但违背事实；"绿色的思想愤怒地睡觉"在语法上没错，但在语义上是逻辑冲突的。

### 2. 修辞 (Rhetoric) —— **"怎么说更好"**

- **核心：** 研究语言的**表达效果**。
- **关注点：** 如何利用各种艺术手段（如比喻、拟人、排比）来增强说服力、感染力或美感。它侧重于在特定语境下，如何选择最恰当的表达方式。
- **判定标准：** 生动/平淡、得体/不得体、优美/拙劣。
- **例子：** "由于天干物燥，森林起火了"是陈述事实；而"大火像一条红色的巨龙吞噬了森林"则是**修辞**，增加了画面感。

### 3. 文法/语法 (Grammar) —— **"怎么组装(宏观)"**

- **核心：** 研究语言的**结构规则**。
- **关注点：** 词、短语、句子如何按照固定的模式组合在一起。它像是一本"组装说明书"，规定了主语、谓语、宾语等成分的`排列顺序`。
- **判定标准：** 正确/错误、规范/不规范。
- **例子：** "我吃饭"符合汉语**语法**；"饭我吃"在特定语气下成立，但"吃我饭"（指人吃我的饭）和"饭吃我"则改变了结构关系或逻辑。

### 4. 句法 (Syntax) —— "微观结构的组装"

"[[句法(Syntax)]]"是一个**子概念**。它是语法中专修 **"句子构造"** 的那一部分。

- **包含范围：** 仅限于词与词、短语与短语如何组合成句子的规则。
- **研究内容：**
    - **语序：** 主语、谓语、宾语的先后顺序。
    - **结构：** 动宾结构、偏正结构、主谓结构等。
    - **虚词：** "的、地、得"在句子里的连接作用。
- **地位：** 它是语法的核心组成部分。

### 总结辨析表

| 维度     | 对应属性 | 目标         | 违背后果    |
| ------ | ---- | ---------- | ------- |
| **文法** | 结构性  | 保证句子**通顺** | 病句、乱码   |
| **语义** | 逻辑性  | 保证意思**明白** | 逻辑混乱、歧义 |
| **修辞** | 艺术性  | 保证表达**精彩** | 语言乏味、生硬 |
|        |      |            |         |

**三者关系：** 文法是**基础**（组装好），语义是**灵魂**（有内容），修辞是**外衣**（穿得漂亮）。
