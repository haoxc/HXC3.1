## 概述
**5W2H 方法**（又称“七何”分析法）是一种**结构化的思维工具**，起源于二战时期的美国陆军兵器修理部，旨在通过回答七个关`键问题`来全面、系统地分析问题、规划项目或描述情况。 
它确保在决策或行动计划中不遗漏任何重要方面，并有助于找出问题的根本原因。 

5W2H 七个关键问题

| 缩写           | 问题 (中文) | 侧重点           | 目的/应用                       |
| ------------ | ------- | ------------- | --------------------------- |
| **What**     | 是什么？    | 问题的性质、目标、任务内容 | 明确要做什么，问题的具体现象是什么。`现象本质的定义` |
| **[[Why]]**  | 为什么？    | 原因、目的、必要性     | 找出`根本原因`，论证行动的`合理性`或`重要性`。  |
| **Who**      | 谁？      | 相关人员、责任人、受影响方 | 确定参与者、负责人和涉及的群体。            |
| **When**     | 何时？     | 时间点、期限、时间表    | 确定事件发生时间、开始/结束日期或频率。        |
| **Where**    | 何地？     | 地点、位置、场景      | 定位问题发生的具体位置或背景环境。           |
| **How**      | 如何做？    | 方式、方法、过程、步骤   | 描述解决问题或执行任务的具体过程和方法。        |
| **How much** | 多少？     | 数量、成本、预算、程度   | 量化结果、评估成本、衡量影响范围或绩效。        |
|              |         |               |                             |

**使用场景：** 5W2H 方法广泛应用于[[01-项目管理]]、质量控制（[[六西格玛 (Six Sigma)]]）、[[故障诊断]]、会议记录和[[持续改进]]流程中。它简单易懂，能够帮助团队理清思路，提高效率。 

## 思维维度

- 事实层面(What):  #解决问题/思维框架/事实层面
- 描述层面(Who, Where ,When):  ==疑问词通常属于**事实性**和**描述性**问题的范畴，是信息收集的**基础层面**==。 #解决问题/思维框架/描述层面
-  "How-to" 指南主要回答==**操作层面**和**程序性**的问题==。它们旨在提供分步说明，解决特定的、实际的挑战，核心关注点是**“如何做” (How)** ，而不是“为什么” (Why) 或“是什么” (What)。 #解决问题/思维框架/操作层面
