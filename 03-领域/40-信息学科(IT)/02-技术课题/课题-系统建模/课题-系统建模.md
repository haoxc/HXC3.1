### 核心对比表 (Quick Comparison)

| 维度        | Mermaid 🧜‍♀️                     | D2 🧊                              |
| --------- | --------------------------------- | ---------------------------------- |
| **语法一致性** | **Low** (每种图语法不同)                 | **High** (统一且简洁的 DSL)              |
| **视觉美观度** | **Standard** (比较基础)               | **Premium** (专业级配色与间距)             |
| **布局引擎**  | **Limited** (Dagre)               | **Powerful** (支持 ELK, Tali 等多种)    |
| **集成度**   | **Ultra High** (GitHub/Notion 原生) | **Rising** (需插件或 CLI)              |
| **特色功能**  | **Gantt / Mindmap**               | **Sketch Mode / SQL / Code Block** |
