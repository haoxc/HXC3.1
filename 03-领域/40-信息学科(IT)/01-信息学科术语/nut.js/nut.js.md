# 总结
nut.js 是一个旨在`简化桌面自动化`任务的 JavaScript/TypeScript 库，它提供了丰富的 API 来控制鼠标、键盘、屏幕内容以及应用程序窗口，支持插件扩展，适用于 Windows、macOS 和 Linux 系统，并提供了不同的定价计划以满足不同用户的需求。

# 摘要
nut.js 是一个`桌面自动化库`，它使得通过鼠标和键盘操作、屏幕图像和文本搜索等自动化任务变得简单。它支持跨平台使用，并具有易于使用的特点，同时提供了一个插件系统，允许用户根据自己的需求进行功能扩展。nut.js 支持对应用程序窗口的操作，如搜索、移动、调整大小和最小化等，并能够直接访问和交互 GUI 元素，增强了测试的可靠性和稳定性。此外，它与 [[MCP框架-DevTools MCP|Playwright MCP]] 和 Jest 集成，扩展了测试能力，并支持 OCR 和图像搜索功能，这对于复杂应用的测试特别有用。
