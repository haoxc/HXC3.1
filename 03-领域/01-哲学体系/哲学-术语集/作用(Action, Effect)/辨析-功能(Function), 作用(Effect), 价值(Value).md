## 内容
三者的逻辑关系

可以用一个公式简单概括：功能 + 环境/使用 = 作用 $\arrow$ 主体认可 = 价值。

| 概念     | 关注点   | 视角           | 稳定性      |
| ------ | ----- | ------------ | -------- |
| **功能** | 属性/构造 | 客观 (Object)  | 强        |
| **作用** | 过程/影响 | 动态 (Process) | 中        |
| **价值** | 意义/效益 | 主观 (Subject) | 弱 (随需求变) |
