## 内容
典型的[[简笔画]] (Doodle/Line Drawing)，与[[素描]] (Sketch) 在视觉表现和创作逻辑上有显著区别。
以下是它们的核心对比：

| 特性   | 简笔画 (Doodle)                            | 素描 (Sketch)                     |
| ---- | --------------------------------------- | ------------------------------- |
| 核心目的 | **提取特征**：用最少的线条清晰表达物体的“意思”。             | 还原真实：通过光影、质感、结构表现物体的“体积感”。      |
| 线条运用 | 单一轮廓线：粗细均匀，通常没有阴影和过渡，线条封闭且肯定。           | 多层排线：线条密集多变，有虚实之分，通过重叠形成明暗面。    |
| 空间感  | **平面化**：通常忽略透视，像这张图中的房子和人基本处于同一平面，没有深度。 | **立体感**：严格遵循透视规律，强调物体在三维空间中的存在。 |
| 视觉元素 | `符号化`：太阳是圆圈加直线，人是火柴人，树是云朵状。             | `细节化`：会刻画树皮的纹理、阳光下物体的阴影、衣服的褶皱。  |
|      |                                         |                                 |

## 针对这张图的分析

这张图是标准的简笔画风格，具体表现为：
- 高度简化：人物使用了经典的“火柴人”变体（圆头、直线躯干），树木和房屋仅由基础几何图形（圆弧、长方形、三角形）构成。
- 符号化表达：太阳带有拟人化的笑脸，云朵是简单的圆弧堆叠。
- 无光影：整幅图只有黑色线条，没有任何灰色调（排线）来表示阴影或光源方向。


> [!important] 重要
> 
**总结**： 简笔画追求的是“易懂”和“童趣”，而素描追求的是“结构”和“真实”。
