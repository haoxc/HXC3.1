### GitNexus 核心总结
GitNexus 是一款主打**极简操作、零配置**的Git图谱可视化工具，核心能力是通过单条命令快速启动本地服务，实现浏览器端自动连接、自动加载Git图谱，全程无需手动配置与页面刷新。

---

### 一、两种启动方式
1.  **临时一键启动（无需全局安装）**
    ```bash
    npx gitnexus@latest serve
    ```
2.  **全局安装后启动**
    ```bash
    npm install -g gitnexus && gitnexus serve
    ```

---

### 二、核心特性与运行流程
1.  **极简启动门槛**：仅需一条命令即可完成本地服务启动，无额外配置步骤
2.  **自动服务监听**：浏览器页面会持续检测本地服务状态，服务启动后自动完成连接，无需手动刷新页面
3.  **自动加载核心功能**：连接成功后，页面会自动打开Git图谱可视化界面，直接进入核心功能环节
