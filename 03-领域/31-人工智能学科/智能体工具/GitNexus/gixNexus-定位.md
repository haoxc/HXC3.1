# GitNexus 核心用途与定位
GitNexus 是一款**零服务器、纯本地运行的代码智能引擎**，核心定位是把任意代码仓库转换为可交互的结构化知识图谱，解决AI编程工具无法深度理解代码架构、依赖与调用关系的核心痛点，同时为开发者提供架构可视化、代码分析、风险评估、AI智能体增强等全链路能力，slogan为**building git for agent context（为AI智能体构建像git一样的上下文感知能力）**。

---

## 一、核心核心用途
### 1. 代码库架构可视化与深度解析，解决“看不懂项目”的痛点
它能自动扫描并解析代码库，基于抽象语法树（AST）提取类、函数、变量、接口等核心元素，预计算所有代码结构关系，生成**交互式知识图谱**，直观呈现：
- 函数全链路调用关系、模块间依赖与继承关系
- 项目层级结构、业务边界与代码聚类
- 跨文件的执行流程与关联链路

支持15+主流编程语言（TypeScript、Python、Java、Go、Rust等），提供拖拽缩放、节点高亮、路径追踪等交互能力，彻底替代传统“逐个文件翻源码、手动脑补调用链”的低效方式，哪怕是上万个文件的大型陌生项目，也能快速理清整体架构。

### 2. 为AI编程助手/智能体赋能，根治AI编码幻觉
这是GitNexus的核心设计目标，也是它最核心的价值：
- 通过**MCP协议（模型上下文协议）** 和Graph RAG能力，把预计算好的代码知识图谱，无缝注入Cursor、Claude Code、Windsurf、Hermes等AI编辑器/智能体，让AI获得架构级的代码上下文感知能力。
- 解决AI“改一个函数崩整个项目”的核心问题：AI不再靠猜测回答，而是能精准知道函数的所有调用方、依赖关系、修改影响范围，大幅减少破坏性变更和幻觉，让AI从“盲目改代码”升级为“架构师级开发”。
- 内置7类MCP工具，支持代码检索、影响范围分析、变更检测、多文件重命名等能力，让AI能直接基于完整代码结构完成复杂开发任务。

### 3. 代码质量与架构诊断，提前规避开发风险
面向开发全流程提供专业的代码与架构分析能力：
- 自动检测循环依赖、高耦合模块、死代码、违规调用链，完成架构腐化诊断，识别代码坏味道。
- 计算圈复杂度、认知复杂度等核心指标，评估项目技术债务水平，给出代码质量综合评分和可落地的重构建议。
- 重构/核心代码修改前，可精准评估改动的影响范围，标注高风险路径，提前规避线上故障，大幅降低大规模重构的风险。

### 4. 自然语言交互式代码查询，降低代码分析门槛
内置Graph RAG智能体，支持用自然语言直接对代码库提问，相比传统文本搜索，能基于代码关系给出精准的结构化结果，而非零散的文本片段：
- 精准查询：“谁调用了UserService.login函数？”“项目有没有循环依赖？”
- 架构分析：“这个项目的认证流程是怎样的？”“核心业务模块有哪些？”
- 辅助开发：自动生成代码解释、标准文档注释、符合规范的Git提交信息。

### 5. 提效团队协作、项目交接与开源项目学习
- **新成员快速上手**：接手陌生项目时，通过知识图谱快速理解整体架构、核心模块与业务边界，大幅降低学习成本，减少团队沟通摩擦。
- **高效代码审查**：CR时快速定位代码变更的跨文件影响，发现隐藏的架构风险，避免只看单文件改动的局限性。
- **开源项目学习**：粘贴GitHub仓库链接/拖入源码ZIP，秒级生成知识图谱，快速理清项目结构与核心逻辑，不用再逐个文件翻读源码。

## 二、核心优势与补充说明
1.  **极致的隐私安全**：采用零服务器架构，Web版所有解析在浏览器内完成，CLI版完全本地运行，代码全程不会上传到任何第三方服务器，索引数据存在本地`.gitnexus/`目录，企业敏感项目也可放心使用。
2.  **极低的使用门槛**：提供两种开箱即用的形态——Web版（零安装，打开gitnexus.vercel.app即可使用）、CLI版（一条命令即可安装、分析本地项目）。
3.  **许可证说明**：项目采用PolyForm Noncommercial许可证，个人非商业用途可免费使用，商业用途需申请官方授权。
