## 内容
- Skills 是知识包，教会 Claude 如何更好地执行任务，而不仅仅是扩展功能的工具。
- Claude 的技能是动态加载的指令、脚本和资源集合，用于提高对特定任务的`性能`，包括创建文档、分析数据等。
- **Agent Skills 开放标准**：技能格式作为开放标准发布，可以跨 AI 平台和工具使用，同时提供了 Python SDK 参考。
