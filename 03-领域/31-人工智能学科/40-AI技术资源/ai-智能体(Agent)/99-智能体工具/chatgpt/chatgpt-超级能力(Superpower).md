## 概述

Superpower ChatGPT 是一款浏览器扩展（Chrome / Firefox），为 ChatGPT 网页端补充原生界面缺失的效率功能。核心定位是**让 ChatGPT 的重度用户更高效地管理和复用对话**。

## 核心功能

| 功能 | 说明 |
|------|------|
| **对话文件夹** | 按主题/项目对聊天记录分组管理，支持嵌套 |
| **提示词库** | 保存常用提示词模板，一键插入对话 |
| **全局搜索** | 跨所有对话搜索历史内容 |
| **批量操作** | 多选对话后批量归档、删除、导出 |
| **对话导出** | 导出对话为 Markdown / JSON / PDF |
| **自定义指令** | 按对话设置独立的 system prompt |
| **社区提示词** | 内置社区分享的提示词合集 |
| **同步备份** | 跨设备同步设置和对话数据 |

## 适用场景

- 日常高频使用 ChatGPT，对话量大（单日 20+ 次对话）
- 需要按项目/主题长期维护对话记录
- 需要复用和迭代优质提示词

## 竞品对比

| 工具 | 定位 | 差异 |
|------|------|------|
| **Superpower** | ChatGPT 增强扩展 | 功能全面，社区提示词丰富 |
| **ChatGPT Folders** | 轻量文件夹 | 仅做对话分类，功能单一 |
| **GPTs 内置** | ChatGPT 官方 | 无批量管理、全局搜索等能力 |
