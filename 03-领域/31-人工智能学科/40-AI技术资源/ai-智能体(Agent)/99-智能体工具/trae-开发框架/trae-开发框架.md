## 特性
- 🌊 **概览视图(Lakeview)**：针对智能体步骤提供简短精炼的总结 (Provides short and concise summarisation for agent steps)
- 🤖 **多大模型支持 (Multi-LLM Support)**：支持 OpenAI, Anthropic, 豆包, Azure, OpenRouter, Ollama 以及 Google Gemini API
- 🛠️ **丰富的工具生态 (Rich Tool Ecosystem)**：文件编辑、Bash 命令执行、**顺序思维/链式思考 (Sequential Thinking)** 等
- 🎯 **交互模式 (Interactive Mode)**：用于迭代开发的对话式界面
- 📊 **轨迹记录 (Trajectory Recording)**：记录所有智能体动作的详细日志，用于调试与分析
- ⚙️ **灵活配置 (Flexible Configuration)**：支持环境变量的 **YAML 配置**
- 🚀 **简易安装 (Easy Installation)**：基于 **pip** 的简单安装方式
