## 内容
| **层级** | **名称**                        | **定义**              | **谁在看？** | **沟通重点**                |
| ------ | ----------------------------- | ------------------- | -------- | ----------------------- |
| **L1** | **北极星指标 (North Star Metric)** | 公司的绝对核心，全员对齐的唯一目标。  | CEO/高管   | **战略方向** (e.g., 总营收)    |
| **L2** | **过程/原子指标 (Process Metrics)** | 影响北极星指标的`拆解因子`。     | 部门负责人    | **归因分析** (e.g., 各渠道转化率) |
| **L3** | **执行/监控指标 (Action Metrics)**  | 一线操作层面的具体数据，用于排查问题。 | 一线执行者    | **异常报警** (e.g., 页面加载时长) |
|        |                               |                     |          |                         |
