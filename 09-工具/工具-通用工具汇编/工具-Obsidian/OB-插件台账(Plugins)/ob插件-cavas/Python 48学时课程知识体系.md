# Python 48学时课程知识体系

- **第一阶段：Python基础入门（12学时）**
  - **环境搭建**
    - 安装Python与PyCharm
    - 开发环境界面熟悉
    - 代码运行与调试
  - **基础语法**
    - 变量与数据类型（int, float, str, bool）
    - 基本运算符（+,-,*,/,%,//,**）
    - 输入输出（input(), print()）
  - **控制流程**
    - 条件语句（if-elif-else）
    - 循环语句（for, while）
    - break/continue
  - **实践案例**
    - *计算BMI指数*
    - *简单计算器*

- **第二阶段：数据结构与函数（12学时）**
  - **数据结构**
    - 列表（list）：增删改查、切片
    - 字典（dict）：键值对、常用方法
    - 元组（tuple）与集合（set）
    - 字符串操作（split, join, format）
  - **函数**
    - 函数定义与调用
    - 参数传递（位置参数、关键字参数）
    - 返回值
    - lambda表达式
  - **文件操作**
    - 文件打开与关闭（open, close）
    - 读写文本文件
    - CSV文件处理
  - **实践案例**
    - *学生成绩管理*
    - *文本词频统计*

- **第三阶段：面向对象与实用库（12学时）**
  - **面向对象编程**
    - 类与对象的概念
    - 属性与方法
    - 继承与多态（简单应用）
    - 封装（简单理解）
  - **常用库**
    - Numpy：数组操作
    - Pandas：数据处理（DataFrame）
    - Matplotlib：数据可视化
  - **实践案例**
    - *创建学生类*
    - *绘制销售数据折线图*

- **第四阶段：数据分析实战（12学时）**
  - **数据采集**
    - 网络爬虫基础（requests库）
    - 简单网页数据提取
  - **数据处理**
    - 数据清洗（缺失值处理、重复值处理）
    - 数据转换（类型转换、分组聚合）
  - **数据可视化**
    - Matplotlib基础绘图（折线图、柱状图、饼图）
    - 优化图表（标题、标签、颜色）
  - **实践案例**
    - *爬取天气数据*
    - *制作用户年龄分布饼图*

- **核心知识点（必学）**
  - **Python基础语法**
    - 变量与数据类型
    - 控制流程
    - 函数
  - **数据结构**
    - 列表、字典、字符串操作
  - **面向对象**
    - 类与对象的基本概念
  - **数据分析库**
    - Pandas（DataFrame）
    - Matplotlib（绘图）

- **学习建议**
  - 从实际问题出发（如分析成绩单）
  - 多动手实践（避免死记硬背）
  - 建立“问题-解决方案”笔记
  - 项目驱动学习（3个实践项目）

- **资源推荐**
  - 《Python编程从入门到实践》
  - B站“Python 400集”系列
  - Kaggle数据集与案例
  - Jupyter Notebook环境
