## 核心工作流

| 工作流类型    | 知识库参考 | 描述                          |
| -------- | ----- | --------------------------- |
| 日记工作流    | [2]   | 待办、闪念与打卡的高效管理，自动化移动未完成事项    |
| 看板工作流    | [1]   | 任务可视化管理，将任务在不同列中流动          |
| 案件管理     | [4]   | 法律案件的全流程管理，从案情到裁判结果         |
| 读书笔记工作流  | [8]   | 读书笔记集中管理，通过templater定义标准化模板 |

| 闪念/灵感工作流 | [5]   | 用Memos插件记录灵感、任务和闪念          |
| 文科研究工作流  | [10]  | 针对文科研究的特定工作流程               |
|          |       |                             |
## 实现工作流的关键工具

1. **Templater**：创建标准化模板，实现笔记自动化
2. **QuickAdd**：通过模板选择、捕获选择、宏选择实现工作流自动化
3. **Tasks**：任务管理插件
4. **Kanban**：看板管理插件
5. **Memos**：轻量级记录和任务管理
