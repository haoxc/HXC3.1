## 内容
- Flags 就是控制工厂里各种“`隐藏开关`”的面板
- **Flags**（旗标/标志位）是 Chrome 浏览器中用于开启或关闭**实验性功能**的隐藏设置。
- **官方定义**：它是开发者用来测试尚未正式发布的特性（新功能、性能优化、UI 调整）的开关。
- **通俗理解**：它是浏览器的“`隐藏菜单`”。这里的功能可能在未来变成标准配置，也可能因为测试不佳而被彻底移除。

## 进入
通过 URL地址栏 chrome://flags

```url
chrome://flags/{{flag_name}}
```
