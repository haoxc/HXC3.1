## 内容
| 快捷键                   | 功能描述        | 适用场景                                    |
| --------------------- | ----------- | --------------------------------------- |
| **`Command + K`**     | **彻底清屏**    | 屏幕上有敏感 API Key 时，防止别人路过看到。              |
| **`Control + C`**     | **终止进程**    | OpenClaw 运行卡死或你想强行退出 `onboard` 模式。      |
| **`Control + L`**     | **移动光标清屏**  | 只是把当前行提到最上方，滚动条往上翻还能看到旧内容（不如 Cmd+K 安全）。 |
| **`Control + A / E`** | **跳到行首/行尾** | 修改长命令（比如修改 `apiKey` 字符串）时非常快。           |
| **`Option + Click`**  | **快速移动光标**  | 在终端的长 JSON 字符串中，按住 Option 点哪，光标就跳到哪。    |
| `Option + Backspace`  | 删除一个词       |                                         |
| `cmd+Backspace`       | 删除整行        |                                         |
