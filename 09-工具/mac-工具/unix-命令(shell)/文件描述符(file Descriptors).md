在计算机系统中，**文件描述符（File Descriptor，简称 FD）** ==是一个非负整数，它是内核为了高效管理已被打开的文件所创建的**索引**==。

你可以把它想象成一个 **“句柄”** 或 **“排号单”**：每当进程打开一个文件（或硬件设备、网络连接）时，操作系统就会发给你一个数字，你通过这个数字来指挥系统读写。

### 1. 三个“天生”的描述符
每个程序在启动时，操作系统都会默认给它分配三个最基础的描述符：

| 描述符 (FD) | 名称       | 缩写       | 默认目的地/来源    |
| -------- | -------- | -------- | ----------- |
| **0**    | **标准输入** | `stdin`  | 键盘          |
| **1**    | **标准输出** | `stdout` | 屏幕 (终端显示)   |
| **2**    | **标准错误** | `stderr` | 屏幕 (通常显示报错) |
