## 概述
- Matplotlib 是一个用于 Python 的静态、动画和交互式可视化库，提供出版质量的图表制作、交互式图形操作、风格和布局定制、多种文件格式导出以及与 JupyterLab 和图形用户界面的集成。
- Matplotlib 还拥有丰富的第三方扩展包，如 [[Seaborn]]，以及专门的`领域工具，`如 [[Cartopy]]。
