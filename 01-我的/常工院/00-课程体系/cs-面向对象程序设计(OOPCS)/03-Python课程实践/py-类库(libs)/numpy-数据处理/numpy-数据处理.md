# 摘要
- NumPy是Python科学计算的基础库，提供了多维数组对象、各种衍生对象和数组快速操作的集合，包括数学、逻辑、形状操作、排序、选择、输入/输出、[[离散傅里叶变换]]、[[线性代数]]、基本统计和[[随机模拟]]等功能。
-  NumPy 提供了标准的文法和 API 规范
	- CuPy 则是这一标准在 **GPU 硬件** 上的具体实现

## 核心功能
NumPy 的基本使用主要围绕其核心的 **N维数组（N-dimensional arrays）** 展开，提供了高效的功能
- **[[向量化（vectorization）]]**
- [[索引（indexing）]]
- [[广播（broadcasting）]]

## 解构
-  数据结构和属性
	- N维数组具有明确的[[形状（Shape）]]和[[数据类型(DType)]]
- 关键操作
	- [[多维切片]]
	- 数据聚合
		- 轴计算 (数组的聚合运算可以沿着特定的维度进行)
			- `x.max(axis=1)` 表示沿着“行”的方向计算最大值，而不是计算整个数组的最大值。
