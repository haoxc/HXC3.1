### 程序引言

**示例代码**  (**festival.py**)
```python
# 定义变量：用"="给变量赋值（变量名=值）
name = "小明"  # 存储姓名
festival = "生日"  # 存储节日

# 使用变量：直接写变量名，计算机会自动替换为它的值
print(f"{name}，祝你{festival}快乐！")  # 输出：小明，祝你生日快乐！
```

> [!NOTE] 课堂观察
    > - 观察以上代码有那些元素，并总结其中结构规律
    > - 通过`调试运行`, 观察程序单步执行前后的变化


如果想给不同的人发祝福，总不能每次都修改代码。这时可以用 "变量" 存储数据（比如姓名），就像用标签纸给盒子贴标签，方便后续使用。
Python 的魅力就在于：简单的语法能实现丰富的功能。从这个小案例出发，你可以继续探索 —— 比如读取 Excel 中的名单批量生成祝福，或者用图形库制作带图片的电子卡片。编程的世界里，创意才是唯一的限制，开始你的探索吧！

### 实践
#### 实践1- 熟悉编译环境和调试过程
- 执行以下代码，观察程序执行每一步的前后变化，分别用文字描述每一行的含义和目的。
```python
name = input("请输入姓名：")
relation = input("关系（朋友/家人/同事）：")

# 根据关系生成不同祝福
if relation == "朋友":
    # 如果关系是朋友，执行这里的代码
    wish = f"嘿，{name}！愿你每天都像拆礼物一样开心，烦恼全被吃掉～"
elif relation == "家人":
    # 如果是家人，执行这里
    wish = f"{name}，平安健康比什么都重要，咱们常回家聚聚呀～"
else:
    # 其他关系（默认同事），执行这里
    wish = f"{name}您好，感谢工作中的支持，祝您事事顺心，效率爆棚！"
print(f"\n{wish}")
```
