## 要求
`功能`:  介绍自己的姓名,班级, 学校; 学校默认为"常州工学院"

---
## 程序
```python
def greet(name="朋友"):
    """生成问候语，默认称呼为"朋友" """
    greeting= f"你好，{name}！"
    print(greeting)

# 调用函数（两种方式）
greet()  # 未传参数，使用默认值，输出：你好，朋友！
greet("小明")  # 传入参数，输出：你好，小明！
