**程序**: 
```python
def print_image_info(**kwargs):
    """打印图片的任意属性（如宽度、高度、格式等）"""
    print("接收的参数（字典）：", kwargs)  # 查看kwargs的结构
    for key, value in kwargs.items():
        print(f"{key}：{value}")

# 调用函数：传入不同的关键字参数
print_image_info(width=1080, height=1920)  # 传入2个参
print_image_info(format='png', size='large', color_mode='RGB')  # 传入3个参数
```

**输出**:
```
# 接收的参数（字典）： {'width': 1080, 'height': 1920}
# width：1080
# height：1920

接收的参数（字典）： {'format': 'png', 'size': 'large', 'color_mode': 'RGB'}
format：png
size：large
color_mode：RGB
```
