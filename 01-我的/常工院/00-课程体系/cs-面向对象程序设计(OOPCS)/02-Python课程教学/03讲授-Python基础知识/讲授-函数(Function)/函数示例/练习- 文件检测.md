## 要求
定义一个函数 `check_img`, 能够检测多个文件的类型,, 其中类型默认为 `png`;
检测所有的 png 文件,  使用场景示例如下:
```python
image_paths = [
    "C:/设计素材/海报1.jpg",
    "C:/设计素材/图标.png",
    "C:/设计素材/背景图.jpeg",
 	"C:/设计素材/头部.png",
 	"C:/设计素材/脚注.png"
]
check_img(image_paths)

```
