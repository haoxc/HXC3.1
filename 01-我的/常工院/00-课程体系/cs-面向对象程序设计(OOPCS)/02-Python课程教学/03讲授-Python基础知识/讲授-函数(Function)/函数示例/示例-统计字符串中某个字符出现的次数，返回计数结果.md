## 要求
`功能`：统计字符串中某个字符出现的次数，返回计数结果。
## 程序
```python
def count_char(text, char):  
    """统计char在text中出现的次数"""  
    count = 0  
    for c in text:  
        if c == char:  
            count += 1  
    print("OK")
    return count  
    
# 调用函数  
text = "Hello, Python!"  
char_count = count_char(text, "o")  
print(f"'o'出现的次数：{char_count}")  # 输出：'o'出现的次数：2
```
