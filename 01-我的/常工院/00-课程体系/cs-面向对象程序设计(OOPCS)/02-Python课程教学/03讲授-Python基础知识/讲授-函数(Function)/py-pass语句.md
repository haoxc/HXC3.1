## 概述
- **定义**: pass 是一个 `空语句`, 为了保持程序结构完整性
- **作用**:
	- **代码可读性**: 用作占位符, 避免语法错误; 用 `pass` 明确标记 “未完成” 或 “空逻辑”，避免他人误判为代码遗漏
	- **方便调试**：快速搭建代码框架时，用 `pass` 占位，优先保证结构完整，后续逐步补充实现。
- **本质**:
	- "语法填充剂"
---
## 场景
###  未完成的函数 / 方法
定义函数但暂时没想好实现逻辑时，用 `pass` 占位，保证函数结构完整。

```python
def calculate_area(width, height):
    # 后续补充计算面积的代码
    pass  # 占位，避免语法错误

# 函数可正常定义和调用（调用时无实际操作）
calculate_area(100, 200)
```

---
###  条件分支中的空逻辑
某些条件下暂时不需要执行任何操作，用 `pass` 明确表示 “此处有意为空”，避免误解为遗漏代码。

```python
score = 65

if score >= 60:
    print("及格")
else:
    # 暂时不处理不及格的情况
    pass  # 明确表示“此处无操作”
```

---
### 循环中的临时占位
在循环结构中，若暂时不需要具体逻辑，用 `pass` 保证循环语法正确。

```python
for i in range(5):
    # 后续补充循环内的操作
    pass  # 占位，循环可正常运行（无实际效果）
```
---
###  未完成的类
定义类但暂时不需要实现属性或方法时，用 `pass` 保持类的结构完整。

```python
class ImageProcessor:
    # 后续补充图片处理相关的属性和方法
    pass  # 占位

# 类可正常实例化（实例暂时无实际功能）
processor = ImageProcessor()
```
