## 概述
`break` 和 `continue` 是循环中的控制语句，用于改变循环的执行流程，常用于 `for` 或 `while` 循环中。

- **作用**：立即终止当前循环，跳出循环体，不再执行循环中剩余的语句和后续迭代。
- **适用场景**：当满足某个条件时，需要提前结束整个循环。
