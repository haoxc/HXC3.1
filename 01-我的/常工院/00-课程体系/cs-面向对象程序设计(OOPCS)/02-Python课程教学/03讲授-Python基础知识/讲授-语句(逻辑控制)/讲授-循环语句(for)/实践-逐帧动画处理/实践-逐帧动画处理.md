## 题目

- **场景**：制作一段 20 帧的逐帧动画，帧文件保存在 “animation/” 文件夹下，命名规则为 “frame_000.png”“frame_001.png”…“frame_019.png”。
- **需求**：用 for 循环遍历 0-19 的帧编号，打印每帧的完整路径（例如 “animation/frame_000.png”）。

> [!hint] **提示**
> - 序列帧编号从 0 开始，共 20 帧（用 `range(20)`）
> - 路径格式为 `"animation/frame_{编号:03d}.png"`

---
## 程序
```python
# 遍历0-19的帧编号（共20帧）
for frame_num in range(20):
    # 格式化路径：文件夹+文件名（编号补零至3位）
    frame_path = f"animation/frame_{frame_num:03d}.png" 
    #...
    # 打印完整路径
    print(frame_path)
    
```

	- 程序结构
	- 基础文法（缩进）
