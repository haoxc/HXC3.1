## 要求
设计模拟用户登录验证的程序，分别获取用户通过键盘的输入信息（用户，密码）
如果用户是 `admin`, 密码是 `123456`, 在控制台输出 `登录成功！欢迎管理员。`；
如果用户是 `admin`, 密码是 `654321`, 在控制台输出 `登录成功！欢迎普通用户。`；
其他提示 `用户名或密码错误，登录失败。`



> [!NOTE] 提示
    > 读取键盘输入文字用 input（`提示信息`）方法。其中参数 `提示信息` 是输入前的提示信息

---
## 程序

```python
# 模拟用户登录验证
username = input("请输入用户名：")
password = input("请输入密码：")

if username == "admin" and password == "123456":
    print("登录成功！欢迎管理员。")
elif username == "user" and password == "654321":
    print("登录成功！欢迎普通用户。")
else:
    print("用户名或密码错误，登录失败。")
```

### 🔍 关键点：
- **`and` 运算符**：同时判断用户名和密码。
- **多条件分支**：区分管理员和普通用户。
- **缩进规则**：每个 `if/elif/else` 对应的代码块需统一缩进。
