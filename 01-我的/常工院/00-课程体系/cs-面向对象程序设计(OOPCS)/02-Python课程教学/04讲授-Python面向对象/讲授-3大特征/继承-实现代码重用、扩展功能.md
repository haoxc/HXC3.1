## 概述
-  “**动态素材**” 是父类，它有共性：时长、帧率、播放状态（播放 / 暂停）；
    - 子类 “**GIF 动图**” 继承这些共性，再新增自己的特性：循环次数（0 = 无限循环）；
    - 子类 “**视频素材**” 继承共性，再新增自己的特性：音频轨道、分辨率（1080P/4K）。

- **代码形式**：
    ```python
    class DynamicMaterial:  # 父类：动态素材
        def __init__(self, duration, fps):
            self.duration = duration  # 时长（秒）
            self.fps = fps  # 帧率（帧/秒）
        
        def play(self):
            print("开始播放")
    
    # 子类GIF：继承父类，新增自己的属性和方法
    class GIF(DynamicMaterial):
        def __init__(self, duration, fps, loop_count):
            super().__init__(duration, fps)  # 继承父类的属性
            self.loop_count = loop_count  # 新增：循环次数
        
        def set_loop(self, new_loop):  # 新增：设置循环的方法
            self.loop_count = new_loop
    ```

- `super()`:   `super()` 是一个用于在子类中*调用父类方法的工具*，核心作用是*复用父类的功能* ，同时让子类在其基础上扩展自己的特性

> [!NOTE] 思想
    > - **父类**是 “通用模板”（如 “动态素材模板”），定义了所有同类素材的基础属性；
    > - **子类**是 “专用模板”（如 “GIF 模板”），通过`super()` 一键套用通用模板的内容，再添加自己的专属设置。
