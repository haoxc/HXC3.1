## 要求
需要定义一个 “图片” 类（`Image`），每个图片对象必须有 “宽度”“高度”“格式” 这 3 个基础属性。其中*格式*默认为 `png`

---
## 定义
```python
class Image:
    # 给format设置默认值'png'，创建对象时可省略该参数
    def __init__(self, width, height, format='png'):
        self.width = width
        self.height = height
        self.format = format

# 此时可以这样创建对象（自动使用默认格式'png'）
pic2 = Image(1024, 768)
print(pic2.format)  # 输出：png
```
