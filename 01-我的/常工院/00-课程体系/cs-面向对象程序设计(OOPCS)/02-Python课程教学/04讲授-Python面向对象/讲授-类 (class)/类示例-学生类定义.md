## 要求
定义一个学生, 包含姓名, 年龄, 分数的属性;  通过学习时间评价学习分值, 构建一个学生类定义.
## 定义
```python
# 定义“学生”类
class Student:  
    # 构造方法：初始化属性（self代表当前对象本身）  
    def __init__(self, name, age):  
        self.name = name  # 姓名属性  
        self.age = age  # 年龄属性  
        self.score = 0  # 成绩属性（默认初始化为0）  
  
    # 方法1：学习（修改成绩）  
    def study(self, hours):  
        """  
        根据学习时长增加学生的成绩。  
        每学习1小时，成绩将增加5分，并打印出学生的学习情况。  
        :param hours: 学习的时长，以小时为单位。必须为非负数。  
        :return: 无返回值。此方法仅修改实例属性并打印一条消息。  
        """
        self.score += hours * 5  # 学习1小时加5分  
        print(f"{self.name}学习了{hours}小时，成绩变为{self.score}")
```
