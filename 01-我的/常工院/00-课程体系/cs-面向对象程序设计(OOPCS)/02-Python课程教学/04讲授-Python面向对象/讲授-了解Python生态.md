### Python 生态
Python 生态就像一个 **“超级工具库”**，里面有无数现成的 “模块”（别人写好的代码集合）。这些模块针对不同需求封装了复杂功能，你不需要从零写代码，直接 “*拿来用*” 即可。

- 你不需要自己造 “PS 的滤镜算法”，直接用别人做好的 “图像处理模块”；
- 不用自己写 “视频剪辑的底层逻辑”，调用 “视频处理模块” 就能批量剪视频。

---

###  如何安装
安装 Python 模块的核心工具是 **`pip`**（Python 官方包管理工具），操作简单且通用。

``` shell
pip install 模块名
```
