#### 课题3：多类型素材类（进阶）
- **实践目标**：掌握类的继承与多态，适配不同类型素材；
- **考查知识点**：继承、重写方法、多态、 isinstance()判断；
- **数媒场景**：统一管理图片、音频、视频三类设计素材；
- **实践内容**：
  1. 定义父类`Material`，包含通用属性（路径、名称、大小）和方法`show_type()`；
  2. 定义子类`ImageMaterial`/`AudioMaterial`/`VideoMaterial`，重写`show_type()`，新增专属方法（如图片`get_resolution()`、音频`get_duration()`）；
  3. 创建混合素材列表，遍历调用`show_type()`，体现多态；
- **交付物**：拆分到`material.py`模块的代码+素材类型识别测试结果。
