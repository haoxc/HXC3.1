## 核心思路
- **阶段1（基础）**：
	- 掌握类的定义、实例化、属性/方法设计，聚焦「单一功能封装」；
- **阶段2（进阶）**：
	- 掌握继承、多态、模块拆分，聚焦「批量素材处理」；
- **阶段3（综合）**：
	- 综合面向对象知识，完成「小型创意工具开发」。

## 实践策略
1. **案例贴合数媒**：所有课题均围绕「素材管理、视觉创作、设计工具」，避免纯编程案例；
2. **分层要求**：基础课题要求全员完成，进阶/综合课题可分「基础版/创意版」（如创意版允许自定义配色算法）；
3. **成果可视化**：要求交付物包含「代码+视觉成果」（截图/图片/报告），符合数媒生的展示习惯；
4. **拓展方向**：综合课题可拓展为小组合作，如「素材管理系统+可视化界面（tkinter）。
