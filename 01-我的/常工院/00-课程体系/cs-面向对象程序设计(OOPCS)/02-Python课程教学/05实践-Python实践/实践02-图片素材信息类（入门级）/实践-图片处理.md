## 试题内容
### 题目要求
#### 1. 模块结构设计
创建名为`material_manager.py`的模块，包含两个类：
- **`ImageMaterial`类**：描述单张图片素材的属性与操作
	  - 初始化参数：素材路径（`path`）
	  - 实例属性：路径（`_path`）、文件名（`_name`）、格式（`_format`）、尺寸（`_size`，元组`(宽,高)`）
	  - 实例方法：`get_info()` → 返回素材信息字典；`convert_format(new_format)` → 转换图片格式并保存
- **`MaterialLibrary`类**：管理素材库的整体操作
	  - 初始化属性：素材列表（`_materials = []`）
	  - 实例方法：
	    - `add_material(path)` → 添加素材到库（自动创建`ImageMaterial`实例）
	    - `search_by_format(format)` → 按格式筛选素材，返回符合条件的实例列表
	    - `batch_convert(target_format)` → 批量转换库中所有素材格式，保存到`converted_materials`文件夹
	    - `generate_report()` → 生成素材库报告（TXT文件），包含总数、各格式数量、总像素面积

#### 2. 主程序测试
创建`main.py`文件，导入`material_manager`模块，完成：
- 创建素材库实例，添加3-5张图片素材
- 按格式筛选素材（如PNG格式）
- 批量转换所有素材为WebP格式
- 生成素材库报告

### 实现提示
1. `ImageMaterial`类中，用`PIL.Image.open()`提取图片尺寸、格式等信息；
2. `MaterialLibrary`类中，用列表存储`ImageMaterial`实例，通过遍历实现批量操作；
3. 格式转换时用`os.path.splitext()`处理文件名，`os.makedirs()`创建保存文件夹；
4. 模块中可添加异常处理（如文件不存在、不支持的格式）。

## 代码示例
### 1. `material_manager.py`模块文件
```python
from PIL import Image
import os

class ImageMaterial:
    def __init__(self, path):
        self._path = path
        self._name = os.path.splitext(os.path.basename(path))[0]
        # 提取图片信息（处理异常）
        try:
            with Image.open(path) as img:
                self._format = img.format.lower()
                self._size = img.size  # (宽, 高)
        except Exception as e:
            raise ValueError(f"无法读取图片{path}：{str(e)}")

    def get_info(self):
        """返回素材信息字典"""
        return {
            "name": self._name,
            "path": self._path,
            "format": self._format,
            "size": self._size,
            "area": self._size[0] * self._size[1]  # 像素面积
        }

    def convert_format(self, new_format, save_folder="converted_materials"):
        """转换图片格式并保存"""
        new_format = new_format.lower()
        if new_format not in ["png", "jpg", "jpeg", "webp"]:
            raise ValueError(f"不支持的格式：{new_format}")
        
        os.makedirs(save_folder, exist_ok=True)
        new_path = os.path.join(save_folder, f"{self._name}.{new_format}")
        
        try:
            with Image.open(self._path) as img:
                # JPG格式需处理透明度
                if new_format == "jpg" and img.mode == "RGBA":
                    img = img.convert("RGB")
                img.save(new_path, new_format.upper())
            return new_path
        except Exception as e:
            raise RuntimeError(f"转换失败{self._name}：{str(e)}")

class MaterialLibrary:
    def __init__(self):
        self._materials = []  # 存储ImageMaterial实例

    def add_material(self, path):
        """添加素材到库"""
        try:
            material = ImageMaterial(path)
            self._materials.append(material)
            print(f"成功添加素材：{material.get_info()['name']}")
        except Exception as e:
            print(f"添加失败：{str(e)}")

    def search_by_format(self, target_format):
        """按格式筛选素材"""
        target_format = target_format.lower()
        return [mat for mat in self._materials if mat.get_info()["format"] == target_format]

    def batch_convert(self, target_format):
        """批量转换所有素材格式"""
        converted_paths = []
        for mat in self._materials:
            try:
                new_path = mat.convert_format(target_format)
                converted_paths.append(new_path)
                print(f"转换完成：{new_path}")
            except Exception as e:
                print(f"批量转换失败：{str(e)}")
        return converted_paths

    def generate_report(self, report_name="material_report.txt"):
        """生成素材库报告"""
        if not self._materials:
            print("素材库为空，无法生成报告")
            return
        
        total_count = len(self._materials)
        format_count = {}  # 统计各格式数量
        total_area = 0
        
        for mat in self._materials:
            info = mat.get_info()
            fmt = info["format"]
            format_count[fmt] = format_count.get(fmt, 0) + 1
            total_area += info["area"]
        
        # 写入报告
        with open(report_name, "w", encoding="utf-8") as f:
            f.write("=== 数媒素材库报告 ===\n")
            f.write(f"总素材数量：{total_count}\n")
            f.write(f"总像素面积：{total_domain:,} 像素\n")
            f.write("\n格式分布：\n")
            for fmt, count in format_count.items():
                f.write(f"- {fmt.upper()}：{count} 张（占比{count/total_count*100:.1f}%）\n")
        
        print(f"报告已生成：{report_name}")
        return report_name
```

### 2. `main.py`主程序文件
```python
from material_manager import MaterialLibrary

if __name__ == "__main__":
    # 创建素材库实例
    library = MaterialLibrary()

    # 添加素材（替换为你的图片路径）
    library.add_material("C:/设计素材/海报1.jpg")
    library.add_material("C:/设计素材/图标.png")
    library.add_material("C:/设计素材/背景图.jpeg")

    # 按格式筛选PNG素材
    png_materials = library.search_by_format("png")
    print("\nPNG格式素材：")
    for mat in png_materials:
        print(mat.get_info()["name"])

    # 批量转换为WebP格式
    print("\n批量转换格式：")
    library.batch_convert("webp")

    # 生成素材报告
    library.generate_report()
```
