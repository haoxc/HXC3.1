### 图片素材信息类
- **实践目标**：封装单张图片的属性与基础查询功能；
- **考查知识点**：
	- 类的封装、属性私有化、getter方法、PIL库基础；
- **数媒场景**：快速查看素材尺寸/格式/大小，标注素材信息；
- **实践内容**：
  1. 定义`ImageInfo`类，初始化参数为图片路径；
  2. 私有化属性：路径、文件名、尺寸、格式、文件大小；
  3. 实现方法：`get_info()`（返回素材信息字典）、`print_info()`（格式化打印）；
  4. 批量实例化5张设计素材，输出信息对比表；
- **交付物**：代码文件+素材信息对比表（TXT/截图）。

> **备注**
> - pillow类库:  https://pypi.org/project/pillow/
