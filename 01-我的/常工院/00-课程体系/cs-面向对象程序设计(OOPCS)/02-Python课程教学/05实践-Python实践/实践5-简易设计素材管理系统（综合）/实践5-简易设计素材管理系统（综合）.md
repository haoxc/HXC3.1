#### 课题5：简易设计素材管理系统（综合）
- **实践目标**：综合运用面向对象，开发可落地的小型工具；
- **考查知识点**：类的封装/继承、模块拆分、文件IO、异常处理、简单交互；
- **数媒场景**：数媒生自用的素材库管理（添加、查询、筛选、导出报告）；
- **实践内容**：
  1. 模块拆分：`material_classes.py`（素材类）、`library_manager.py`（素材库管理类）、`main.py`（交互入口）；
  2. 核心功能：
     - 添加图片/音频素材到库；
     - 按格式/尺寸/名称筛选素材；
     - 批量导出素材信息为Excel/TXT报告；
     - 异常处理（文件不存在、格式不支持）；
  3. 简单交互：命令行输入指令（如「1-添加素材」「2-筛选素材」）；
- **交付物**：完整代码工程+使用说明文档+素材库测试报告。
