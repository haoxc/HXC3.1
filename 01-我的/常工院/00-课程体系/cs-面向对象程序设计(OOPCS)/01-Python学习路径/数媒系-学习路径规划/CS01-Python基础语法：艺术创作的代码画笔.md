# Python基础语法：艺术创作的代码画笔

## 一、代码规范性：让代码像艺术品一样整洁

### 1. 注释 - 作品的创作说明

```python
# 画一个彩色的圆圈（这是注释，代码不会执行）
# 参数说明：半径100，颜色#FF5733（橙色）
turtle.circle(100, 360, fillcolor="#FF5733")
```

**艺术生视角**：
- 单行注释 `#` = 作品说明文字
- 多行注释 `"""` = 作品创作理念说明

> 💡 小技巧：写代码前先写注释，就像画草图一样
---
### 2. 代码缩进 - 作品的构图层次

```python
# 画一朵花（缩进表示画花瓣的层次）
def draw_flower():
    for i in range(36):  # 画36个花瓣
        turtle.circle(50)  # 画一个圆
        turtle.right(10)   # 旋转10度
```

**艺术生视角**：

- 缩进 = 作品的层次感
- 4个空格 = 画布的网格线
- 缩进错误 = 作品结构混乱

> 💡 小技巧：使用4个空格缩进，像画网格一样规范

---
## 二、变量：艺术创作的"调色板"

### 1. 变量 = 艺术作品的"颜色/形状/大小"参数

```python
# 定义颜色参数
primary_color = "#FF5733"  # 橙色
secondary_color = "#33FF57"  # 绿色

# 定义形状参数
circle_radius = 100
flower_petals = 36
```

**艺术生视角**：

- 变量名 = 你想要的创作元素（如：`circle_radius`）
- 变量值 = 具体的创作参数（如：`100`）
- 变量命名 = 用下划线分隔的英文单词（`circle_radius`）

> 💡 小技巧：用`color1`、`color2`这样的命名，就像调色板上的颜色标签

---
### 2. 命名规则：创作的规范

```python
# 正确命名
flower_size = 150
background_color = "#FFFFFF"
# 错误命名（艺术生常犯的错误）
flower_size1 = 150
back_color = "#FFFFFF"
```

**艺术生视角**：
- 以字母或下划线开头
- 不要使用中文或拼音（如`背景色`）
- 用英文描述，保持简洁

> 💡 重点：变量名要能让人一眼看出它的作用

---
## 三、数据类型：艺术创作的"材料库"

### 1. 字符串：艺术作品的"文字/标题"

```python
# 作品标题
title = "数字星空"
# 艺术家签名
artist = "数字艺术生"

# 文字组合
print(f"作品: {title} | 作者: {artist}")
# 输出: 作品: 数字星空 | 作者: 数字艺术生
```

**艺术生视角**：
- 字符串 = 用引号包裹的文字
- `f-string` = 用花括号`{}`插入变量的高级字符串

> 💡 重点：`f"文本{变量}"` 是艺术作品展示的必备技能

---
### 2. 数字类型：艺术作品的"参数"

```python
# 整数 - 用于计数
petals = 36  # 花瓣数量
# 浮点数 - 用于精确控制
opacity = 0.7  # 透明度
# 布尔值 - 用于条件判断
is_gradient = True  # 是否渐变
```

**艺术生视角**：

- `int` = 整数（如花瓣数量）
- `float` = 小数（如透明度、角度）
- `bool` = 真/假（如是否渐变）

> 💡 重点：数字类型决定你的作品能精确到什么程度

---

## 四、运算符：艺术创作的"混合调色"

### 1. 基本运算符：调色的"混合规则"

```python
# 色彩混合
red = 255
green = 87
blue = 51

# 生成新颜色（混合红绿蓝）
new_color = (red + green + blue) // 3  # 平均值
print(f"混合色: {new_color}")  # 输出: 混合色: 131
```

**艺术生视角**：

- `+` = 混合颜色
- `-` = 减少颜色
- `*` = 增强颜色
- `/` = 分解颜色

> 💡 重点：运算符是数字艺术创作的"色彩混合规则"

---
### 2. 逻辑运算符：创作条件的"判断"

```python
# 选择不同的创作方式
is_day = True
is_night = False

# 如果是白天，使用明亮的色彩
if is_day:
    color = "#FFD700"  # 金色
else:
    color = "#0000FF"  # 蓝色

print(f"当前使用颜色: {color}")
```

**艺术生视角**：

- `and` = 同时满足两个条件
- `or` = 满足其中一个条件
- `not` = 反转条件
> 💡 重点：逻辑运算符是创作的"决策规则"

---
## 五、流程控制：艺术创作的"步骤规划"

### 1. 条件语句：创作的"分步决策"

```python
# 根据心情选择创作方式
mood = "happy"

if mood == "happy":
    print("今天用明亮的色彩创作！")
elif mood == "sad":
    print("今天用柔和的色彩创作！")
else:
    print("今天用大胆的色彩创作！")
```

**艺术生视角**：
- `if` = 如果...
- `elif` = 否则如果...
- `else` = 否则...

> 💡 重点：条件语句是艺术创作的"灵感触发器"

### 2. 循环语句：创作的"重复模式"

```python
# 用循环创建重复的图案
for i in range(10):  # 画10个圆
    turtle.circle(50 + i*10)
    turtle.right(36)
```

**艺术生视角**：

- `for` = 重复绘制（适合几何图案）
- `while` = 条件满足时重复（适合动态效果）

> 💡 重点：循环语句是数字艺术中"重复美学"的核心

## 六、函数：艺术创作的"模块化工具"

### 1. 函数 = 你的"创作工具包"

```python
# 定义一个画花瓣的函数
def draw_petal(size, color):
    turtle.color(color)
    turtle.begin_fill()
    turtle.circle(size, 180)
    turtle.left(180)
    turtle.circle(size, 180)
    turtle.end_fill()

# 使用函数
draw_petal(50, "#FF5733")  # 画一个橙色花瓣
draw_petal(40, "#33FF57")  # 画一个绿色花瓣
```
**艺术生视角**：
- `def` = 定义工具
- `()` = 工具的参数（大小、颜色）
- `return` = 工具的输出（如果需要）
> 💡 重点：函数是艺术创作的"可重复使用工具"

## 七、数据结构：艺术创作的"作品管理"

### 1. 列表：艺术作品的"作品集"
```python
# 创建一个颜色列表
colors = ["#FF5733", "#33FF57", "#3357FF", "#F3FF33"]

# 选择第一个颜色
first_color = colors[0]  # "#FF5733"
print(f"第一个颜色: {first_color}")

# 添加新颜色
colors.append("#FF33A8")
```

**艺术生视角**：

- 列表 = 作品集，可以按顺序存储
- `[]` = 作品编号
- `.append()` = 添加新作品

> 💡 重点：列表是管理多个创作元素的"最佳工具"

---
### 2. 字典：艺术作品的"属性管理"
```python
# 创建一个作品信息字典
artwork = {
    "title": "数字星空",
    "artist": "数字艺术生",
    "year": 2023,
    "colors": ["#0000FF", "#3357FF", "#FF5733"]
}

# 获取作品信息
print(f"作品: {artwork['title']}")
print(f"使用颜色: {artwork['colors'][0]}")
```

**艺术生视角**：
- 字典 = 作品属性，用键值对存储
- `"title"` = 作品标题
- `"colors"` = 作品使用的颜色

> 💡 重点：字典是管理作品详细信息的"最佳工具"

## 今日小任务：用Python创作你的第一个数字艺术作品

```python
import turtle

# 设置画布
turtle.bgcolor("#000000")
turtle.speed(0)

# 定义颜色
colors = ["#FF5733", "#33FF57", "#3357FF", "#F3FF33"]

# 画一个多彩的螺旋
for i in range(100):
    turtle.color(colors[i % 4])
    turtle.circle(i * 2)
    turtle.left(59)

turtle.done()
```
