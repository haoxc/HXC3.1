## Abstract
>D-Tale is the combination of a [[Flask]] back-end and a [[React@React overview|React]] front-end to bring you an easy way to view & analyze Pandas data structures. It integrates seamlessly with [[ipython]] notebooks & python/ipython terminals. Currently this tool supports such [[CZU常工院/课程安排/归档/A01-游戏引擎课程/游戏引擎课程教学/Python类库/Pandas/Pandas]] objects as DataFrame, Series, MultiIndex, DatetimeIndex & RangeIndex.	

## Features
### Data  Preparation
- Change datatype with one click
- Create a new column with different format
- Dropping columns
- Change the column’s name
### Data Transformation
- Filtering data
- Merging data
- Grouping By
- Describe data

### Data Visualization
- Bar plot
- Plot code
- Scatter Plot
- Time Series
- Describe data

### Advances
- Change language and dark mode
- Highlight columns
