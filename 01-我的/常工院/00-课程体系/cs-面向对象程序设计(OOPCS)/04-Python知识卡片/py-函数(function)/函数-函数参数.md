### 函数的 “参数”

参数是函数的 “输入选项”，让同一个函数能处理不同需求（比如同一 “加边框” 动作，可改边框宽度）。

| 参数类型  | 数媒类比           | 示例（加边框函数）                                                |
| ----- | -------------- | -------------------------------------------------------- |
| 位置参数  | 必选选项（如边框必须有宽度） | `add_border(img, 10)` → 宽度 10（按顺序传值）                     |
| 关键字参数 | 可选选项（指定名称修改）   | `add_border(img, color=(0,0,0))` → 黑色边框[[py-缺省参数(None)]] |
| 默认参数  | 预设选项（不调就用默认值）  | 不传 `width` 时，自动用默认值 5                                    |
|       |                |                                                          |
> 📌 就像 PS 动作里的 “可调选项”
