### 二、集合（Set）练习
集合核心特点 ：无序、不可重复、元素必须可哈希 （如数字、字符串、元组），适合去重和集合运算。

#### 练习 1：创建集合与去重

```python
# 1. 创建集合（用{}或set()，空集合必须用set()）
num_set = {1, 2, 3, 4, 4, 5}  # 自动去重
str_set = set(["apple", "banana", "apple"])  # 从列表转换（去重）
empty_set = set()  # 空集合（不能用{}，{}是字典）

print("数字集合（去重后）：", num_set)
print("字符串集合（去重后）：", str_set)
print("空集合：", empty_set, type(empty_set))
```

#### 练习 2：集合的添加、删除操作

```python
s = {1, 2, 3}

# 1. 添加元素（add()单个，update()多个）
s.add(4)  # 添加单个元素
s.update([5, 6])  # 添加多个元素（可传列表、元组等）
print("添加后：", s)

# 2. 删除元素（remove()不存在会报错，discard()不会报错）
s.remove(3)  # 删除存在的元素
s.discard(10)  # 删除不存在的元素（无报错）
print("删除后：", s)

# 3. 随机删除一个元素（pop()，因无序，结果不确定）
removed = s.pop()
print("随机删除的元素：", removed, "剩余集合：", s)
```

#### 练习 3：集合运算（交集、并集、差集）
```python
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}

# 1. 交集（A和B都有的元素）：& 或 intersection()
print("交集：", A & B)  # 等价于 A.intersection(B)

# 2. 并集（A和B所有元素，去重）：| 或 union()
print("并集：", A | B)  # 等价于 A.union(B)

# 3. 差集（A有而B没有的元素）：- 或 difference()
print("A-B差集：", A - B)  # 等价于 A.difference(B)

# 4. 对称差集（A和B中不重复的元素）：^ 或 symmetric_difference()
print("对称差集：", A ^ B)
```
