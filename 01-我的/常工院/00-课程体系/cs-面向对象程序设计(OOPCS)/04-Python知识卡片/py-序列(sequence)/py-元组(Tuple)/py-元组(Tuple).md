### 定义
- 一种**有序的、不可变的序列类型**，用于存储多个元素的集合

`语法`: 
- 用圆括号 `()` 包裹元素，元素之间用逗号 `,` 分隔（元素类型可混合）
- 或者通过函数 tuple ()
**示例:**
```python
t1 = (1, "a", True)  # 多元素元组
t2 = ()  # 空元组
t3 = (5,)  # 单元素元组（必须加逗号，否则会被解析为整数5）
```
---
**特性**
- `用途`:  在一个变量中保存多个项的值
- `不可变性`:其值和顺序都不可变更
- `重复性`: 允许值重复(Allow Duplicates)
示例:
```python
t1 = (1,2); t[0] = 3 #会系统保存
t2 = ("apple", 3.14, [1,2], (5,6)) #不同类型的元素（数字、字符串、列表、甚至其他元组等）
 up1 = ('Google', 'W3CSchool', 1997, 2020) #类型不一样
tup2 = (1, 2, 3, 4, 5 )
tup3 = "a", "b", "c", "d" # 不需括号也可以
```

### 总结
- 元组是 Python 中一种**不可变的序列类型**，通过逗号分隔元素，常用圆括号包裹，适用于存储`不需要修改的数据`（如配置信息、函数返回的多个值等），
- 因其`不可变性`，比列表更节省内存且更安全。
