### 一、元组（Tuple）练习
元组核心特点：**不可变、有序、可重复**，适合存储固定数据。
#### 练习 1：创建不同类型的元组

```python
# 1. 创建空元组
empty_tuple = ()
print("空元组：", empty_tuple, type(empty_tuple))

# 2. 创建单元素元组（注意必须加逗号）
single_tuple = ("hello",)  # 不加逗号会被识别为字符串
print("单元素元组：", single_tuple, type(single_tuple))

# 3. 创建混合类型元组
mix_tuple = (10, "python", 3.14, True, [1, 2])  # 允许包含可变元素（如列表）
print("混合类型元组：", mix_tuple)
```

#### 练习 2：元组的访问与切片

```python
t = ("a", "b", "c", "d", "e")

# 1. 通过索引访问元素
print("索引1的元素：", t[1])  # 正索引
print("倒数第2个元素：", t[-2])  # 负索引

# 2. 切片（截取部分元素，返回新元组）
print("索引1到3（不含3）：", t[1:3])
print("从索引2到末尾：", t[2:])
print("步长为2的元素：", t[::2])  # 隔一个取一个
```

#### 练习 3：元组的不可变性验证

```python
t = (1, 2, 3)

# 尝试修改元素（会报错，验证不可变性）
try:
    t[0] = 100
except TypeError as e:
    print("修改元素报错：", e)

# 尝试删除元素（会报错）
try:
    del t[1]
except TypeError as e:
    print("删除元素报错：", e)

# 但元组中的可变元素（如列表）可以修改内部内容
t_with_list = (1, [2, 3], 4)
t_with_list[1].append(5)  # 修改列表内部
print("修改元组中的列表后：", t_with_list)
```
