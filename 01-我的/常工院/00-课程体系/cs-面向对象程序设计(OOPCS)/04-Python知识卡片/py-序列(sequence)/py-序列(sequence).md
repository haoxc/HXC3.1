## 概述

序列类型是 Python 中一类**元素有序排列**的数据结构
核心共性：
- **核心特性**：
    1. **有序性**：元素的位置固定，顺序由插入 / 定义时的顺序决定（与集合 “无序” 特性对立）。
    2. **可索引**：每个元素对应唯一的位置编号（索引），可通过索引直接访问元素。
    3. **可切片**：支持获取子序列（通过指定起始和结束索引）。
    4. **可迭代**：可通过循环（如`for`循环）依次访问所有元素。
## 常见类型
- **列表（list）**：`[1, "a", True]`
	- 可变（可修改、新增、删除元素），元素类型可混合
- **元组（tuple）**：`(1, "a", True)`
    - 不可变（定义后无法修改元素），元素类型可混合。
- **字符串（str）**：`"hello"`
	- 不可变（本质是字符的序列），元素为单个字符。
- **范围（range）**：`range(0, 5)`
    - 不可变，生成整数序列（常用于循环）。

## 操作
### 索引 (index) 
- 序列的`索引`是元素的位置编号，用于精准访问单个元素。
- - **正索引**：从序列第一个元素开始，编号为 `0`，依次递增（如第 1 个元素索引为`0`，第 2 个为`1`，以此类推）。

### 切片（Slice）：获取子序列
切片用于从序列中提取连续的子序列，语法为：
`序列[start:end:step]`
- `start`：起始索引（包含该位置元素，默认从`0`开始）。
- `end`：结束索引（不包含该位置元素，默认到序列末尾）。
- `step`：步长（每次取元素的间隔，默认`1`，可为负数表示反向切片）
**示例:**
```python
s = "abcdef"
print(s[1:4])   # 从索引1到3（end=4不含）："bcd"
print(s[:3])    # 从开头到索引2："abc"
print(s[3:])    # 从索引3到末尾："def"
print(s[::2])   # 步长2，取0、2、4索引："ace"
print(s[::-1])  # 步长-1（反向）："fedcba"（反转序列）
```

### 可变与不可变序列的区别
- **可变序列**（如列表）：可直接修改元素、新增 / 删除元素，操作后序列地址不变。
    例：`lst = [1,2]; lst[0] = 0` → `lst` 变为 `[0,2]`。
- **不可变序列**（如元组、字符串）：无法直接修改元素，修改需创建新序列。
    例：`s = "abc"; s[0] = "x"` 会报错；需通过拼接实现：`s = "x" + s[1:]` → `"xbc"`

## 应用场景
- 存储有序数据（如用户列表、时间序列、步骤流程）。
- 通过索引快速访问指定位置元素（如获取第 1 名成绩、最后一条日志）。
- 利用切片处理批量数据（如截取字符串前 5 位、获取列表前半部分）。
- 作为迭代对象，用于循环遍历（如`for item in 序列: ...`）。
