### 身份运算
- **身份运算**用于判断**两个变量是否指向同一个内存对象**（即它们的内存地址是否完全相同），核心运算符是 `is` 和 `is not`

### 身份运算符的基本用法

| 运算符      | 描述                      | 示例           | 结果                     |
| -------- | ----------------------- | ------------ | ---------------------- |
| `is`     | 判断两个变量是否指向同一个对象（内存地址相同） | `a is b`     | 若相同则返回`True`，否则`False` |
| `is not` | 判断两个变量是否指向不同对象（内存地址不同）  | `a is not b` | 若不同则返回`True`，否则`False` |

### 二、核心逻辑：比较的是 “内存地址”

Python 中，每个对象（如数字、字符串、列表等）都有唯一的内存地址，可用 `id()` 函数获取（返回一个整数，表示对象的内存标识）。

身份运算的本质是比较两个变量对应的 `id()` 结果是否相等：

- `a is b` 等价于 `id(a) == id(b)`
- `a is not b` 等价于 `id(a) != id(b)`

> 在Python生态中， 对象都有一个id，等同于我们的身份证号码；
