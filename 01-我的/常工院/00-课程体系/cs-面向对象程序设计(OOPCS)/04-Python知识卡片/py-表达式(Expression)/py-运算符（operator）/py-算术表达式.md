## 概述
-  **算术表达式**是由**操作数**（如数字、变量）和**算术运算符**（如加、减、乘、除）按照一定规则组合而成的`式子`，用于表示`数值计算`过程，最终会得到一个具体的数值结果。它是数学计算和编程中处理数值运算的基础。
- 计算后是一个 `数值型` 变量 
---

## 基本构成

```
(10 - 2×3) ÷ 4 + 5²
```
算术表达式的核心组成包括：
1. **操作数**：参与运算的数值或代表数值的符号（如[[py-变量认知|变量]]、[[py-常量|常量]]）。
    - 例如：`3`（整数）、`2.5`（浮点数）、`x`（变量）、`π`（常量）。
2. **算术运算符**：规定操作数之间的运算关系（如加、减、乘、除等）。

---
## 运算符

| 运算符  | 名称      | 作用说明                             | 数学示例          | Python 示例              | 结果         |
| ---- | ------- | -------------------------------- | ------------- | ---------------------- | ---------- |
| `+`  | 加法      | 两个数相加                            | `3 + 5`       | `3 + 5`                | `8`        |
| `-`  | 减法      | 第一个数减去第二个数；或表示负数（单目运算符）          | `10 - 4`；`-3` | `10 - 4`；`-3`          | `6`；`-3`   |
| `*`  | 乘法      | 两个数相乘（数学中可省略，编程中必须显式写出）          | `2 × 6`       | `2 * 6`                | `12`       |
| `/`  | 除法      | 第一个数除以第二个数（数学中结果可为分数，编程中通常返回浮点数） | `8 ÷ 3`       | `8 / 3`                | `2.666...` |
| `//` | 整除（地板除） | 除法后取不大于结果的最大整数（仅编程中常用）           | -             | `8 // 3`（整数）；`-8 // 3` | `2`；`-3`   |
| `%`  | 取余（模运算） | 除法后返回余数（结果符号与除数一致，仅编程中常用）        | -             | `8 % 3`；`-8 % 3`       | `2`；`1`    |
| `**` | 幂运算     | 第一个数的第二个数次方（数学中为`aⁿ`，编程中用`**`）   | `2³`          | `2 ** 3`               | `8`        |

## 运算规则
算术表达式的计算顺序遵循**优先级**和**结合性**规则，确保结果唯一：
- **优先级**（从高到低）：
    - 括号 `()` 优先（可改变默认顺序）；
    - 幂运算 `**`；
    - 单目运算符（如负数 `-x`）；
    - 乘、除、取余 `*`、`/`、`//`、`%`；
    - 加、减 `+`、`-`。
-  **结合性**（同一优先级的运算顺序）：
    - 大多数运算符是**左结合**（从左到右计算）：如 `8 - 3 - 2` 等价于 `(8 - 3) - 2 = 3`；
    - 幂运算 `**` 是**右结合**（从右到左计算）：如 `2** 3 **2` 等价于 `2** (3 **2) = 2^9 = 512`。

## 总结
算术[[py-表达式(Expression)|表达式]] 是数值计算的基础工具，无论是数学问题还是编程中的数值处理，都需要遵循其运算规则以得到正确结果
