## 概述
- **描述**
	- **关系表达式**（也称为 “**比较表达式**”）是用于判断两个操作数（如数值、变量、表达式）之间关系的表达式，其结果是一个**布尔值**（`True` 或 `False`，即 “真” 或 “假”）。
	- 它是编程中实现 `逻辑判断` 的基础，广泛用于[[py-条件语句|条件语句]]（如 `if`）、循环控制等场景。
- **作用**
	- 判断两个值的关系是否成立，例如 “a 是否等于 b”“x 是否大于 y” 等，最终返回 “成立（`True`）” 或 “不成立（`False`）”。
---
## 运算符
关系表达式通过**关系运算符**连接两个操作数，常见运算符如下

| 运算符 | 描述                     | 实例                 |
| --- | ---------------------- | ------------------ |
| ==  | 等于 -- 比较对象是否相等         | (a == b) 返回 False。 |
| !=  | 不等于 -- 比较两个对象是否不相等     | (a != b) 返回 True.  |
| >   | 大于 -- 返回 x 是否大于 y      | (a > b) 返回 False。  |
| <   | 小于 -- 返回 x 是否小于 y。     | (a < b) 返回 True。   |
| >=  | 大于等于 -- 返回 x 是否大于等于 y。 | (a >= b) 返回 False。 |
| <=  | 小于等于 -- 返回 x 是否小于等于 y。 | (a <= b) 返回 True。  |

---
## 运算规则与特性
### 1. 操作数类型
关系运算符两侧的操作数可以是数值（整数、浮点数）、字符串、变量等，但需注意类型兼容性。
**例如**：
- 数值之间可直接比较（如 `3 > 2.5` → `True`）；
- 字符串比较按 Unicode 编码顺序（如 `'a' < 'b'` → `True`，因 'a' 的编码值小于 'b'）；
- 不同类型（如数值与字符串）比较在 Python 3 中会报错（如 `3 == '3'` → `False`，但 `3 > '3'` 会报错）。

###  2. 与算术表达式结合

关系运算符的优先级低于**算术运算符** ，因此表达式中若包含算术运算，会先计算算术部分，再进行关系判断。
**示例:** 
```python
# 先算 2+3=5，再判断 5 > 4 → True
print(2 + 3 > 4)  # 输出：True

# 先算 10//3=3，再判断 3 == 3 → True
print(10 // 3 == 3)  # 输出：True
```

### 3. 连续比较（链式比较） 
多个关系表达式可连续书写，表示 “`且`” 的逻辑关系（等价于用 `and` 连接）。 #关系表达式/连续比较
**示例**:
```python
# 判断 x 是否在 1~10 之间（1 ≤ x ≤ 10）
x = 5
print(1 <= x <= 10)  # 输出：True（等价于 1<=x and x<=10）
y = 12
print(1 <= y <= 10)  # 输出：False
```
