## 概述
- **描述**
	- 生成一系列连续的整数
- **作用**
	- 常用于控制 for 循环的次数或生成有规律的编号（如动画帧编号、图片序号)
### 基本文法

| 语法格式                       | 含义                              | 生成的序列                                |
| -------------------------- | ------------------------------- | ------------------------------------ |
| `range(stop)`              | 从 0 开始，到 stop-1 结束（默认步长 1）      | 0, 1, 2, ..., stop-1                 |
| `range(start, stop)`       | 从 start 开始，到 stop-1 结束（步长 1）    | start, start+1, ..., stop-1          |
| `range(start, stop, step)` | 从 start 开始，每次加 step，到小于 stop 为止 | start, start+step, start+2*step, ... |
- **参数说明**
  

- 3 种形式
	- range(*结束数字*)
		- 示例: range(5) 生成 0..4
	- range(*开始数字*,*结束数字*)
		- 示例: range(5,9) 生成 5,6,7,8
	- range(*开始数字*,*结束数字*,*步幅*)
		- 示例: range(0,10,3) 生成 0,3,6,9
