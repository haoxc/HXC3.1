### 命名规则：创作的规范
- 创作的规范

```
# 正确命名
flower_size = 150
background_color = "#FFFFFF"
# 错误命名（艺术生常犯的错误）
flower_size1 = 150
back_color = "#FFFFFF"
```

**艺术生视角**：
- 以字母或下划线开头
- 不要使用中文或拼音（如`背景色`）
- 用英文描述，保持简洁  

> 💡 重点： **见名知意**； 变量名要能让人一眼看出它的作用
