在 Python 字符串中，“特殊字母” 通常指由反斜杠 `\` 与字母（或特定符号）组成的**转义序列**，它们在字符串中不表示字面意义，而是用于表示无法直接输入的特殊字符（如换行、制表符）或转义本身有特殊含义的符号（如引号）。以下是常见的字符串转义序列表格：

| 转义序列 | 含义（特殊功能）           | 示例字符串              | 输出 / 效果（print 打印时） | 备注                  |
| ---- | ------------------ | ------------------ | ------------------ | ------------------- |
| `\n` | 换行符                | `"Hello\nWorld"`   | Hello<br><br>World | 光标移至下一行开头           |
| `\t` | 水平制表符（Tab，约 4 个空格） | `"Name:\tAlice"`   | Name: Alice        | 用于对齐文本              |
| `\r` | 回车符                | `"abc\r123"`       | 123c               | 光标移至当前行开头，覆盖后续字符    |
| `\b` | 退格符（删除前一个字符）       | `"abc\bd"`         | abd                | 相当于键盘 Backspace 键   |
| `\f` | 换页符                | `"page1\fpage2"`   | 打印时换页（终端中通常无明显效果）  | 多用于文档打印场景           |
| `\v` | 垂直制表符              | `"line1\vline2"`   | line1<br><br>line2 | 垂直方向对齐（较少使用）        |
| `\'` | 单引号                | `'It\'s a cat'`    | It's a cat         | 在单引号字符串中表示单引号（避免闭合） |
| `\"` | 双引号                | `"He said \"Hi\""` | He said "Hi"       | 在双引号字符串中表示双引号（避免闭合） |
| `\\` | 反斜杠本身              | `"Path: C:\\user"` | Path: C:\user      | 转义反斜杠，避免被解析为转义序列开头  |

### 补充说明：

1. **原始字符串（Raw String）**：在字符串前加 `r` 或 `R` 可禁用转义，转义序列会被当作普通字符。例如：
    `print(r"Hello\nWorld")` 输出 `Hello\nWorld`（`\n` 不换行）。
2. **特殊场景**：转义序列仅在**普通字符串**（单 / 双引号包裹）中生效，三引号字符串（`'''` 或 `"""`）中也支持转义，但通常用于多行文本时可直接换行，无需 `\n`。
