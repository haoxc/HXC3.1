### 概念
- **描述**
	- **字符串变量**是指用来存储**字符串（string）** 类型数据的 [[py-变量(Variables)]]。字符串是由字符（如字母、数字、符号等）组成的序列，是编程中最常用的数据类型之一，用于表示文本信息
- *特性*
	- 字符串一旦定义后值不可改变, 因此字符串值是一个 `不可变量`;
	- Python 没有单独的字符类型，一个字符就是长度为 1 的字符串
	- 反斜杠可以用来转义，使用 `r `可以让反斜杠不发生转义。 如 **r"this is a line with \n"** 则 \n 会显示，并不是换行。 ([[py-特殊字符]])
- 类型
	- type()返回值:`str`
### 如何定义
``` python
 变量名 = 字符串字面量
```

### 使用场景

#### 基本形式 
```python
#定义
name1  = "张晓明" #双引号形式
name2  = `张晓明` #单引号形式
name1[0] = "王"  #报错：'str' object does not support item assignment
```
> 📢 
> - 单引号 `(')`, 双引号 `(")` 使用完全相同
> - 字符串一旦创建，其内容不能被*直接修改*（但可以重新赋值）

#### 转义形式
```python
name  = `这是一行文字.\n` #转移符号
```

### 操作
#### 字符串拼接 (`+`)
**可拼接与重复**：通过运算符可对字符串变量进行操作：

```python
a = "Hello"
b = "World"
print(a + " " + b)  # 拼接：Hello World
print(a * 3)        # 重复：HelloHelloHello
```


---
#### 索引获取字符 (\[\])
**可索引与切片**：通过索引（从 0 开始）访问单个字符，或通过切片获取子串:
```python
s = "Python"
print(s[0])   # 索引：P（第1个字符）
print(s[2:5]) # 切片：tho（从索引2到4的字符）
```

---
#### 字符串重复(\*)
便捷的运算符重载功能，用于**将字符串重复指定的次数**，生成一个新的字符串
```python
新字符串 = 原字符串 * 整数n
```
- 作用：将`原字符串`重复`n`次，拼接成一个新字符串。
- 要求：`n`必须是整数（正整数、0 或负整数），否则会报错。

```python
# 正整数n：重复n次
s = "abc"
print(s * 3)  # 输出：abcabcabc（"abc"重复3次）

# 特殊值n=0：返回空字符串
print(s * 0)  # 输出：（空字符串，长度为0）

# 负整数n：同样返回空字符串（无论负数绝对值多大）
print(s * -2)  # 输出：（空字符串）

# 多行字符串也支持*操作
line = "=" * 20  # 重复20次"="
print(line)  # 输出：====================（20个等号）

# 结合变量使用
star = "*"
triangle = star * 1 + "\n" + star * 2 + "\n" + star * 3
print(triangle)
# 输出：
# *
# **
# ***
```
>  💥 `*` 右侧必须是整数，否则会抛出 `TypeError`

---

## 总结
字符串变量的创建灵活多样，可适应单行、多行、特殊字符等场景；而 `*` 操作则提供了一种简洁的方式重复字符串，在格式化输出、快速生成重复文本时非常实用.
