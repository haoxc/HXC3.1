| 转义字符 | 含义说明                           | 示例                                                      |
| ---- | ------------------------------ | ------------------------------------------------------- |
| `\n` | **换行符**，将光标移到下一行开头             | `print("第一行\n第二行")` 输出两行内容                              |
| `\t` | **水平制表符**（Tab 键效果，通常相当于 4 个空格） | `print("姓名\t年龄\t性别")` 输出对齐的列                            |
| `\\` | 表示一个反斜杠本身（避免被解析为转义符）           | `print("路径：C:\\Users\\文档")` 输出包含反斜杠的路径                  |
| `\'` | 表示单引号                          | `print('He said \'Hello\'')` 输出包含单引号的字符串                |
| `\"` | 表示双引号                          | `print("She said \"Hi\"")` 输出包含双引号的字符串                  |
| `\r` | 回车符，将光标移到当前行开头（会覆盖原有内容）        | `print("abc\r123")` 输出 `123c`（`123` 覆盖了 `abc` 的前 3 个字符） |
| `\b` | 退格符（删除前一个字符）                   | `print("abcd\b")` 输出 `abc`（删除了最后一个 `d`）                 |
| `\f` | 换页符（较少使用，主要用于打印机）              | -                                                       |
| `\v` | 垂直制表符（较少使用）                    | -                                                       |
