## 命名空间
- 命名空间是一个**映射关系**，将名字（如变量名、函数名）映射到对象
- **特点**
	- 同一个名字在不同命名空间中可以代表不同对象。
	- 命名空间在 Python 中用字典实现（`{'name': object}`）

> [!NOTE] 如何理解
> - “名字” 的 “地盘”—— 变量名的管理系统
> - 代码里的 “名字”（比如变量名、函数名，像 “图片路径”“动画帧率”）也有 **“归属地**”—— 这个 “归属地” 就是命名空间
> - 避免 “同名不同用” 的混乱，就像你不会把 不同图层中的素材混在一起管理。
### 例子：

```
# math 模块有自己的命名空间
import math
print(math.sqrt(16))  # 使用 math 命名空间中的 sqrt

# 你也可以定义自己的 sqrt（不会冲突！）
def sqrt(x):
    return x ** 0.5

print(sqrt(16))  # 使用你自己的命名空间中的 sqrt
```

✅ **关键**：`math.sqrt` 和 `sqrt` 是两个不同命名空间中的同名函数，互不干扰。
