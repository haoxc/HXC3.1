## 概述
- **格式化字符串**是指将变量、表达式或值插入到字符串的指定位置，生成动态文本的过程。
- 它解决了字符串与变量拼接时的繁琐问题，让文本生成更灵活、可读性更强。

## 方法

###   % 格式化
- Python 早期的格式化方式，通过 `%` 作为占位符，将变量替换到字符串中。

#### 基本语法
```python
格式化字符串 % (变量1, 变量2, ...)
```
- 字符串中的`%`后接**格式符**（如`%s`、`%d`），表示要插入的变量类型。
- 右侧的变量需用`括号包裹`，多个变量用`逗号分隔`。
#### 常用格式符

| 格式符  | 含义                   | 示例                           |
| ---- | -------------------- | ---------------------------- |
| `%s` | 字符串（可接收任何类型，自动转为字符串） | `%s` 可接收字符串、数字、列表等           |
| `%d` | 整数（仅接收整数，否则报错）       | `%d` 接收 `123`，不接收 `123.45`   |
| `%f` | 浮点数（默认保留 6 位小数）      | `%f` 接收 `3.14` 输出 `3.140000` |
| `%x` | 整数的十六进制表示（小写）        | `%x` 接收 `255` 输出 `ff`        |
#### **示例**
```python
# 单个变量
name = "小明"
print("姓名：%s" % name)  # 输出：姓名：小明

# 多个变量
age = 18
score = 95.5
print("%s今年%d岁，成绩是%f分" % (name, age, score))  
# 输出：小明今年18岁，成绩是95.500000分

# 格式化浮点数（保留1位小数）
print("成绩：%.1f分" % score)  # 输出：成绩：95.5分

# 通过字典传入（用%(key)s指定键）
info = {"name": "小红", "age": 20}
print("姓名：%(name)s，年龄：%(age)d" % info)  
# 输出：姓名：小红，年龄：20
```

### str. format 方法
引入的`format()`方法，通过`{}`作为占位符，比`%`格式化更灵活，支持位置、名称、属性等多种引用方式。

#### 基本语法
```python
"格式化字符串".format(变量1, 变量2, ...)
```
#### 常用方法
#####  按位置引用
-   `{n}` 表示引用第 `n` 个参数（从 0 开始）：

```python
name = "小李"
age = 22
print("姓名：{0}，年龄：{1}，再次打印姓名：{0}".format(name, age))  
# 输出：姓名：小李，年龄：22，再次打印姓名：小李
```
##### 名称引用
`{key}` 表示引用`format()`中指定`key=`的参数：
```python
print("姓名：{name}，年龄：{age}".format(name="小张", age=19))  
# 输出：姓名：小张，年龄：19
```
##### 格式化数字 / 字符串
在`{}`中添加`:`和格式规则，可控制对齐、填充、精度等：
```python
# 浮点数保留2位小数
pi = 3.1415926
print("圆周率：{:.2f}".format(pi))  # 输出：圆周率：3.14

# 整数补0至5位（不足的前面补0）
num = 123
print("补0后：{:05d}".format(num))  # 输出：补0后：00123

# 字符串居中对齐（总长度20，两侧用-填充）
title = "标题"
print("{:-^20}".format(title))  # 输出：--------标题--------
```

### f-string（Python 3.6+，推荐）
f-string（格式化字符串字面值）是最简洁高效的方式，在字符串前加 `f` 或 `F`，直接在 `{}` 中写入变量或表达式。

#### 基本语法
```python
f"字符串内容{变量/表达式}"
```

#### 示例
##### 直接引用变量
```python
name = "小王"
age = 25
print(f"姓名：{name}，年龄：{age}")  
# 输出：姓名：小王，年龄：25
```

##### 支持表达式计算
```python
a = 10
b = 20
print(f"{a} + {b} = {a + b}")  # 输出：10 + 20 = 30

# 调用函数
def get_greeting():
    return "你好"
print(f"{get_greeting()}，{name}")  # 输出：你好，小王
```
##### 格式化规则（与 `format()` 一致）
```python
# 浮点数保留3位小数
price = 99.9
print(f"价格：{price:.3f}元")  # 输出：价格：99.900元

# 百分比格式（自动乘以100并加%）
rate = 0.85
print(f"通过率：{rate:.1%}")  # 输出：通过率：85.0%
```

### 三种方法的对比与推荐

| 方法           | 优点                | 缺点             | 适用场景             |
| ------------ | ----------------- | -------------- | ---------------- |
| % 格式化        | 兼容性好（支持 Python 2） | 功能有限，多个变量时可读性差 | 旧代码维护            |
| str.format() | 功能灵活，支持复杂格式化      | 语法略繁琐          | Python 3.5 及以下版本 |
| f-string     | 简洁高效，支持表达式，可读性强   | 需 Python 3.6+  | 新代码（推荐优先使用）      |
|              |                   |                |                  |
|              |                   |                |                  |

总结：日常开发中，优先使用**f-string**，因其简洁且性能优异；若需兼容旧版本，可使用`str.format()`；`%`格式化尽量避免在新代码中使用。
