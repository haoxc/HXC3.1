### 先搞懂UE

- **UE-是什么**，能做什么？
    - Epic Games 开发的全能型实时 3D 创作工具
    - 广泛用于：
        - 游戏开发（主机 / PC / 手游 / VR/AR）；
        - 影视动画（实时渲染短片、虚拟制片）；
        - 建筑可视化（室内外漫游、方案展示）；
        - 仿真训练（医疗、军事、工业模拟）等。
- **UE-优势**
    - 可视化编程（蓝图）
    - 强大的宣发能力（Lumen)
    - 完善的资源管线
### 学习路径
- 入门（熟悉引擎基础）
    - **目标**: 创建简单场景， 理解引擎界面和基本操作
    - 课题
        - UE环境准备
            - 安装和环境配置
            - [UE-编辑器界面](https://3willartlab.feishu.cn/wiki/MJBmwrZwpiw8BHkOmEQcZlH4nNc)
        - 熟悉引擎界面
            - 项目创建（模版项目区别）
            - 试图操作（平移， 旋转， 缩放试图）， 快捷键：wasd
            - 资源管理
                - 导入/创建模型，材质、蓝图
                - 学习用“文件夹+ 标签” 分类资源
            - 关卡基础
                - 添加Actor，调整Transform， 使用等灯光和相机
        - 核心技能： 蓝图和资环制作
            - 蓝图系统（Blueprints）
                - 蓝图基础
                    - 理解蓝图类（Blueprint Class）、函数（Function）、事件（Event）
                - 核心逻辑
                    - 常用事件
                        - `Event BeginPlay`初始化、`Event Tick`每帧更新
                    - 分支
                    - 循环
                    - 数组
                - 交互案例
                    - 实现角色移动（绑定键盘 / 手柄输入）、物体拾取、开关门、触发对话等简单交互。
            - 材质渲染
                - 材质用途
                    - 决定场景的视觉风格（写实、卡通、科幻等
                - 材质基础
                    - 材质编辑器（节点逻辑）
                    - 材质实例
                - 核心节点
                    - `Texture Sample`（纹理采样）
                    - `Multiply`（颜色混合）
                    - `Normal Map`（法线贴图，增加细节）`Metallic/Roughness`（金属 / 粗糙度，控制质感）
                - 进阶
                    - UELumen（全局光照）
                    - `Nanite`（虚拟几何体）
                        
            - 动画与角色
                - 基础
                    - 导入骨骼动画资源（FBX格式）
                    - 动画蓝图控制角色状态
                    - 理解 “动画蒙太奇（Animation Montage）” 实现攻击、技能等片段动画
### 学习建议

1. “边学边练” 是核心：看教程时，每学一个知识点就立刻在引擎里复现（比如学完 “蓝图变量”，就创建一个变量并在场景中显示它的值）；
    
2. 从 “抄作业” 开始：先用官方模板拆解逻辑（比如打开 “第三人称模板”，看角色移动的蓝图是如何绑定输入的），再尝试修改参数、添加新功能；
    
3. 聚焦一个方向：初期不要贪多，先搞定 “蓝图 + 材质 + 简单关卡”，再分方向（游戏 / 影视 / 建筑）深入；
    
4. 接受 “卡壳”：遇到问题先查官方文档或搜索（关键词 +“UE5”），多数问题已有解决方案。
