# 数媒系学生游戏开发课程学习路线规划（96学时）

---

## 一、课程体系总览

### 📋 两门课程定位

|课程|学时|定位|核心目标|前置关系|
|---|---|---|---|---|
|**游戏设计基础**|32学时|设计思维+原型验证|掌握游戏设计理论，完成可玩原型|无前置|
|**游戏引擎**|64学时|技术实现+作品产出|掌握UE5工作流，完成作品集项目|建议先修游戏设计基础|

## 二、课程一：游戏设计基础（32学时）

### 📚 课程目标

> 培养学生游戏设计思维能力，掌握核心设计方法论，能够独立完成游戏原型设计与验证

### 📖 详细教学大纲

| 模块           |    学时    | 教学内容                                                                 | 实践任务            | 资源链接                                                                                           |
| ------------ | :------: | -------------------------------------------------------------------- | --------------- | ---------------------------------------------------------------------------------------------- |
| **模块1：游戏理论** | **8学时**  |                                                                      |                 |                                                                                                |
| 1.1          |    2h    | 游戏定义与分类  <br>• 游戏本质与核心要素  <br>• 游戏类型详解（RPG/FPS/SLG等）  <br>• 游戏产业发展趋势 | 分析3款不同类型游戏的核心玩法 | [游戏设计基础理论](https://dev.epicgames.com/community/learning/courses/896b/zh-cn)                    |
| 1.2          |    2h    | 玩家心理学  <br>• 心流理论  <br>• 玩家类型分类（Bartle模型）  <br>• 动机与奖励机制             | 设计玩家动机分析表       | [玩家体验设计](https://www.unrealengine.com/zh-CN/onlinelearning)                                    |
| 1.3          |    2h    | 游戏叙事设计  <br>• 叙事结构（三幕式）  <br>• 环境叙事  <br>• 角色塑造                      | 撰写500字游戏故事大纲    | [叙事设计教程](https://dev.epicgames.com/community/learning?categories=world-creation)               |
| 1.4          |    2h    | 游戏美学与风格  <br>• 视觉风格定位  <br>• 音画配合  <br>• 风格参考收集                      | 制作游戏风格情绪板       | [ArtStation参考](https://www.artstation.com/)                                                    |
| **模块2：机制设计** | **12学时** |                                                                      |                 |                                                                                                |
| 2.1          |    3h    | 核心玩法设计  <br>• MDA框架（机制-动态-美学）  <br>• 核心循环设计  <br>• 玩法创新方法            | 设计游戏核心循环图       | [MDA框架详解](https://dev.epicgames.com/community/learning/tutorials/523d/zh-cn)                   |
| 2.2          |    3h    | 关卡设计基础  <br>• 关卡节奏控制  <br>• 引导设计技巧  <br>• 难度曲线设计                     | 绘制关卡布局草图        | [关卡设计教程](https://dev.epicgames.com/community/learning?categories=world-creation)               |
| 2.3          |    3h    | 系统设计  <br>• 经济系统设计  <br>• 成长系统设计  <br>• 战斗系统设计                       | 设计游戏系统架构图       | [系统设计案例](https://dev.epicgames.com/community/learning/courses/4392/zh-cn)                      |
| 2.4          |    3h    | UI/UX设计  <br>• 界面布局原则  <br>• 交互反馈设计  <br>• 信息层级设计                    | 绘制UI线框图         | [UMG UI教程](https://dev.epicgames.com/community/learning/courses/1428/zh-cn)                    |
| **模块3：原型制作** | **12学时** |                                                                      |                 |                                                                                                |
| 3.1          |    3h    | 原型工具使用  <br>• 纸面原型  <br>• Figma/墨刀原型  <br>• 简易数字原型                   | 制作纸面原型          | [原型设计工具](https://www.figma.com/)                                                               |
| 3.2          |    3h    | 可玩原型开发  <br>• 最小可行产品(MVP)概念  <br>• 核心功能实现  <br>• 快速迭代方法              | 完成可玩原型V1.0      | [快速原型方法](https://dev.epicgames.com/community/learning/courses/1737/zh-cn)                      |
| 3.3          |    3h    | 测试与反馈  <br>• 用户测试方法  <br>• 数据收集分析  <br>• 反馈迭代优化                      | 组织5人测试并收集反馈     | [游戏测试指南](https://dev.epicgames.com/documentation/zh-cn/unreal-engine/testing-and-optimization) |
| 3.4          |    3h    | 设计文档撰写  <br>• GDD结构规范  <br>• 视觉化表达  <br>• 版本管理                       | 完成完整GDD文档       | [GDD模板下载](https://dev.epicgames.com/documentation/zh-cn/unreal-engine/project-settings)        |

|**设计文档**|20%|完整GDD文档（20页+）|
|**课堂参与**|10%|讨论、展示、互评|

### 📋 1. 教育学依据

|依据|理论来源|应用说明|
|---|---|---|
|**布鲁姆认知分类**|Bloom's Taxonomy|从记忆理解→应用分析→评价创造，两门课程形成完整认知链条|
|**建构主义学习理论**|Piaget/Vygotsky|以学生为中心，通过项目实践建构知识体系|
|**70-20-10法则**|成人学习理论|70%实践(67h)+20%反馈(19h)+10%理论(10h)|
|**最近发展区**|Vygotsky ZPD|每阶段设置略高于当前水平的挑战任务|
|**项目驱动学习**|PBL教学法|以最终作品集为导向组织教学内容|

|**课程衔接**|避免内容重复与断层|游戏设计基础产出直接作为游戏引擎输入|

|建议项|具体措施|
|---|---|
|**开课顺序**|建议游戏设计基础先修，或两门课程同学期并行|

|**作品评审**|邀请行业专家参与期末作品评审|
