## 术语理解：宜家风格 (IKEA manual style)

“宜家风格”通常指宜家（IKEA）组装说明书([[指南]])所采用的无文字、纯图示、高度简约的视觉沟通方式。它通过极简的线条图引导用户完成复杂的任务。![[ikea manual(宜家安装说明书).png|396]]

## 1. 核心含义

- 视觉引导：完全依靠插图表达步骤，不依赖特定语言。
- 低认知负荷：通过拆解极细微的动作，降低理解难度。
- 趣味人物 (IKEA Man)：使用标志性的无脸小人（Fille）演示正确与错误的动作。
- 防错设计：强调关键零件的朝向和对比（如“对”与“错”的打叉对比）。

## 2. 英文表述 (按场景分类)

| 场景   | 推荐词汇                      | 典型用法                                         |
| ---- | ------------------------- | -------------------------------------------- |
| 视觉风格 | Pictorial style           | 纯图画风格：_Instruction in pictorial style._      |
| 设计理念 | Wordless instruction      | 无字说明书：_A wordless instruction manual._       |
| 极简主义 | Minimalist illustration   | 极简插画：_Clean, minimalist illustrations._      |
| 沟通方式 | Universal visual language | 通用视觉语言：_Create a universal visual language._ |
| 操作步骤 | Step-by-step diagram      | 分步图解：_Follow the step-by-step diagrams._     |
| 核心角色 | IKEA Man / Fille          | 宜家小人：_The iconic IKEA Man illustration._     |

---

## 3. 常见搭配短语

- 宜家式说明书: IKEA-style manual
- 视觉叙事: Visual storytelling
- 自解释设计: Self-explanatory design
- 装配图解: Assembly diagrams
