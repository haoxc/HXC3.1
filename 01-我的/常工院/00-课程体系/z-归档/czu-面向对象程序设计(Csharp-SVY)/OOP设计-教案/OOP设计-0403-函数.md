## 概要
- 在 C# 中，函数（在 C# 中通常被称为方法）是一个执行一系列操作并返回结果的代码块。
- 它可以执行操作，然后返回值给调用者，也可以执行一些操作而不返回值（这种方法称为 "`void`" 方法）。
- 方法是`类`或`对象`行为的基本单元，通常在类中定义。


## 定义函数
**函数定义的语法**

```csharp
返回类型  函数名(参数...){
  //在这儿写自己的程序(语句)

  return 返回类型的值;
}

void  函数名(参数...){
  //在这儿写自己的程序(语句)
}

```
 

**定义一个方法**：
```csharp
int Add(int a, int a)
{
    return a + b;  // 返回两个参数的和
}
int c = add(2,3);


```
在上面的例子中，`Add` 是一个方法，它接受两个整数参数并返回它们的和。

**void 方法**：
```csharp
 void DisplayGreeting(string name)
{
    Console.WriteLine("Hello, " + name + "!");
}
```

**调用一个方法**： 你可以通过创建包含该方法的类的实例（如果是非静态方法），或直接通过类名（如果是静态方法）来调用方法。

```csharp
void DisplayGreeting(string name)
{
    Console.WriteLine("Hello, " + name + "!");
}

DisplayGreeting("小明"); //调用
```
