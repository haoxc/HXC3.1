### 控制台输入

- 在 C# 中，如果你想通过[[OOP设计-0202-控制台|控制台]]读取用户输入，可以使用 `Console.ReadLine()` 方法;
- 这个方法会暂停程序执行，等待用户在控制台输入文字后按 Enter 键，然后它会读取用户输入的所有字符直到 *Enter 键*（换行符）为止;
- 并将这个***字符串***返回给程序。

---
### 简单输入
程序提示用户输入一个参数。用户可以输入任何文本，然后按 Enter 键
```csharp
using System;
class Program
{
    static void Main()
    {
        Console.Write("请输入一个参数:");
        // 等待用户输入，然后将输入的内容存储在 'input' 变量中
        string input = Console.ReadLine();
        // 打印出用户输入
        Console.WriteLine("你输入的参数是: " + input);
    }
}
```

### 读取整数

```csharp
using System;
class Program
{
    static void Main()
    {
        Console.WriteLine("请输入一个参数:");
        // 等待用户输入，然后将输入的内容存储在 'input' 变量中
        string strInput = Console.ReadLine();
        int a  = int.Parse(strInput);
        // 打印出用户输入
        Console.WriteLine("你输入的参数是: " +a);
    }
}
```
