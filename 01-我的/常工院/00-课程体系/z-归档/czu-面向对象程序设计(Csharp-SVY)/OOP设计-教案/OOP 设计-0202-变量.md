## 变量声明 
> 又称为: *变量定义*  #语句-statement

在编程中，变量定义是指创建一个变量并为其分配一个名称和可选的初始值
- 选择一个合适的**变量名**
- 指定一变量的**数据类型**
- 分配一个**初始值**

 语法: 
 
```csharp
Type  variable_name = initValue;
```

### 课堂练习-002-字符串

> ` @` 字符串中可以任意换行，换行符及缩进空格都计算在字符串长度之内。 #csharp/练习/0202 

```csharp
//String 
String str = "www.baidu.com";

string path = @"C:\windows"; 
string path = "C:\\windows"; 
```

> 理解赋值 #csharp/练习/0202 

```csharp
string name1 = "小李";
string name2 = name1; // name2现在引用了与name1相同的字符串

name1 = "小张"; // 创建了一个新的字符串，name1现在引用"小张"，但name2仍然引用"小李"
```
