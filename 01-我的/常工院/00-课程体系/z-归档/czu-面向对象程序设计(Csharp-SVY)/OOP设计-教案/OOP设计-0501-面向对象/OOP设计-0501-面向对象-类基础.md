### 定义类
- **文法**: 
```csharp
访问修饰符 class ClassName{
   //类内部成员定义
}
```
其中
- 访问修饰符
	- ![[OOP设计-面向对象-访问修饰符#访问修饰符]]
			

以`Person`为例 :
```csharp
public class  Person{
	//成员(字段)
	//行为
}

```


### 定义成员
- 类的字段或属性,又称为`成员变量,字段`

- **文法**: 
```csharp
访问修饰符 class ClassName{
   //字段(成员)
   访问修饰符 类型 成员名称;
}
```

- **示例**
```csharp
public class Person{
	//成员
	private  string name; //姓名
	private  int    age;  //年龄
	private  int    gender; //性别
	...
}
```

## 定义属性
- 属性（Properties）是类的成员之一，用于提供对类的字段的安全访问。
- 属性可以让你控制字段的`读取`（get）和`写入`（set）操作，从而为数据的访问提供更高层次的封装和保护。

> [!important] 特性
> - **封装**：属性提供了一种机制来获取和设置私有字段的值，而不必直接暴露这些字段。
> - **验证**：在设置属性值时，可以添加逻辑来验证数据。
> - **简化语法**：属性使用与字段相似的语法，但同时提供了方法的灵活性。
	
```csharp
public class Person{
	//属性
	public   string name;  //姓名
	public   int    age;   //年龄
	private  string grade; //年级


	public  Person(string name, int age,string grade){
		this.name= name;
		this.age = age;
		this.grade =grade;
	}
	public  Person(string name, int age){
		this.name= name;
		this.age = age;
		this.grade = "一年级";
	}
```
