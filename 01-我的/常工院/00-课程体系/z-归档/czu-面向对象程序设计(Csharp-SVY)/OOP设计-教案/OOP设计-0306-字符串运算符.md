## 理解值类型

- 在 C# 中，所有的简单类型 `int`, `float`, `double`, `decimal`, `bool`, 和 `char``字符`,`数值`, `字符串`是***值类型***，这意味着一旦创建了,就无法更改它的内容.

>[!warning] 思考题
```csharp
string name = "小明";
string oldName = name;
name = "小明明"; 
//程序执行到此处时, name的值是多少?

```

## 操作符
### 连接操作符
可以使用 `+` 运算符或 `String.Concat` 方法来连接两个或多个字符串。

**示例**: 
```csharp
string firstName = "小";
string lastName = "明";
string fullName = firstName + lastName;  // "小明"
// 或者
string fullNameUsingConcat = String.Concat(firstName, lastName);  // "小明"


```

---
### 格式化字符串

- 可以使用[[OOP设计-0202-格式字符串|字符串插值]]或 `String.Format` 方法。
 
- **示例**: 
```csharp
int age = 30;
string name = "小明";
string info = $"我叫{name}，我今年{age}岁。";
// 或者
string infoUsingFormat = String.Format("我叫{0}，我今年{1}岁。", name, age);


```

---
### 更改大小写
- 使用 `ToUpper` 和 `ToLower` 方法来更改字符串中的字符大小写。
- 在中文中，这个操作并不常见，因为中文字符通常没有大小写之分。但方法仍然适用。

 **示例**:
```csharp
string response = "Hello World!";
string upper = response.ToUpper(); // 输出: "HELLO WORLD!"
string lower = response.ToLower(); // 输出: "hello world!"
```



---
### 修剪字符串
- 使用 `Trim`, `TrimStart`, `TrimEnd` 方法去除字符串开头和/或结尾的空格或指定字符

**示例:**
```csharp
string original = "   extra   space   ";
string trimmed = original.Trim(); // 输出: "extra space"
```
---


---
### 替换字符串内容
使用 `String.Replace` 方法替换字符串中的字符或子字符串。
```csharp
string message = "Hello,world!";
int index = message.IndexOf("world"); // 返回 7, 这是 "world" 开始的位置
bool containsHello = message.Contains("Hello"); // 返回 true
```

---
### 截取字符串
使用 `String.Substring` 方法从字符串中提取子字符串。

```csharp
string phrase = "programming is fun";
string part = phrase.Substring(4, 8); // 输出: "gramming"
```
