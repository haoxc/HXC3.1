## 概要
- 这些表达式使用算术运算符（如 `+`, `-`, `*`, `/`）来***计算数值***。
- 在编程中，算术运算符用于执行各种数学运算;
	- 加法`+`、减法`-`、乘法`*`、除法`/`
	- 取余运算符 `%` 
	- 自增运算符 `++`, 自减运算符 `--`
> #PP-173-算术运算符 

**语法示例:** 
```csharp
int result = 3 + 4; // 3 + 4 是一个算术表达式
```

## 运算符
###  加减乘除
- 在C#中，常见的算术运算符包括：

```csharp
int sum = 5 + 3; // 结果为8
int difference = 10 - 4; // 结果为6
int product = 3 * 4; // 结果为12
double quotient = 12.0 / 4.0; // 结果为3.0
int remainder = 13 % 5; // 结果为3
```
### 自增/自减

-  **自增运算符 `++`**：用于将变量的值增加1;  #csharp/复习/重点 
- **自减运算符 `--`**：用于将变量的值减小1。

### 优先级
C#中的算术运算符具有不同的优先级，这意味着它们在表达式中的计算顺序是有规定的。以下是常见算术运算符的优先级，从高到低：
1. **括号 `()`**：括号用于指定运算的优先级，最高优先级。任何位于括号中的表达式都会首先计算。
2. **自增 `++` 和自减 `--`**：自增和自减运算符具有较高的优先级。它们在其他运算之前执行。
3. **乘法 `*`、除法 `/` 和取余 `%`**：乘法、除法和取余运算符的优先级相同，按从左到右的顺序执行。
4. **加法 `+` 和减法 `-`**：加法和减法运算符的优先级相同，按从左到右的顺序执行。
在表达式中，运算符的优先级决定了它们的计算顺序。如果需要改变默认的计算顺序，可以使用括号来指定运算的优先级。括号内的表达式会首先计算。
