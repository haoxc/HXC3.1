## 标识符(Identifier)
### 标识符要点
- **用途**:
	-  标识符是分配给类型（*类、接口、结构、委托或枚举*）、*成员、变量或命名空间*的`名称`。
- **特点**:
	- 标识符区分大小写
	- 标识符必须以英文字母或`下划线` (`_`,`@`) 构成
		- 首字母***允许***英文字母或`下划线` (`_`,`@`) ,***不允许***数字
		- `@`***只允许***出现在首字母![[Pasted image 20231001153506.png]]

## 语句(Statement)

- *描述*一个`类型`或告诉程序去执行某个操作的一个源代码`指令`;
- 语句有2个类型(简单语句, 块)
	- **简单语句**以一个`分号`结束;
	- **块(Block)** 是有`大括号`包围的0或多条简单语句序列,在文法上相当于一个语句;

## 注释(Comment)
- **目的:** 注释,也成为文档注释,是对程序语句解释,便于他人能够快速理解程序(程序可读性), *编译器会忽略注释的条目*;
- 在 C# 程序中
	- **单行注释** 以`//`作为行的开始
	- **多行注释**以 `/*` 开始，并以字符` */` 终止

### 示例
```csharp
/* 以下代码是声明相关变量 */
int    age; //年龄
string name; //姓名
int    id; //身份证号
...
```
