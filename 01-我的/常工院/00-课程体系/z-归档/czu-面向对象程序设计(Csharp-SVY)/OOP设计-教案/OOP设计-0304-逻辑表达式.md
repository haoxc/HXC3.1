## 逻辑操作符
逻辑运算符用于执行`布尔逻辑运算`，它们通常用于将多个布尔值组合成更复杂的布尔表达式。
在C#中，常见的逻辑运算符包括以下几种：

- **逻辑与 `&&`**：用于检查两个条件是否同时为真。如果两个条件都为真，则整个表达式的结果为真；否则，结果为假。
```csharp
bool condition1 = true;
bool condition2 = false;
bool result = condition1 && condition2; // 结果为false
```
- **逻辑或 `||`**：用于检查两个条件是否至少一个为真。如果其中一个或两个条件为真，则整个表达式的结果为真。
```csharp
bool condition3 = true;
bool condition4 = false;
bool result = condition3 || condition4; // 结果为true
```

- **逻辑非 `!`**：用于对单个条件取反。如果条件为真，则取反后为假，反之亦然。
```csharp
bool condition5 = true;
bool result = !condition5; // 结果为false
```
