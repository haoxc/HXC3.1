## 概要
- 通过 `.` 运算符访问类或结构的成员（如字段，属性，方法）。例如：

```csharp
string name ="HAO xinchang";
int length = name.Length; //方位属性

Console.WriteLine($"变量name的值{name}是{length}"})

```
