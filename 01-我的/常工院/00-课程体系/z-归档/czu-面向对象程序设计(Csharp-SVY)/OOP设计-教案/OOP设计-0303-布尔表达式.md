## 概要
- **关系表达式**是一种比较两个或多个`操作数`并返回一个布尔值（`true` 或 `false`）的表达式
- 这些运算符返回*布尔值*，表示比较的结果是否为真, 因此有成为***布尔表达式***。
- 常见的关系比较运算符：
	-  **等于** `*==*`  - 检查左边的值是否等于右边的值。
	- **不等于 `!=`**：检查左边的值是否不等于右边的值。
	- **大于 `>`**：检查左边的值是否大于右边的值。
	- **小于 `<`**：检查左边的值是否小于右边的值。
	- **大于等于 `>=`**：检查左边的值是否大于或等于右边的值。
	- **小于等于 `<=`**：检查左边的值是否小于或等于右边的值。

> #PP-171-关系比较运算符   #csharp/复习/重点 

 **标准示例**:
```csharp
int a = 10;
int b = 20;

bool isEqual = (a == b);  // false，因为 a 不等于 b
bool isNotEqual = (a != b);  // true，因为 a 不等于 b
bool isGreaterThan = (a > b);  // false，因为 a 不大于 b
bool isLessThan = (a < b);  // true，因为 a 小于 b
bool isGreaterThanOrEqual = (a >= b);  // false，因为 a 小于 b
bool isLessThanOrEqual = (a <= b);  // true，因为 a 小于 b
```


## 浮点型关系运算
- 使用这些运算符时，特别需要注意*浮点数的精确度问题*。
- 由于计算机的表示方式，有时候两个看似相等的浮点数*可能会因为极小的差异而不完全相等*。 
> #csharp/复习/难点/易错  

>[!error] **易错题**
```csharp
double num1 = 0.3 * 3 + 0.1;
double num2 = 1.0;

//输出 num1,和num2的值,并指定小数点20;

Console.WriteLine(num1 == num2); // 这通常会返回 false，因为浮点运算的精度问题。

```
