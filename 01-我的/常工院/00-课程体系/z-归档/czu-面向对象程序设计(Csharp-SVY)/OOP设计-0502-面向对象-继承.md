## 概要
- **继承**是面向对象程序设计中最重要的概念之一;
- **继承**允许我们根据一个类来定义另一个类，这使得**创建和维护**应用程序变得更容易;
- 继承的思想实现了 **属于（IS-A）** 关系;

**示例:**
![[img-oop-继承.png]]
- 哺乳动物 **属于（IS-A）** 动物，
- 狗 **属于（IS-A）** 哺乳动物，
- 因此狗 **属于（IS-A）** 动物

## 定义
- 首先，我们定义一个基类（父类）`Animal`，它包含所有动物共有的特性和行为。
- 然后，我们将创建两个派生类（子类）：`Dog` 和 `Cat`，它们继承自 `Animal` 类，并添加各自特有的行为。

### 基类: Animal
```csharp
public class Animal
{
    public string Name { get; set; }
    public int Age { get; set; }

    public Animal(string name, int age)
    {
        Name = name;
        Age = age;
    }

    public void Eat()
    {
        Console.WriteLine($"{Name} is eating.");
    }

    public void Sleep()
    {
        Console.WriteLine($"{Name} is sleeping.");
    }
}
```

- 这个 `Animal` 类定义了所有动物共有的属性（如`Name`和`Age`）和方法（如`Eat`和`Sleep`）。

### 派生类：Dog
```csharp
public class Dog : Animal
{
    public Dog(string name, int age) : base(name, age)
    {
    }

    public void Bark()
    {
        Console.WriteLine($"{Name} is barking.");
    }
}

```
`Dog` 类继承自 `Animal` 类，并添加了一个特有的方法 `Bark`。

### 派生类：Cat
```csharp
public class Cat : Animal
{
    public Cat(string name, int age) : base(name, age)
    {
    }

    public void Meow()
    {
        Console.WriteLine($"{Name} is meowing.");
    }
}
```

类似地，`Cat` 类继承自 `Animal` 类，并添加了一个特有的方法 `Meow`。


### 使用
```csharp
static void Main(string[] args)
{
	Dog myDog = new Dog("Buddy", 3);
	myDog.Eat();    // 调用继承自 Animal 的方法
	myDog.Bark();   // 调用 Dog 特有的方法

	Cat myCat = new Cat("Whiskers", 2);
	myCat.Sleep();  // 调用继承自 Animal 的方法
	myCat.Meow();   // 调用 Cat 特有的方法
}

```
