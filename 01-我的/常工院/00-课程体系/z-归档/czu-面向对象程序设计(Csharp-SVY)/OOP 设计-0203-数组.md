## 概要
- 在 C# 中，数组是一个*同类型元素*的集合，其元素可以使用`索引`进行访问;
- 数组的`长度`在声明时确定并且在整个生命周期中是*固定的*;

## 语法

### 声明数组
```csharp
dataType[] arrayName;
```
- dataType: 数据类型 
	- ![[OOP-Variable-Type.png]]
- arrayName: 数组名称

### 初始化数组
1. 数组可以在声明时进行初始化：
```csharp
int[] numbers = { 1, 2, 3, 4, 5 };
```

2. 使用 `new` 关键字创建数组，并指定其大小：
```csharp
int[] numbers = new int[5];
```

3. 使用 `new` 关键字创建数组，并指定其大小：
```csharp
int[] numbers = new int[]{1, 2, 3, 4, 5};
```


### 访问数组元素
数组元素可以使用`索引`进行访问。索引从 ***0*** 开始：
```csharp
int firstNumber = numbers[0];  // 获取第一个元素
numbers[4] = 10;               // 设置第五个元素的值为 10

```

### 数组方法和属性
- `Length`：返回数组中元素的总数。
- `Rank`：返回数组的维度。
- `GetValue` 和 `SetValue`：用于从数组获取和设置值。
