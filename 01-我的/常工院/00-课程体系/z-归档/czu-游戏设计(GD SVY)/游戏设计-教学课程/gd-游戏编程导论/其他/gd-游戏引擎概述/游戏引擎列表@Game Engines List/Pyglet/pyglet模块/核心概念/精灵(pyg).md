## 概述
- `来源`: *精灵*是日耳曼神话中所出现的有灵性的生物，它们被描绘成拥有稍长的尖耳、手持弓箭、金发碧眼、高大且与人类体型相似的日耳曼人的形象。也经常用来形容人机警、聪明。
- `角色`: 在游戏场景下,各种设定的角色,也是拟人化的设定设计性格,能力等特征.
- `形式`: 
	- 形象(模型)
	- 能力,魔法(动效)


## 精灵实例
魔兽争霸III 中每一个模型的重做都需要大量的工作
- 角色设定背景:
	- 暗夜精灵是第一支修习魔法的种族，约一万年前，是他们让魔法遍及了整个世界。他们曾拼尽全力将燃烧军团的恶魔逐出此界，但他们美丽的家园也因此支离破碎，而后暗夜精灵便遁身于他们的圣山海加尔，从此隐居避世。重新设计暗夜精灵的单位时真正地抓住了这个*种族的精髓*：大自然是他们永恒的主题和功能。弯曲的树枝和根茎形成了古树，还有许多暗夜精灵单位（例如树妖）都融合了森林动物和精灵的特征。

## 创建精灵
```python
import pyglet

ball_image = pyglet.image.load('ball.png')
ball = pyglet.sprite.Sprite(ball_image, x=50, y=50)

window = pyglet.window.Window()

@window.event
def on_draw():
    ball.draw()

pyglet.app.run()
```
