## 安装
> [Quick Start - Agent TARS](https://agent-tars.com/guide/get-started/quick-start.html)

### 安装 ui-tars
- 安装包
```shell
	pip install ui-tars
```
- 确认
```shell
	pip show ui-tars
```


### 安装 node.js
- [[安装Nodejs(本地)]]

### 安装 agent-tars
安装最新版本：
```bash
npm install @agent-tars/cli@latest -g
```
