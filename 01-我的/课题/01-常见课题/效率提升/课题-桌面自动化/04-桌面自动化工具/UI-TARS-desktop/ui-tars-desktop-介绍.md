# 摘要

UI-TARS-desktop 项目由 ByteDance 维护，是一个开源项目，旨在构建一个多模态人工智能代理栈。该项目的目标是将最新的 AI 模型和代理基础设施连接起来，促进它们之间的`协同工作`。项目使用 Apache-2.0 许可证，并且在 GitHub 上拥有超过 24k 的 star 和 2.3k 的 fork。项目的最新更新包括修复 typo、关闭 agent tars desktop 远程操作器功能、优化会话状态动作设计等。最后一次提交是在 2026 年 1 月 5 日，提交内容是修复 typo。
