## 内容
📋 数字孪生平台组件事件定义表

| 事件类型      | 事件名称 (Event Name)  | 触发源 (Trigger)      | 典型应用场景           | 预期响应动作                           |
| --------- | ------------------ | ------------------ | ---------------- | -------------------------------- |
| **数据/状态** | `onDataUpdate`     | 物联网传感器数据推送         | 仪表盘数值刷新、温度传感器异常  | 实时更新 UI 数值；若超过阈值则触发警报。           |
|           | `onStatusChange`   | 设备开关机、离线、故障        | 产线停机监控、电力开关切换    | 改变模型材质颜色（如：绿色变红色）；弹出状态弹窗。        |
|           | `onThresholdAlarm` | 实时数据达到设定阈值         | 锅炉压力过载、环境烟雾报警    | 触发 3D 场景中的红色闪烁特效；发送系统通知。         |
| **空间/地理** | `onTransform`      | 移动物体坐标更新 (GPS/UWB) | AGV 小车定位、物流车辆追踪  | 驱动 3D 模型平滑移动；实时绘制行驶轨迹轨迹线。        |
|           | `onGeofenceEnter`  | 模型进入预设 3D 空间区域     | 吊装作业进入禁区、人员进入危险区 | 触发区域高亮；联动现场摄像头弹窗。                |
|           | `onCameraFocus`    | 用户点击导航或搜索结果        | 快速定位某台特定变压器      | 相机视角平滑飞行（Fly-to）至目标，并自动调整角度。     |
| **用户交互**  | `onComponentClick` | 用户在 3D 画布点击模型实体    | 查看设备详细参数、查看维修记录  | 展开右侧侧边栏 (Sidebar)；显示关联的视频监控。     |
|           | `onHoverHighlight` | 鼠标悬停在模型上方          | 快速识别设备 ID        | 显示 Hover Tooltip（气泡提示）；模型边缘描边高亮。 |
|           | `onReverseControl` | 用户点击界面控制按钮         | 远程关闭风机、远程重启网关    | 弹出二次确认框；向 MQTT 控制台 下发下行控制指令。     |
| **系统/构建** | `onLoadComplete`   | 场景资源/高精模型加载完毕      | 平台初始化、大屏首屏展示     | 隐藏 Loading 动画；开启实时数据轮询。          |
|           | `onConfigChange`   | 编辑模式下修改组件参数        | 调整设备旋转角度、修改比例尺   | 实时更新模型 Transform 数据；同步保存至后端数据库。  |
|           | `onTimeTravel`     | 拖拽历史回溯时间轴          | 事故调查、历史能耗分析      | 模型与数据切片回滚至指定时间点的历史状态。            |
