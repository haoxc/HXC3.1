## 交互对象 (Interactive Objects)
- **原则**：采用 **大驼峰 (PascalCase)** 命名类与事件，**小驼峰 (camelCase)** 命名属性与方法。

### 场景对象 (SceneObject)
- _类型 (Types)_
    - 地球 (Earth)
    - 图层 (Layer/3DTiles)
    - 初始镜头/摄像机 (Camera)
    - 模型 (Model)
        - 分层模型 (ComplexModel)
    - 设备 (Device)
    - 特效 (Effect)
    - POI (PointOfInterest)
    - 管道 (Pipeline)
- _属性 (Attributes)_
    - **基础信息 (Basic Info)**
        - `id` (全局标识 / UUID)
        - `name` (显示名称 / Display Name)
        - `visible` (可见性 / Visibility)
        - `tags` (标签集 / Tags)
    - **空间信息 (Spatial Info)**
        - `position` (位置 / Position) - _Type: Vector3_
        - `rotation` (角度 / Rotation) - _Type: Euler/Quaternion_
        - `scale` (比例 / Scale) - _Type: Vector3_
    - **镜头信息 (Camera Info)**
        - `viewOffset` (最佳观察偏移 / View Offset)
        - `viewAngle` (最佳观察角度 / View Angle)
    - **视觉信息 (Visual Info)**
        - `renderMode` (材质模式 / Render Mode)
        - `colorTint` (叠加颜色 / Color Tint)
        - `opacity` (透明度 / Opacity)
    - **显示控制 (Display Control)**
        - `isVisible` (是否显示 / Is Visible)
- _操作 (Operations)_
    - `translateTo(position, duration)` (移动 / Move)
    - `rotateTo(rotation, duration)` (旋转 / Rotate)
    - `lookAt(targetId)` (朝向 / Look At)
    - `focus()` (聚焦 / Focus)
    - `pick()` (拾取/选中 / Pick)
    - `setHighlight(color, thickness)` (描边 / Outline)
    - `toggleEffect(effectId)` (触发特效 / Toggle Effect)
- _事件 (Events)_
    - `OnSelected` (选中时)
    - `OnHover` (悬停时)
    - `OnDoubleClicked` (双击时)
    - `OnVisibilityChanged` (显隐改变时)

### 地球 (EarthTerrain)
- _属性 (Attributes)_
    - `mapProvider` (地图供应商 / Map Provider)
    - `mapStyle` (地图样式 / Map Style)
        - _Enumeration: Satellite, Vector, Dark_
    - `clippingRegions` (裁剪区域列表 / Clipping Regions)
- _操作 (Operations)_
    - `addClippingRegion(points)` (新增裁剪 / Add Clipping)
    - `removeClippingRegion(index)` (删除裁剪 / Remove Clipping)
    - `updateClippingRegion(index, points)` (编辑裁剪 / Edit Clipping)
    - `focusGlobal()` (聚焦全球 / Focus Global)
- _事件 (Events)_
    - `OnMapLoaded` (地图加载完成)
    - `OnLevelChanged` (层级改变)
    - `OnSurfaceClicked` (表面点击)

### 图层 (TilesLayer)

- _属性 (Attributes)_
    - `dataSourceUrl` (数据源地址 / Data Source URL)
    - `heightOffset` (高度偏移 / Height Offset)
    - `flattenRegions` (压平/裁剪区域 / Flatten Regions)
- _操作 (Operations)_
    - `refresh()` (刷新 / Refresh)
- _事件 (Events)_
    - `OnTileLoadProgress` (加载进度)
    - `OnLoadComplete` (加载完成)

### 初始镜头/摄像机 (Camera)

- _属性 (Attributes)_
    - **镜头参数 (Lens Params)**
        - `fov` (视场角 / Field of View)
        - `projectionMode` (投影模式 / Projection Mode)
    - **镜头状态 (Camera State)**
        - `initialPose` (初始位姿 / Initial Pose)
        - `currentPose` (当前位姿 / Current Pose)
- _操作 (Operations)_
    - `flyTo(pose, duration)` (飞向 / Fly To)
    - `captureSnapshot()` (截屏 / Capture Snapshot)
    - `reset()` (重置 / Reset)
- _事件 (Events)_
    - `OnMoveStart` (移动开始)
    - `OnMoveEnd` (移动结束)

### 模型 (Model)

- _属性 (Attributes)_
    - **基础/空间/镜头信息** (Inherited)
    - `animationClips` (动画列表 / Animation Clips)
- _操作 (Operations)_
    - `playAnimation(clipName, speed, loop)` (播放动画 / Play Animation)
    - `stopAnimation()` (停止 / Stop)
    - `pauseAnimation()` (暂停 / Pause)
    - `resumeAnimation()` (继续 / Resume)
- _事件 (Events)_
    - `OnModelReady` (模型就绪)
    - `OnAnimFinished` (动画结束)

### 分层模型 (ComplexModel)

- _操作 (Operations)_
    - `setExplodeMode(distance)` (炸楼/退出 / Set Explode Mode)
    - `isolateFloor(floorId)` (隔离/退出 / Isolate Floor)
    - `enterLevelMode()` (进入分层 / Enter Level Mode)
- _事件 (Events)_
    - `OnLevelChanged` (楼层切换)
    - `OnExplodeStateChanged` (炸楼状态变更)

### 设备 (IoTDevice)
- _属性 (Attributes)_
    - **物联网属性 (IoT Props)**
        - `deviceIdentity` (设备标识 / Device Identity)
            - `type` (类型), `vendor` (供应商), `model` (型号)
        - `dataSchema` (数据模型 / Data Schema)
        - `controlInterface` (接口信息 / Control Interface)
- _操作 (Operations)_
    - `sendCommand(key, params)` (发送指令 / Send Command)
    - `syncTelemetry(data)` (同步数据 / Sync Telemetry)
- _事件 (Events)_
    - `OnTelemetryUpdate` (数据更新)
    - `OnStatusChanged` (状态变更)
    - `OnAlertTriggered` (触发告警)
    - `OnCommandAck` (指令反馈)

### 特效 (Effect)

- _类型 (Types)_
    - 环境 (Environmental), 后期 (PostProcess)
- _属性 (Attributes)_
    - `intensity` (强度 / Intensity)
    - `color` (颜色 / Color)
    - `isActive` (开关状态 / Is Active)
- _操作 (Operations)_
    - `toggle(bool)` (开关 / Toggle)
    - `setParam(key, value)` (设置参数 / Set Parameter)
- _事件 (Events)_
    - `OnStateChanged` (状态改变)

### POI (PointOfInterest)

- _属性 (Attributes)_
    - `iconUrl` (图标地址 / Icon URL)
    - `labelContent` (文字内容 / Label Content)
    - `labelStyle` (文字样式 / Label Style)
    - `interactionMode` (交互模式 / Interaction Mode)
        - _Enum: Click, Hover, Auto_
    - `popupConfig` (弹窗配置 / Popup Config)
        - `size` (大小), `offset` (偏移), `embedUrl` (嵌入页面)
- _操作 (Operations)_
    - `triggerInteraction()` (模拟触发 / Trigger Interaction)
- _事件 (Events)_
    - `OnClicked` (点击)
    - `OnPopupOpen` (弹窗打开)

### 面板 (Panel)

- _属性 (Attributes)_
    - `sourceUrl` (页面地址 / Source URL) - _For Web_
    - `componentTree` (组件树 / Component Tree) - _For Native_
- _操作 (Operations)_
    - `reload()` (刷新 / Reload)
    - `postMessage(json)` (发送数据 / Post Message)
    - `getComponents()` (获取组件列表 / Get Components)
- _事件 (Events)_
    - `OnMessageReceived` (收到消息)
    - `OnLoadFinished` (加载完成)

### 路径 (Path)

- _属性 (Attributes)_
    - `waypoints` (途经点集 / Waypoints)
    - `pathStyle` (路径样式 / Path Style)
    - `flowParams` (流动参数 / Flow Parameters)
    - `agentConfig` (移动角色配置 / Agent Config)
        - `modelRef` (模型引用), `clampToGround` (是否贴地)
    - `speed` (移动速度 / Speed)
- _操作 (Operations)_
    - `playPatrol()` (播放 / Play Patrol)
    - `stopPatrol()` (停止 / Stop Patrol)
    - `pausePatrol()` (暂停 / Pause Patrol)
    - `seek(progress)` (设置进度 / Seek)
- _事件 (Events)_
    - `OnPatrolStart` (巡逻开始)
    - `OnWaypointReached` (到达途经点)
    - `OnPatrolFinish` (巡逻结束)

### 动画 (Animation)

- _类型 (Types)_
    - 序列帧 (Sequence), 骨骼 (Skeletal)
- _操作 (Operations)_
    - `play(clipName)` (播放 / Play)
    - `stop()` (停止 / Stop)
- _事件 (Events)_
    - `OnLoop` (循环)
    - `OnComplete` (完成)

## 设计要点
1. **JSON 序列化标准：** 前后端交互时，所有属性名（Key）严格使用 **小驼峰 (camelCase)**，例如 `device_id` (下划线) 或 `DeviceID` (帕斯卡) 都不推荐，统一使用 `deviceId`。
2. **枚举值映射：** 对于 `RenderMode` 或 `MapStyle` 这类属性，建议在文档中附带 **整数映射表**（例如 `0: Default`, `1: Stylized`），以减少网络传输的数据量，但代码中应使用语义化的字符串常量。
3. **多语言字段：** 如果系统支持国际化，`name` 属性可以设计为对象结构：`name: { "zh-CN": "空调", "en-US": "HVAC" }`，而不是单纯的字符串。
