## 管线生成工具
使用样条线快速生成管线，管线具有可调整的材质参数，以区分不同类型的管线
使用特效材质/粒子效果实现对流体的可视化。
![[321920896793fcf14733cba8d0df75fd_MD5.png]]

- 数据驱动特效动效系统

重点设备具备动效/特效以显示设备运行状态，在三维系统中可非常直观的查看设备启停、告警、故障等状态信息。
- 由数据驱动的特效事件
特效/动效事件均由后台接口获取控制
- 泵机类：振动幅度反映实际运行工况
- 多态报警系统

|   |   |   |
|---|---|---|
|状态等级|触发条件|特效组合|
|一级|参数超阈值80%|黄色脉冲+声光报警|
|二级|参数超阈值100%|红色粒子爆炸+三维预警标识|
|三级|设备持续异常|空间标记+关联设备连锁闪烁|
