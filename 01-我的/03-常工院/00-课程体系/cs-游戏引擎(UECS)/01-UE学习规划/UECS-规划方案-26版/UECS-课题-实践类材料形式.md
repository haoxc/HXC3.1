## 形式对比

| **形式**     | **优点**          | **致命缺陷**            |
| ---------- | --------------- | ------------------- |
| 当前这份文字方案   | 逻辑完整(`指南`)      | 学生不会读，打开就关          |
| PPT课件      | 已有体系            | 投影看得到，但手边没有，边看边做要切屏 |
| 视频教程       | 最直观             | 制作成本高，学生会直接跳到结尾     |
| Word/PDF文档 | 可打印             | 同第一条，文字太多           |
| **单页任务卡**  | 一张纸全部信息，手边放着对照做 | 需要重新设计排版            |
|            |                 |                     |

---

## 结论
**最适合这类学生的是：可打印的单页「任务卡」**
原因只有三个：
1. **物理存在感** — 打印出来放桌上，不用切屏，边看边操作
2. **信息密度可控** — 一张A4，扫一眼知道自己在哪一步，不会迷失
3. **格式像「游戏攻略」** — 艺术生熟悉这种视觉语言，不排斥
