## 术语清单
- 课程体系(Curriculum)
	- **教学大纲(Curriculum Outline):**
		- **释义**：比 Course Outline 正式度稍高，偏「课程体系下的单课大纲」，教研报告、课程申报中更常用。
		- **[[课程大纲（Syllabus）]](Syllabus/Course Outline):** 教学内容的框架梳理,适合本科 / 高职课程大纲
			- **[[教案（Lesson Plan）]]**（单课时教学设计,*Lesson Plan*）: 教案（单课时教学设计）

| 术语              | 核心指代                | 适用场景             | 绝对不适用          |
| --------------- | ------------------- | ---------------- | -------------- |
| Syllabus        | 大学**单门课**教学大纲       | 国内外高校通用、所有场景     | 专业整体培养方案、中小学   |
| Course Outline  | 大学单门课框架式教学大纲        | 美式高校、日常教研        | 教育专业正式论文、英式模块课 |
| Module Handbook | 英式大学模块课大纲手册         | 英 / 澳 / 新高校模块化课程 | 美式高校、单门课简易大纲   |
| Curriculum Plan | 大学**专业**教学大纲 / 培养方案 | 高校专业建设、院部教研      | 单门课程的教学大纲      |
| **Lesson Plan** | 教案（单课时教学设计）         | 所有学段单课时备课        | 大学课程 / 专业的教学大纲 |
| **Curriculum**  | 课程体系（整体）            | 教育理论 / 专业建设      | 单独的课程 / 专业教学大纲 |
