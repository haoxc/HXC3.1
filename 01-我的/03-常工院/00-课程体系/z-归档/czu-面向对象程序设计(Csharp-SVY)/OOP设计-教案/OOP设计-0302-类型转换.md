## 概要
在C#中，类型转换是将一个数据类型的`值转换`为另一个数据类型的过程。类型转换通常用于处理不同数据类型之间的值，并确保它们可以正确进行计算或比较。C#支持两种主要类型的类型转换：

## 内容
### 隐式类型转换
**隐式类型转换**（Implicit Type Conversion）：这是自动的类型转换，编译器可以自动将一个数据类型的值转换为另一个数据类型，而无需显式操作符。通常是从较小数据类型向较大数据类型的转换，以防止数据丢失。

```csharp
int numInt = 42;
double numDouble = numInt; // 隐式转换，int转换为double
```


>[!iimportant]  要点
> 从小类型到大类型自动转换
  >  `char` -> `short`-> `int` -> `long` -> `float` -> `double`  
### 显式类型转换
**显式类型转换**（Explicit Type Conversion）：这是需要使用显式操作符来执行的类型转换，通常是从较大数据类型向较小数据类型的转换。在执行显式类型转换时，可能会发生数据损失。

- **强制类型转换（Casting）**：使用显式强制类型转换操作符来进行类型转换。例如，将 `double` 转换为 `int` 可以使用 `(int)` 操作符.

```csharp
double numDouble = 42.5;
int numInt = (int)numDouble; // 显式强制类型转换，double转换为int
```

- **Convert 方法**：C#提供了一个 `Convert` 类，它包含了许多用于类型转换的方法。例如，使用 `Convert.ToInt32` 方法将值转换为整数。
```csharp
double numDouble = 42.5;
int numInt = Convert.ToInt32(numDouble); // 使用Convert方法将double转换为int
```
在进行类型转换时，要注意避免数据溢出或数据损失，特别是在执行显式类型转换时。强制类型转换和 `Convert` 方法是确保类型转换执行正确且安全的方式，因此在进行数据类型转换时，应格外小心。如果可能导致数据丢失或错误，应该首先检查数据是否有效。
>[!iimportant]  要点
> 从大类型到小类型需要强制转换
  >  `double` -> `float` -> `long` -> `int` ->`short`-> `char`
### 类型对象转换

#### 字符串转换为数值型
字符串`string`→ `int`, `float`, `double`等数值类型, 你可以使用 
- **int.Parse()**
- **double.Parse()**
- ...

**示例**: 我们尝试将用户的输入转换为一个整数

```csharp
	string ageValue = Console.ReadLine();
	int age = int.Parse(ageValue);
	Console.WriteLine("------------------------");
	Console.WriteLine($"收到,你的年龄是:{age}");
```

> - 如果输入`非数字`,观察程序结果
> - 如果输入`小数的数字`,观察程序结果
