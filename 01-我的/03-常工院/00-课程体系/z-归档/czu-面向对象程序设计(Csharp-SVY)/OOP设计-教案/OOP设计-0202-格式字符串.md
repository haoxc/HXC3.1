## 占位符格式化

###  占位符格式化

这是一个`字符串`，其中包含文本和格式说明符。格式说明符用于定义如何显示参数（arg0）的值。格式说明符通常


**格式项的文法**如下:
```csharp
{索引, 项的宽度及右对齐或左对齐: 项的格式字段 }
```

- `格式字段`: 要求如下:![[Pasted image 20231002140549.png]]


示例如下: 
```csharp
string name = "Alice";
int age = 30;

// 使用单个参数
Console.WriteLine(
"Hello, {0}! You are {1} years old.",
name,
age);

// 使用多个参数
float price = 9.85f;
float  quantity = 3.75f;
float total =  price* quantity;
Console.WriteLine("价格: {0,10:C2} 元/kg", price);
```

### 格式说明符
| 格式说明符 | 属性      | 说明                 |
| ----- | ------- | ------------------ |
| C,c   | 货币      |                    |
| D,d   | 十进制     | 精度说明符: 输出字符串的最少位数  |
| F,f   | 浮点数(定数) | 精度说明符: 小数的位数       |
| G,g   | 定点      | 精度说明符: 根据值         |
| X,x   | 十六进制    |                    |
| N,n   | 数字      | 和定数相似,输出是有逗号或空格分隔符 |
| P,p   | 百分比     | 精度说明符: 小数的位数       |
| E,e   | 科学记数法   | 精度说明符: 小数的位数       |

> [!warning] 注意
>  - **要点**
> 	 - 其中格式项的编号(`索引,index`)从`0`开始

## 插值格式化
可以使用字符串插值的功能，这是一种更简洁、更可读的创建格式化字符串的方法。

```csharp
int age = 25;
string name = "John";
string result = $"大家好, 我叫{name}, 今年 {age}岁, 今后有活动请多多带带我哟.";
Console.WriteLine(result);
```
- **要点**
	- 通过在字符串前面加上`$`符号，您可以直接在字符串内部使用变量。
	- 变量`name`和`age`直接嵌入到了以`$`开始的字符串中。
	- 在执行时，花括号内的内容会被替换为变量的实际值。
