## Python命令 概述
> Python有版本信息: 我们用的`版本3`
-  **安装模块**
	- pip install 模块名
- **卸载模块**
	- pip uninstall 模块名
- 显示模块
	- pip show 模块明
- **升级模块**
	- pip install xxx --upgrade
- **列表模块**
	- pip list
- **启动编程环境**
	- Jupyter notebook
	- Jupyter lab
- `补充知识 `
	- 计算机交互模式(Interaction Mode)
		- CLI (Command Language Interface)
		- GUI (Graphical User Interface)
