

## 摘要
本练习设计一个**颜色管理模块**（`color_tools.py`），包含`Color`类（单颜色操作）和`ColorPalette`类（调色板管理），实现RGB颜色转换、调色板添加/查看功能，是模块与类的入门级练习。

## 核心要点（考查知识点）
- 要点1：**自定义模块**（把类写在独立`.py`文件，主程序导入）
- 要点2：**类的基础定义**（属性初始化、简单方法）
- 要点3：**类的实例化**（创建颜色/调色板对象）
- 要点4：**简单数据处理**（RGB转十六进制、列表管理调色板）

## 试题内容
### 题目要求
#### 1. 模块设计：创建`color_tools.py`
包含两个简单类：
- **`Color`类**：处理单个RGB颜色
	  - 初始化参数：r（红，0-255）、g（绿，0-255）、b（蓝，0-255）
	  - 实例方法：
	    - `to_hex()`：把RGB转为十六进制颜色码（如(255,0,0)→"#FF0000"）
	    - `adjust_brightness(factor)`：调整亮度（factor>1变亮，<1变暗）
- **`ColorPalette`类**：管理调色板
	  - 初始化属性：调色板名称（`name`）、颜色列表（`colors = []`）
	  - 实例方法：
	    - `add_color(color)`：添加`Color`实例到调色板
	    - `show_palette()`：打印调色板所有颜色的RGB和十六进制码

#### 2. 主程序测试：创建`main.py`
导入`color_tools`模块，完成：
- 创建3个颜色实例（如红色、蓝色、黄色）
- 创建调色板实例（如“海报主色调”）
- 添加颜色到调色板并查看
- 调整某颜色亮度并打印结果
### 实现提示
1. RGB转十六进制：用`hex()`函数，补零到两位（如`hex(255)`→"ff"）
2. 调整亮度：RGB值乘以factor，超过255则取255，低于0取0
3. 模块中无需复杂异常处理，保持代码简单

## 参考代码
### 1. `color_tools.py`模块文件
```python
class Color:
    def __init__(self, r, g, b):
        # 确保RGB值在0-255之间
        self.r = max(0, min(255, r))
        self.g = max(0, min(255, g))
        self.b = max(0, min(255, b))

    def to_hex(self):
        """RGB转十六进制颜色码（如#FF0000）"""
        return f"#{self.r:02X}{self.g:02X}{self.b:02X}"

    def adjust_brightness(self, factor):
        """调整亮度，返回新的Color实例"""
        new_r = int(self.r * factor)
        new_g = int(self.g * factor)
        new_b = int(self.b * factor)
        return Color(new_r, new_g, new_b)

class ColorPalette:
    def __init__(self, name):
        self.name = name
        self.colors = []  # 存储Color实例

    def add_color(self, color):
        """添加Color实例到调色板"""
        if isinstance(color, Color):
            self.colors.append(color)
            print(f"已添加颜色：{color.to_hex()}")
        else:
            print("只能添加Color类的实例！")

    def show_palette(self):
        """显示调色板所有颜色信息"""
        print(f"\n=== 调色板：{self.name} ===")
        for i, color in enumerate(self.colors):
            print(f"颜色{i+1}：RGB({color.r},{color.g},{color.b}) → {color.to_hex()}")
```

### 2. `main.py`主程序文件
```python
from color_tools import Color, ColorPalette

# 1. 创建颜色实例
red = Color(255, 0, 0)       # 红色
blue = Color(0, 0, 255)      # 蓝色
yellow = Color(255, 255, 0)  # 黄色

# 2. 查看颜色的十六进制码
print("红色十六进制：", red.to_hex())
print("蓝色十六进制：", blue.to_hex())

# 3. 调整黄色亮度（变亮1.2倍）
bright_yellow = yellow.adjust_brightness(1.2)
print("调亮后的黄色：RGB({0},{1},{2})".format(bright_yellow.r, bright_yellow.g, bright_yellow.b))

# 4. 创建调色板并添加颜色
palette = ColorPalette("海报主色调")
palette.add_color(red)
palette.add_color(blue)
palette.add_color(bright_yellow)

# 5. 显示调色板
palette.show_palette()
```
