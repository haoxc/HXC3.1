## 题目

- **练习**：模拟调整图片对比度，初始对比度为 50，每次增加 10，直到对比度≥100 时停止，打印每次调整后的对比度。

---
## 提示
- 初始值`contrast = 50`
- 条件`while contrast < 100:`
- 循环内`contrast += 10`并打印
---
## 程序
```python
# 遍历0-19的帧编号（共20帧）
for frame_num in range(20):
    # 格式化路径：文件夹+文件名（编号补零至3位）
    frame_path = f"animation/frame_{frame_num:03d}.png" 
    
    #...

    # 打印完整路径
    print(frame_path)
```
