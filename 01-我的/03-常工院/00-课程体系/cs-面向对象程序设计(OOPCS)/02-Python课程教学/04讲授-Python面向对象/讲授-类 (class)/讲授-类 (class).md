## 概念
**类（Class）** 是对**具有相同属性（数据）和行为（方法）的对象的抽象描述**，可以理解为 “创建对象的模板”。它定义了一类事物共同的特征和能做的事情，而通过类创建的具体实例则称为 “对象（Object）”。

### 类的核心作用
类的本质是**封装 “数据” 和 “操作数据的逻辑”**，将相关的信息和功能打包在一起，形成一个独立的 “逻辑单元”。例如：
- “汽车” 类可以包含属性（颜色、排量、速度）和方法（启动、加速、刹车）；
- “学生” 类可以包含属性（姓名、年龄、成绩）和方法（上课、考试、做作业）。
---
### 类的基本构成
1. **属性（Attributes）**：
    描述类的 “数据特征”，相当于变量。例如 “学生” 类的`name`（姓名）、`age`（年龄）。
2. **方法（Methods）**：
    描述类的 “行为 / 功能”，相当于函数，用于操作属性或实现特定逻辑。例如 “学生” 类的`study()`（学习）、`take_exam()`（考试）。

---
### 类与对象的关系
- **类是抽象的 “模板”**： 比如 “手机” 类是对所有手机的抽象，不特指某一部手机。
- **对象是类的具体 “实例”**：比如 “我的 iPhone 15” 是 “手机” 类的一个具体对象，有具体的颜色（黑色）、内存（256G）等属性。

> [!NOTE] 要点
    >  简单说：**类 → 定义 “是什么” 和 “能做什么”；对象 → 基于类创建的 “具体个体”**。

---
### 定义类
``` python
class 类名:
    def __init__(self, 参数1, 参数2, ...):
        # 初始化属性
        self.属性1 = 参数1
        self.属性2 = 参数2
    
    def 方法名(self, 其他参数):
        # 定义行为
        ...
```
- `构造函数 (__init__)`: **构造方法**，创建对象时*自动调用*, 目的对新建的对象属性进行初始赋值 (也就是, 初始化);
- `self` 代表**当前对象本身**，必须作为第一个参数;通过 `self.属性名` 才能将参数值绑定到对象上（否则属性不属于对象）

### 创建对象
通过 “类名 (参数)” 创建对象，参数对应构造方法中的属性。
```python
# 用Student类创建两个具体对象（学生）
stu1 = Student("张三", 18)  # 第一个学生：张三，18岁
stu2 = Student("李四", 17)  # 第二个学生：李四，17岁
```

---
### 调用对象的属性和方法
通过 “对象。属性” 访问数据，“对象。方法 ()” 调用行为。
```python
# 访问属性
print(stu1.name)  # 输出：张三
print(stu2.age)   # 输出：17

# 调用方法
stu1.study(3)  # 输出：张三学习了3小时，成绩变为15
stu2.study(2)  # 输出：李四学习了2小时，成绩变为10

stu1.take_exam()  # 输出：张三的考试成绩是15分
stu2.take_exam()  # 输出：李四的考试成绩是10分
```

### 类的意义
1. **代码复用**：定义一次类，可创建多个对象，避免重复编写相同逻辑（比如多个学生共享 “学习”“考试” 的逻辑）。
2. **逻辑清晰**：将相关数据和功能封装在类中，使代码结构更接近现实世界的逻辑（如 “学生” 类只处理学生相关的操作）。
3. **可扩展性**：基于类的继承、多态等特性，可快速扩展新功能（如从 “学生” 类派生出 “大学生”“研究生” 子类，添加新属性和方法）。

>  💡类是面向对象编程的 “基石”，它将数据和操作数据的逻辑绑定在一起，让程序更易理解、维护和扩展。
