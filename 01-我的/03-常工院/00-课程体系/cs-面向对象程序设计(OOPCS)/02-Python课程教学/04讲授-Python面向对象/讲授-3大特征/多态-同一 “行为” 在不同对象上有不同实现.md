### 多态：同一 “行为” 在不同对象上有不同实现

## 描述
**核心逻辑**：不同对象对同一 “**指令**”（方法名）可以有不同的响应，让代码更灵活。
- 对 “**导出**” 这个指令：
    - 图片对象响应为 “导出为 PNG/JPG”；
    - 视频对象响应为 “导出为 MP4/AVI”；
    - 音频对象响应为 “导出为 MP3/WAV”。
同样是 “导出”，不同对象有不同实现，但调用方式一致（比如都用 `export()` 方法）。

>  💡  如何识别对象的行为 , 在自然语言表达中,**动词**指代的含义.

上面示例, **代码体现**：

```python
class Image:
	def export(self):
		print("导出为PNG图片")

class Video:
	def export(self):
		print("导出为MP4视频")

# 统一调用“export”，不同对象自动执行对应逻辑
def batch_export(materials):
	for material in materials:
		material.export()  # 多态：同一方法名，不同实现
# 测试
batch_export([Image(), Video()])
# 输出：
# 导出为PNG图片
# 导出为MP4视频
```
