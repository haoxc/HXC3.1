# Python面向对象程序设计 复习知识点目录（标注重点）
## 一、基础语法（★★ 必考基础）
1. 标识符命名规范（合法/非法判断，如`_student`合法、`1_user`非法）
2. 变量声明与初始化（无类型声明，直接赋值，如`age = 20`）
3. 运算符分类与用法
   - 赋值运算符（`=`、`+=`等）
   - 算术运算符（`+`、`-`、`*`、`/`、`%`）
   - 比较运算符（`==`、`!=`、`>=`、`<=`）
   - 逻辑运算符（`and`、`or`、`not`，区别于`&&`/`||`）
4. 注释规范（单行/多行注释，代码可读性）

## 二、数据类型与操作（★★★ 高频考点）
1. 基本数据类型
   - 数值型（int/float，类型转换如`int(3.14)`）
   - 字符串（str，核心考点）
   - 布尔型（bool，逻辑判断基础）
2. 字符串操作
   - 字符串插值（f-string语法，如`f"年龄：{age}"`）
   - 切片（`msg[2:6]`，左闭右开规则）
   - 常用方法（`strip()`、`replace()`、`+`拼接、`*`重复）
   - 转义字符（`\\`处理路径，如`"c:\\test\\file.txt"`）
3. 列表（list）
   - 创建方式（直接赋值`[]`、`list(range())`、`append()`添加）
   - 元素访问（索引`list1[3]`，无负数索引陷阱）
4. 数据类型转换（int↔str↔float，如`str(15) + "20"`）

## 三、控制流（★★★ 必考核心）
1. if条件语句
   - 语法格式（无括号，缩进区分代码块）
   - 完整形式（if→elif→else）
   - 单行写法（`if x>5: print("xxx")`）
   - 条件表达式（必须返回布尔值）
2. 循环结构
   - for/while循环（支持break关键字）
   - 循环控制关键字（`break`退出循环、`continue`跳过后续代码）
3. 输入输出
   - 控制台输入（`input()`，注意类型转换）
   - 格式化输出（f-string、`format()`函数）

## 四、异常处理（★★ 实战必备）
1. try-except捕获异常（处理输入非整数、无效值等场景）
2. 异常类型判断（如`ValueError`处理类型转换错误）
3. 结合循环实现输入验证（重复输入直到合法）

## 五、函数（★★ 基础核心）
1. 函数定义（`def`关键字，无返回值用`pass`或`return None`）
2. 函数调用（直接写函数名+参数，如`calc_sum(10,20)`）
3. 返回值（`return`语句，无返回值声明为`None`）
4. 函数参数（位置参数传递，如`func(score1, score2, score3)`）

## 六、面向对象编程（★★★★ 核心重点，占分比高）
1. 核心特征（必考）
   - 封装（属性私有化、通过方法访问）
   - 继承（子类继承父类属性/方法，后续进阶考点）
   - 多态（后续进阶考点，基础阶段侧重前两个）
2. 类与对象
   - 类的定义（`class`关键字）
   - 构造方法（`__init__(self, 参数)`，初始化属性核心）
   - 对象实例化（`obj = 类名(参数)`）
3. 类的方法
   - 普通方法（如`get_info(self)`）
   - 属性方法（返回对象状态，如成绩合格状态`get_status()`）
4. 实例属性（如学生类的`name`、`student_id`，作品类的`work_name`）

## 七、实战应用与计算（★★★ 编程题核心）
1. 数值计算
   - 比例计算
   - 平均分计算（去掉最高分/最低分，保留指定小数位）
   - 小数处理（`round()`函数，保留1/2位小数）
2. 逻辑判断场景
   - 条件组合判断（如`a < b and b%2 == 0`）
   - 多分支评级（如平均分对应优秀/良好/合格/不合格）
3. 输入验证（循环+异常处理，确保输入为正整数）

## 八、高频易错点（★★ 避坑重点）
1. 字符串拼接与数值运算的区别（如`2 + 3 + "5"`结果为"55"）
2. 循环关键字混用（`break`退出循环 vs `continue`跳过本次）
3. 逻辑表达式短路求值（`and`前为False则不执行后续，`or`前为True则不执行后续）
4. 异常处理的适用场景（输入验证、文件操作等）
