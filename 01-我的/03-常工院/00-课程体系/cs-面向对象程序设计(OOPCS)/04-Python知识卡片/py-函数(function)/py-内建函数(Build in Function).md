## Python 内置函数
下表按**功能类别**分类，涵盖函数名、核心功能、实用示例及版本差异说明， 


| 函数类别         | 函数名             | 功能说明                                                                  | 示例（Python 3 适用）                                                                                                                 | 备注（版本差异 / 注意事项）                                                                                   |
| ------------ | --------------- | --------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------- |
| **数值操作类**    | abs()           | 返回数字的绝对值                                                              | abs(-5) → 5                                                                                                                     | 支持整数、浮点数                                                                                          |
|              | divmod()        | 接收两个数字，返回 “商 + 余数” 组成的元组（(商, 余数)）                                     | divmod(7, 3) → (2, 1)                                                                                                           | 结果中 “商” 为地板除法（向下取整）                                                                               |
|              | pow()           | 计算幂运算，支持两参数（x^y）或三参数（(x^y) % z，即幂运算后取模）                               | pow(2, 3) → 8；pow(2, 3, 5) → 3                                                                                                  | 三参数格式效率高于 (x**y) % z                                                                              |
|              | round()         | 对数字四舍五入，可指定保留小数位数（默认保留整数）                                             | round(3.1415, 2) → 3.14；round(2.5) → 2                                                                                          | 注意：对 n.5 会向最接近的偶数取整（非单纯四舍五入）                                                                      |
|              | sum()           | 计算可迭代对象（列表、元组等）中所有元素的和，支持指定初始值（默认 0）                                  | sum([1,2,3]) → 6；sum([1,2], 10) → 13                                                                                            | 元素需为数值类型（整数、浮点数等）                                                                                 |
|              | max()           | 返回可迭代对象中的最大值，或多个参数中的最大值                                               | max([1,3,2]) → 3；max(5,1,4) → 5                                                                                                 | 支持指定 key 参数自定义比较规则（如 max(strs, key=len)）                                                          |
|              | min()           | 返回可迭代对象中的最小值，或多个参数中的最小值                                               | min([1,3,2]) → 1；min(5,1,4) → 1                                                                                                 | 同 max()，支持 key 参数                                                                                 |
|              | format()        | 格式化数据（数字、字符串等），支持自定义格式符（如千分位、小数位）                                     | format(1234, ",") → "1,234"；format(3.14, ".1f") → "3.1"                                                                         | 也可用于字符串格式化（如 "{:>5}".format("abc") → " abc"）                                                      |
| **类型转换类**    | int()           | 将对象转为整数，支持指定进制（默认 10）                                                 | int("123") → 123；int(3.9) → 3；int("101", 2) → 5                                                                                 | 浮点数转整数时直接截断小数（非四舍五入）                                                                              |
|              | str()           | 将对象转为字符串，适用于所有数据类型                                                    | str(123) → "123"；str(True) → "True"；str([1,2]) → "[1,2]"                                                                        | 转换后保留对象的可读性表示                                                                                     |
|              | bool()          | 将对象转为布尔值，“空 / 零 / None” 类对象转为 False，其余转为 True                         | bool("") → False；bool([1]) → True；bool(0) → False                                                                               | 常用作条件判断的简化（如 if bool(list): 可简写为 if list:）                                                        |
|              | float()         | 将对象转为浮点数                                                              | float("3.14") → 3.14；float(5) → 5.0                                                                                             | 字符串需为合法浮点数格式（如 "3.14" 有效，"3a" 无效）                                                                 |
|              | complex()       | 创建复数（形式为 实部+虚部j），支持传入两个参数（实部、虚部）或字符串                                  | complex(2, 3) → (2+3j)；complex("1+2j") → (1+2j)                                                                                 | 复数的实部和虚部均为浮点数                                                                                     |
|              | list()          | 将可迭代对象（元组、集合、字符串等）转为列表                                                | list((1,2,3)) → [1,2,3]；list("abc") → ["a","b","c"]                                                                             | 列表为可变序列（可修改元素）                                                                                    |
|              | tuple()         | 将可迭代对象转为元组                                                            | tuple([1,2,3]) → (1,2,3)；tuple("abc") → ("a","b","c")                                                                           | 元组为不可变序列（不可修改元素）                                                                                  |
|              | dict()          | 创建字典，支持传入键值对参数、可迭代键值对序列或映射对象                                          | dict(a=1, b=2) → {"a":1, "b":2}；dict([("a",1), ("b",2)]) → {"a":1, "b":2}                                                       | 字典的键需为不可变类型（如整数、字符串、元组）                                                                           |
|              | set()           | 创建可变集合（自动去重，无序），接收可迭代对象                                               | set([1,2,2]) → {1,2}；set("aab") → {"a","b"}                                                                                     | 支持集合运算（如交集 &、并集 `                                                                                 |
|              | frozenset()     | 创建不可变集合（无法添加 / 删除元素），接收可迭代对象                                          | frozenset([1,2]) → frozenset({1,2})                                                                                             | 可作为字典的键（普通 set 不可）                                                                                |
|              | bytearray()     | 创建字节数组（可变二进制序列），支持传入整数、字符串（需指定编码）或可迭代对象                               | bytearray("abc", "utf-8") → bytearray(b'abc')；bytearray([97,98]) → bytearray(b'ab')                                             | 用于二进制数据的修改操作（普通 bytes 不可变）                                                                        |
|              | bin()           | 将整数转为二进制字符串（前缀为 0b）                                                   | bin(5) → "0b101"；bin(0) → "0b0"                                                                                                 | 仅支持整数，非整数需先转 int（如 bin(int(3.9)) → "0b11"）                                                        |
|              | hex()           | 将整数转为十六进制字符串（前缀为 0x）                                                  | hex(15) → "0xf"；hex(255) → "0xff"                                                                                               | 结果字母默认小写，可通过 upper() 转大写（如 hex(15).upper() → "0XF"）                                               |
|              | oct()           | 将整数转为八进制字符串（前缀为 0o）                                                   | oct(8) → "0o10"；oct(7) → "0o7"                                                                                                  | Python 2 中前缀为 0，Python 3 统一为 0o                                                                   |
|              | chr()           | 将 Unicode 编码（整数）转为对应字符                                                | chr(65) → "A"；chr(122) → "z"；chr(128512) → "😂"                                                                                 | Python 3 中覆盖所有 Unicode 字符，Python 2 中仅支持 ASCII（需用 unichr()）                                        |
|              | ord()           | 将单个字符转为对应的 Unicode 编码（整数）                                             | ord("A") → 65；ord("😂") → 128512                                                                                                | 仅支持单个字符，多字符字符串会报错                                                                                 |
| **逻辑与判断类**   | all()           | 接收可迭代对象，所有元素为 True 时返回 True，否则 False（空对象返回 True）                      | all([True, 1, "a"]) → True；all([True, 0]) → False                                                                               | 短路判断：遇到 False 立即返回                                                                                |
|              | any()           | 接收可迭代对象，任意元素为 True 时返回 True，否则 False（空对象返回 False）                     | any([False, 0, "a"]) → True；any([False, 0]) → False                                                                             | 短路判断：遇到 True 立即返回                                                                                 |
|              | isinstance()    | 判断对象是否为指定类（或类元组）的实例，语法：isinstance(obj, class)                         | isinstance(5, int) → True；isinstance("a", (str, int)) → True                                                                    | 推荐用于类型判断（优于 type()，支持继承关系）                                                                        |
|              | issubclass()    | 判断类 A 是否为类 B（或类元组）的子类，语法：issubclass(A, B)                             | issubclass(bool, int) → True；issubclass(list, (tuple, list)) → True                                                             | 类与自身比较时返回 True                                                                                    |
|              | callable()      | 判断对象是否可调用（如函数、类、实现 __call__ 方法的对象）                                    | callable(print) → True；callable(5) → False；callable(int) → True                                                                 | 类是可调用的（调用后返回实例）                                                                                   |
|              | hasattr()       | 判断对象是否具有指定属性（或方法），语法：hasattr(obj, attr_name)                          | hasattr([1,2], "append") → True；hasattr(5, "len") → False                                                                       | 属性名需传入字符串（如 "append" 而非 append）                                                                   |
| **输入输出与文件类** | input()         | 接收用户输入，返回字符串（Python 3 中整合了 Python 2 的 raw_input()）                    | name = input("请输入姓名：") → 输入“张三”后，name 为 "张三"                                                                                    | Python 2 中 input() 会执行输入的表达式（如输入 “1+1” 返回 2），需注意兼容                                                |
|              | print()         | 输出内容到控制台，支持多参数、分隔符（sep）、结束符（end）等参数                                   | print("a", "b", sep="-", end="!") → 输出“a-b!”                                                                                    | Python 2 中为语句（如 print "hello"），Python 3 中改为函数                                                     |
|              | open()          | 打开文件并返回文件对象，支持指定打开模式（如读 r、写 w、追加 a）                                   | f = open("test.txt", "r", encoding="utf-8")                                                                                     | 推荐用 with 语句管理（自动关闭文件：with open(...) as f:）                                                        |
|              | file()          | Python 2 中用于创建文件对象，功能与 open() 一致                                      | -                                                                                                                               | **Python 3 已移除**，统一用 open()                                                                       |
|              | raw_input()     | Python 2 中接收用户输入，返回字符串（同 Python 3 的 input()）                          | -                                                                                                                               | **Python 3 已移除**，直接用 input() 替代                                                                   |
| **迭代与序列处理类** | enumerate()     | 为可迭代对象添加索引，返回 “(索引，元素)” 元组的迭代器，支持指定起始索引（默认 0）                         | list(enumerate(["a","b"])) → [(0,"a"), (1,"b")]；list(enumerate(["a","b"], 1)) → [(1,"a"), (2,"b")]                              | 常用于循环中同时获取索引和元素（for i, v in enumerate(...)）                                                       |
|              | iter()          | 将可迭代对象转为迭代器（支持 next() 方法获取下一个元素）                                      | it = iter([1,2]); next(it) → 1; next(it) → 2                                                                                    | 迭代器只能遍历一次，遍历完后再调用 next() 会抛 StopIteration                                                         |
|              | len()           | 返回对象的长度（如字符串、列表、元组、字典等）                                               | len("abc") → 3；len([1,2,3]) → 3；len({"a":1}) → 1                                                                                | 字典返回键值对数量，集合返回元素数量                                                                                |
|              | range()         | 生成整数序列的迭代器（Python 3），支持指定起始值、终止值、步长（左闭右开）                             | list(range(5)) → [0,1,2,3,4]；list(range(1,5,2)) → [1,3]                                                                         | Python 2 中 range() 返回列表，Python 3 中改为迭代器（节省内存）；Python 2 的 xrange() 对应 Python 3 的 range()           |
|              | xrange()        | Python 2 中生成整数序列的迭代器（同 Python 3 的 range()）                            | -                                                                                                                               | **Python 3 已移除**，直接用 range() 替代                                                                   |
|              | slice()         | 创建切片对象，用于序列（列表、字符串等）的切片操作，语法：slice(start, stop, step)                 | s = slice(1,4); [1,2,3,4,5][s] → [2,3,4]                                                                                        | 可复用切片规则（如多次用同一 slice 对象切片不同序列）                                                                    |
|              | sorted()        | 对可迭代对象排序，返回新列表（不修改原对象），支持 key 和 reverse 参数                            | sorted([3,1,2]) → [1,2,3]；sorted(["b","a"], reverse=True) → ["b","a"]；sorted([(1,3),(2,1)], key=lambda x: x[1]) → [(2,1),(1,3)] | 与列表方法 sort() 区别：sorted() 返回新列表，sort() 原地排序                                                        |
|              | reverse()       | 列表的方法，将列表元素反转（原地修改）                                                   | lst = [1,2,3]; lst.reverse() → lst 变为 [3,2,1]                                                                                   | 仅列表可用，其他序列（如元组、字符串）需用 reversed() 函数                                                               |
|              | zip()           | 将多个可迭代对象打包为 “(元素 1, 元素 2, ...)” 元组的迭代器，长度取最短对象                        | list(zip([1,2], ["a","b"])) → [(1,"a"), (2,"b")]；list(zip([1,2,3], ["a","b"])) → [(1,"a"), (2,"b")]                             | Python 3 中返回迭代器，Python 2 中返回列表；需解压时用 *（如 a, b = zip(*zip_list)）                                   |
| **函数与类辅助类**  | staticmethod()  | 定义类的静态方法（无需 self 或 cls 参数，可直接通过类或实例调用）                                | class A: @staticmethod def f(): print(1); A.f() → 输出 1                                                                          | 静态方法不依赖类或实例状态，常用于工具类函数                                                                            |
|              | classmethod()   | 定义类的类方法（第一个参数为 cls，代表类本身，可通过类或实例调用）                                   | class A: @classmethod def f(cls): print(cls); A.f() → 输出 <class '__main__.A'>                                                   | 常用于创建类的备选构造方法（如 cls() 创建实例）                                                                       |
|              | property()      | 将类的方法转为属性（可通过 “实例。属性” 访问，而非 “实例。方法 ()”）                               | class A: def __init__(self, x): self._x = x; @property def x(self): return self._x; a = A(5); a.x → 5                           | 支持设置属性的 getter/setter/deleter，控制属性访问权限                                                            |
|              | super()         | 调用父类（超类）的方法，常用于子类重写父类方法时保留父类逻辑                                        | class A: def f(self): print(1); class B(A): def f(self): super().f(); B().f() → 输出 1                                            | Python 3 中可直接用 super()，Python 2 中需指定 super(子类, self)                                              |
|              | **type()**      | 1. 返回对象的类型；2. 动态创建类（语法：type(类名, 父类元组, 属性字典)）                          | type(5) → <class 'int'>；A = type("A", (), {"x":1}); A.x → 1                                                                     | 类型判断推荐用 isinstance()（支持继承），type() 仅判断精确类型                                                         |
|              | __import__()    | 动态导入模块（通过模块名字符串导入，而非 import 模块名 语句）                                   | math = __import__("math"); math.sqrt(4) → 2.0                                                                                   | 常用于动态加载模块（如根据配置文件导入不同模块）；复杂场景推荐用 importlib.import_module()                                        |
| **其他常用类**    | eval()          | 执行字符串中的 Python 表达式，并返回表达式结果                                           | eval("1+2") → 3；eval('"a"+"b"') → "ab"；eval("max([1,2])") → 2                                                                   | 禁止执行不可信字符串（存在安全风险，如 eval("os.system('rm -rf /')")）                                                |
|              | execfile()      | Python 2 中执行指定文件中的 Python 代码（需传入文件路径）                                 | -                                                                                                                               | **Python 3 已移除**，可用 exec(open("file.py").read()) 替代                                               |
|              | filter()        | 用指定函数过滤可迭代对象，保留函数返回 True 的元素，返回迭代器                                    | list(filter(lambda x: x%2==0, [1,2,3,4])) → [2,4]                                                                               | Python 3 中返回迭代器，Python 2 中返回列表；推荐用列表推导式替代（如 [x for x in [1,2,3,4] if x%2==0]）                     |
|              | map()           | 用指定函数处理可迭代对象的每个元素，返回迭代器（元素为函数处理结果）                                    | list(map(lambda x: x*2, [1,2,3])) → [2,4,6]；list(map(lambda x,y: x+y, [1,2], [3,4])) → [4,6]                                    | Python 3 中返回迭代器，Python 2 中返回列表；推荐用列表推导式替代（如 [x*2 for x in [1,2,3]]）                               |
|              | reduce()        | 用指定函数累积可迭代对象的元素（从左到右），返回单个结果                                          | from functools import reduce; reduce(lambda x,y: x+y, [1,2,3,4]) → 10                                                           | Python 2 中为内置函数，Python 3 中移至 functools 模块；需先导入再使用                                                 |
|              | getattr()       | 获取对象的指定属性（或方法），语法：getattr(obj, attr_name, default)                    | getattr([1,2], "append") → <built-in method append of list object>；getattr(5, "len", "不存在") → "不存在"                             | 当属性不存在时，可指定 default 避免报错（否则抛 AttributeError）                                                      |
|              | setattr()       | 为对象设置指定属性（或方法），语法：setattr(obj, attr_name, value)                      | class A: pass; a = A(); setattr(a, "x", 5); a.x → 5                                                                             | 若属性已存在则覆盖，不存在则新增                                                                                  |
|              | delattr()       | 删除对象的指定属性，语法：delattr(obj, attr_name)                                  | class A: x=5; a = A(); delattr(a, "x"); hasattr(a, "x") → False                                                                 | 若属性不存在则抛 AttributeError，也可直接用 del obj.attr                                                        |
|              | globals()       | 返回当前全局作用域的变量字典（键为变量名，值为变量值）                                           | x=1; globals()["x"] → 1；globals() → {'__name__':'__main__', 'x':1, ...}                                                         | 可通过字典修改全局变量（如 globals()["x"] = 2 等效于 x=2）                                                         |
|              | locals()        | 返回当前局部作用域的变量字典（如函数内部的变量）                                              | def f(): x=1; print(locals()) → {'x':1}                                                                                         | 局部作用域中修改 locals() 字典，不一定能同步修改实际变量（依赖解释器实现）                                                        |
|              | [[内建函数-help()]] | 查看对象的帮助文档（如函数、类、模块的用法说明）                                              | help(print) → 输出 print() 函数的详细文档；help(list) → 输出 list 类的详细文档                                                                    | 交互式环境中常用，也可在脚本中使用（输出到控制台）                                                                         |
|              | [[内建函数-id()]]   | 返回对象的唯一标识符（内存地址的整数表示）                                                 | a=5; id(a) → 140708483456256；b=a; id(b) → 140708483456256                                                                       | 同一对象的 id() 始终相同，不同对象（即使值相同）id() 可能不同（如小整数缓存机制：id(5) == id(5) → True，id(1000) == id(1000) → False） |
|              | hash()          | 返回对象的哈希值（整数），用于字典的键存储和集合的去重（需对象不可变）                                   | hash(5) → 5；hash("abc") → 123456...（具体值依赖解释器）                                                                                   | 可变对象（如列表、字典）不可哈希（调用 hash() 抛 TypeError）；不可变对象（如整数、字符串、元组）可哈希                                      |
|              | compile()       | 将 Python 代码（字符串、文件、AST 对象）编译为字节码对象，供 exec() 或 eval() 执行               | code = compile("print(1)", "<string>", "exec"); exec(code) → 输出 1                                                               | 第三个参数 mode 需指定："exec"（执行语句块）、"eval"（执行表达式）、"single"（执行交互式语句）                                      |
|              | memoryview()    | 创建内存视图对象，用于直接访问对象的内存（无需复制数据）                                          | b = bytearray("abc", "utf-8"); mv = memoryview(b); mv[0] → 97; mv[0] = 65; b → bytearray(b'Abc')                                | 常用于大型二进制数据的高效处理（如避免数据复制）                                                                          |
|              | reload()        | 重新导入已导入的模块（更新模块内容）                                                    | -                                                                                                                               | Python 2 中为内置函数，Python 3 中移至 importlib 模块（需 from importlib import reload）；仅支持已导入的模块，未导入模块会报错      |
|              | repr()          | 返回对象的 “官方字符串表示”（可通过 eval() 还原对象，若支持）                                  | repr(123) → "123"；repr("abc") → "'abc'"；repr([1,2]) → "[1,2]"                                                                   | 与 str() 区别：str() 侧重可读性，repr() 侧重准确性（如 str("abc") → "abc"，repr("abc") → "'abc'"）                   |
|              | vars()          | 1. 无参数时，返回当前局部作用域的变量字典（同 locals()）；2. 有参数时，返回对象的 __dict__ 属性（存储对象的属性） | x=1; vars() → {'x':1, ...}；class A: x=5; a=A(); vars(a) → {}（实例无自定义属性时为空）                                                       | 若对象无 __dict__ 属性（如整数、字符串），调用 vars(obj) 会抛 TypeError                                               |
|              | object()        | Python 中所有类的基类（顶级父类），创建空对象（无属性和方法）                                    | o = object(); hasattr(o, "x") → False                                                                                           | 自定义类默认继承 object（如 class A: pass 等效于 class A(object): pass）                                        |
|              | exec            | 执行动态 Python 代码（语句块，无返回值），是关键字而非函数                                     | exec("x=1; print(x)") → 输出 1；code = compile("y=2", "<string>", "exec"); exec(code); y → 2                                       | 与 eval() 区别：exec 执行语句块（无返回），eval() 执行表达式（有返回）；Python 2 中也为关键字                                     |
