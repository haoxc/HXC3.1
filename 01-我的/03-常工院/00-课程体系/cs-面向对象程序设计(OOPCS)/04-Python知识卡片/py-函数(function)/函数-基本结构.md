### 基本结构
```python
def 函数名(参数1, 参数2=默认值):  # 定义工具名称和可调参数
    """函数说明（像动作的备注：这个工具做什么）"""
    # 函数体：具体操作步骤（调亮度、加边框等）
    处理步骤1
    处理步骤2
    return 处理结果  # 可选：返回处理好的内容（如调整后的图片）
```

---
**示例**： 定义一个 “给图片加边框” 的函数：
```python
def add_border(image, width=5, color=(255,255,255)):
    """给图片加白色边框（默认宽度5像素）
    参数：
        image：输入图片
        width：边框宽度（默认5）
        color：边框颜色（默认白色，BGR格式）
    返回：
        加了边框的图片
    """
    # 具体加边框的代码（调用OpenCV工具）
    bordered = cv2.copyMakeBorder(image, width, width, width, width, cv2.BORDER_CONSTANT, value=color)
    return bordered  # 返回处理后的图片
```
