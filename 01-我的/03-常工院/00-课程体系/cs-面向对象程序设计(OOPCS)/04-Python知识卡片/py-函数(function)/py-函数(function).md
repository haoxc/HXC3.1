## 函数概述
- **定义**：函数是封装好的，可重复使用的，用来实现单一，或相关联**功能**的代码段。
- **作用**： 函数能提高程序的**模块性**，和代码的`重复利用率`。

  
> 🎨 Python 函数
> - 你的“数字艺术创作工具”
> - **函数 = 一个可重复使用的创作动作**
> - **函数就是你的“代码笔刷”**

---
## 内容

### 基本结构
```python
def 函数名(参数1, 参数2=默认值):  # 定义工具名称和可调参数
    """函数说明（像动作的备注：这个工具做什么）"""
    # 函数体：具体操作步骤（调亮度、加边框等）
    处理步骤1
    处理步骤2
    return 处理结果  # 可选：返回处理好的内容（如调整后的图片）
```

---
**示例**： 定义一个 “给图片加边框” 的函数：
```python
def add_border(image, width=5, color=(255,255,255)):
    """给图片加白色边框（默认宽度5像素）
    参数：
        image：输入图片
        width：边框宽度（默认5）
        color：边框颜色（默认白色，BGR格式）
    返回：
        加了边框的图片
    """
    # 具体加边框的代码（调用OpenCV工具）
    bordered = cv2.copyMakeBorder(image, width, width, width, width, cv2.BORDER_CONSTANT, value=color)
    return bordered  # 返回处理后的图片
```

### 函数的 “参数”

参数是函数的 “输入选项”，让同一个函数能处理不同需求（比如同一 “加边框” 动作，可改边框宽度）。

| 参数类型  | 数媒类比           | 示例（加边框函数）                               |
| ----- | -------------- | --------------------------------------- |
| 位置参数  | 必选选项（如边框必须有宽度） | `add_border(img, 10)` → 宽度 10（按顺序传值）    |
| 关键字参数 | 可选选项（指定名称修改）   | `add_border(img, color=(0,0,0))` → 黑色边框 |
| 默认参数  | 预设选项（不调就用默认值）  | 不传 `width` 时，自动用默认值 5                   |
|       |                |                                         |
> 📌 就像 PS 动作里的 “可调选项”
