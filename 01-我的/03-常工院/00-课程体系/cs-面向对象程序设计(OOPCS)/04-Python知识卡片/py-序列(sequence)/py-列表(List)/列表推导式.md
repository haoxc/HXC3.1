## 概述
**列表推导式（List Comprehension）** 是一种简洁高效的创建列表的方式，它通过一行代码实现原本需要多行循环 + 条件判断才能完成的列表生成，既简化了代码，又保持了可读性。

---
## 基本语法
列表推导式的核心结构是：
```python
[表达式 for 变量 in 可迭代对象]
```
- **作用**：遍历 “可迭代对象”（如列表、字符串、range 等）中的每个元素，将其代入 “表达式” 计算，最终将所有结果组成一个新列表。

---
## 应用场景
####  1. 生成简单列表
用列表推导式替代传统的`for`循环创建列表，更简洁：
```python
# 传统方法：用for循环生成1-5的平方列表
squares = []
for i in range(1, 6):
    squares.append(i **2)
print(squares)  # 输出：[1, 4, 9, 16, 25]

# 列表推导式：一行生成
squares = [i** 2 for i in range(1, 6)]
print(squares)  # 输出：[1, 4, 9, 16, 25]
```
---
#### 2. 带条件筛选的列表推导式
可以在遍历过程中添加`if`条件，只保留符合条件的元素：
```python
[生成表达式 for 变量 in 可迭代对象 if 条件]
```

**示例**： 筛选出 1-10 中的偶数，并计算其平方：
```python
# 传统方法
even_squares = []
for i in range(1, 11):
    if i % 2 == 0:  # 仅保留偶数
        even_squares.append(i **2)
print(even_squares)  # 输出：[4, 16, 36, 64, 100]

# 列表推导式
even_squares = [i** 2 for i in range(1, 11) if i % 2 == 0]
print(even_squares)  # 输出：[4, 16, 36, 64, 100]
```

---
#### 3. 多条件判断
支持多个`if`条件（逻辑上为 “且” 关系），或用`if-else`实现分支逻辑：
```python
# 多if条件：筛选1-20中能同时被2和3整除的数
nums = [x for x in range(1, 21) if x % 2 == 0 if x % 3 == 0]
print(nums)  # 输出：[6, 12, 18]（同时满足被2和3整除）

# if-else分支：奇数转负数，偶数保留原值
nums = [x if x % 2 == 0 else -x for x in range(1, 6)]
print(nums)  # 输出：[-1, 2, -3, 4, -5]
```

- `for x in range(1, 6)`：遍历可迭代对象 `range(1,6)`（即取值 `1,2,3,4,5`），变量 `x` 依次代表每个值。
- `x if x % 2 == 0 else -x`：这是「元素生成规则」，是一个**条件表达式**，用于根据条件为每个 `x` 生成对应的元素。

---
#### 4. 嵌套循环的列表推导式 (高阶)
适用于处理嵌套的可迭代对象（如二维列表），语法类似嵌套`for`循环：
```python
# 传统嵌套循环：将二维列表展平为一维列表
matrix = [[1, 2], [3, 4], [5, 6]]
flat = []
for row in matrix:
    for num in row:
        flat.append(num)
print(flat)  # 输出：[1, 2, 3, 4, 5, 6]

# 列表推导式（嵌套循环）
flat = [num for row in matrix for num in row]
print(flat)  # 输出：[1, 2, 3, 4, 5, 6]
```
---
## 优势与适用场景
1. **简洁高效**：一行代码替代多行循环，减少冗余代码。
2. **可读性强**：简单逻辑下，比循环更直观（复杂逻辑建议用循环，避免可读性下降）。
3. **性能优化**：底层执行效率略高于传统`for`循环（减少了 append () 方法的多次调用）。
---
## 注意事项
- **避免过度复杂**：如果推导式包含多层循环或复杂条件，建议拆分为传统循环，否则可读性会变差。
- **与生成器的区别**：列表推导式会一次性生成整个列表并占用内存；若处理大量数据，可考虑生成器表达式（将`[]`改为`()`），按需生成元素。

>  🔑**总结**：列表推导式是 Python 的特色语法之一，适合用简洁代码生成新列表，是提高编码效率的实用工具。
