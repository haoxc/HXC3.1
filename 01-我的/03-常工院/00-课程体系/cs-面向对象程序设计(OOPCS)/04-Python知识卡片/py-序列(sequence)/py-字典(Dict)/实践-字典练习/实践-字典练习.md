

### 三、字典（Dictionary）练习

字典核心特点：**键值对（key:value）、键唯一且不可变、Python 3.7 + 有序**，适合存储关联数据（如配置、属性）。

#### 练习 1：创建字典与访问值

```python
# 1. 创建字典（用{}或dict()）
person = {
    "name": "Alice",
    "age": 25,
    "hobbies": ["reading", "hiking"],  # 值可以是列表等可变类型
    (1, 2): "tuple_key"  # 键可以是元组（不可变）
}

# 从键值对列表创建
animal = dict([("name", "Dog"), ("legs", 4)])

print("个人信息字典：", person)
print("动物字典：", animal)

# 2. 通过键访问值（若键不存在，用get()返回默认值，避免报错）
print("姓名：", person["name"])  # 直接访问（键不存在会报错）
print("性别（默认未知）：", person.get("gender", "未知"))  # get()访问
```

**预期输出**：

```plaintext
个人信息字典： {'name': 'Alice', 'age': 25, 'hobbies': ['reading', 'hiking'], (1, 2): 'tuple_key'}
动物字典： {'name': 'Dog', 'legs': 4}
姓名： Alice
性别（默认未知）： 未知
```

#### 练习 2：字典的修改、添加、删除
```python
car = {"brand": "Toyota", "color": "red", "year": 2020}

# 1. 修改值（通过键赋值）
car["year"] = 2023  # 修改已有键的值
print("修改年份后：", car)

# 2. 添加新键值对（直接赋值新键）
car["price"] = 200000  # 新增键值对
print("添加价格后：", car)

# 3. 删除键值对（del 或 pop()）
del car["color"]  # del删除
year = car.pop("year")  # pop()删除并返回值
print("删除颜色和年份后：", car, "被删除的年份：", year)
```

**预期输出**：

```plaintext
修改年份后： {'brand': 'Toyota', 'color': 'red', 'year': 2023}
添加价格后： {'brand': 'Toyota', 'color': 'red', 'year': 2023, 'price': 200000}
删除颜色和年份后： {'brand': 'Toyota', 'price': 200000} 被删除的年份： 2023
```

#### 练习 3：字典的常用方法与推导式

```python
scores = {"math": 90, "english": 85, "chinese": 95}

# 1. 获取所有键、值、键值对
print("所有科目：", scores.keys())  # 键
print("所有分数：", scores.values())  # 值
print("科目-分数对：", scores.items())  # 键值对

# 2. 字典推导式（类似列表推导式，生成新字典）
# 示例：将分数大于90的科目筛选出来，值翻倍
high_scores = {k: v*2 for k, v in scores.items() if v > 90}
print("分数>90的科目（值翻倍）：", high_scores)
```

**预期输出**：

```plaintext
所有科目： dict_keys(['math', 'english', 'chinese'])
所有分数： dict_values([90, 85, 95])
科目-分数对： dict_items([('math', 90), ('english', 85), ('chinese', 95)])
分数>90的科目（值翻倍）： {'chinese': 190}
```
