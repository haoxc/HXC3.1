## 概述
字符串格式化中，`格式说明符` 用于控制字符串的**对齐方式**、**填充字符**和**总长度**，各部分含义拆解如下：

```python
格式说明符的基本结构为：`[填充字符][对齐方式][宽度]`，中间用冒号 `:` 开头标识格式说明
```

**示例分析:**
```python
"{:-^20}".format("标题")
```
- **`-`**：填充字符
    当原始字符串长度小于指定宽度时，用于填充空白部分的字符（这里是减号 `-`）。
- **`^`**：对齐方式
    表示**居中对齐**（原始字符串放在总长度的中间位置）。
    其他对齐方式：`>` 表示右对齐，`<` 表示左对齐。
- **`20`**：总宽度
    最终输出的字符串总长度为 20 个字符（包含原始字符串和填充字符）。
