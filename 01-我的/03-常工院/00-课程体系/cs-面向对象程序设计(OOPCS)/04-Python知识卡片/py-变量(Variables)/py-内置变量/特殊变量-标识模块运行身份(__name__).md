### 一、`__name__`的核心作用

`__name__`的取值只有两种两种情况 **，取决于[[模块]]的运行方式：

1. 当模块**被直接运行**时（比如用`python 模块名.py`执行），`__name__`的值为字符串`"__main__"`**。
2. 当模块**被其他模块导入**时（比如`import 模块名`），`__name__`的值为**模块本身的文件名**（不含`.py`后缀）。


### 总结
`__name__`是 Python 标识模块运行身份的特殊变量：
- 直接运行模块 → `__name__ == "__main__"`
- 模块被导入 → `__name__ == 模块名`
通过`if __name__ == "__main__":`，可以让模块同时满足 “被导入复用” 和 “自身运行测试” 的需求，是 Python 模块化编程的重要技巧。
