## 变量（Variables）

变量是存储数据的容器，用于在程序中保存和引用值。

### 变量命名规则
- 由字母、数字和下划线组成
- 不能以数字开头
- 不能使用Python关键字
- 区分大小写
- 建议使用小写字母和下划线命名法（snake_case）

```python
# 正确的变量名
name = "张三"
age = 25
is_student = True
user_score_1 = 95.5

# 错误的变量名
2name = "错误"      # 不能以数字开头
class = "Python"    # class是关键字
```

### 变量赋值
使用等号（=）为变量赋值，Python是动态类型语言，变量类型由赋值决定。

```python
# 变量赋值
counter = 10           # 整数
price = 29.99         # 浮点数
message = "Hello"     # 字符串
is_active = True      # 布尔值
```

## 基本数据类型

### 1. 整数（int）
表示整数值，没有大小限制（受内存限制）。

```python
age = 25
score = -10
count = 0
big_number = 10_000_000  # 使用下划线提高可读性
```

### 2. 浮点数（float）
表示带小数点的数值。

```python
pi = 3.14159
price = 19.99
temperature = -5.5
scientific = 1.23e4     # 科学计数法：1.23 × 10^4 = 12300
```

### 3. 字符串（str）
表示文本数据，用单引号、双引号或三引号包围。

```python
name = "Alice"
greeting = 'Hello'
multiline = """这是一个
多行字符串"""
empty = ""              # 空字符串
```

### 4. 布尔值（bool）
表示真或假，只有两个值：`True` 和 `False`。

```python
is_raining = True
has_permission = False
```

### 5. 空值（None）
表示空值或无值，类似于其他语言中的`null`。

```python
result = None
data = None
```

## 运算符

### 1. 算术运算符
用于基本的数学运算。

```python
a = 10
b = 3

print(a + b)    # 加：13
print(a - b)    # 减：7
print(a * b)    # 乘：30
print(a / b)    # 除：3.333...
print(a // b)   # 整除：3
print(a % b)    # 取余：1
print(a ** b)   # 幂运算：1000
```

### 2. 比较运算符
用于比较两个值，返回布尔值。

```python
x = 5
y = 10

print(x == y)   # 等于：False
print(x != y)   # 不等于：True
print(x > y)    # 大于：False
print(x < y)    # 小于：True
print(x >= 5)   # 大于等于：True
print(x <= 4)   # 小于等于：False
```

### 3. 逻辑运算符
用于组合多个条件。

```python
has_ticket = True
has_money = False
is_weekend = True

# and：两个条件都为True时返回True
print(has_ticket and has_money)  # False

# or：至少一个条件为True时返回True
print(has_ticket or has_money)   # True

# not：取反
print(not is_weekend)            # False
```

### 4. 赋值运算符
用于给变量赋值。

```python
x = 10
x += 5      # x = x + 5，结果：15
x -= 3      # x = x - 3，结果：12
x *= 2      # x = x * 2，结果：24
x /= 4      # x = x / 4，结果：6.0
x //= 2     # x = x // 2，结果：3.0
x %= 2      # x = x % 2，结果：1.0
x **= 3     # x = x ** 3，结果：1.0
```

### 5. 身份运算符
检查两个对象是否为同一个对象。

```python
a = [1, 2, 3]
b = [1, 2, 3]
c = a

print(a is b)       # False，不是同一个对象
print(a is c)       # True，是同一个对象
print(a is not b)   # True，不是同一个对象
```

### 6. 成员运算符
检查序列中是否包含某个值。

```python
colors = ["red", "green", "blue"]

print("red" in colors)      # True
print("yellow" not in colors) # True
```

## 类型检查与转换

### 类型检查
使用`type()`函数检查变量类型。

```python
x = 10
y = 3.14
z = "Hello"

print(type(x))      # <class 'int'>
print(type(y))      # <class 'float'>
print(type(z))      # <class 'str'>
```

### 类型转换
使用内置函数进行类型转换。

```python
# 转换为整数
print(int(3.14))    # 3
print(int("100"))   # 100

# 转换为浮点数
print(float(10))    # 10.0
print(float("3.14")) # 3.14

# 转换为字符串
print(str(100))     # "100"
print(str(3.14))    # "3.14"

# 转换为布尔值
print(bool(0))      # False
print(bool(1))      # True
print(bool(""))     # False（空字符串）
print(bool("Hi"))   # True（非空字符串）
```

## 最佳实践

1. **变量命名要有意义**
   ```python
   # 好的命名
   student_age = 20
   user_name = "张三"
   
   # 不好的命名
   a = 20
   n = "张三"
   ```

2. **一行只定义一个变量**
   ```python
   # 好的写法
   name = "Alice"
   age = 25
   
   # 不好的写法
   name, age = "Alice", 25  # 虽然语法正确，但可读性差
   ```

3. **避免使用魔法数字**
   ```python
   # 好的写法
   MAX_USERS = 100
   if user_count > MAX_USERS:
       print("已达到用户上限")
   
   # 不好的写法
   if user_count > 100:  # 100是什么意思？
       print("已达到用户上限")
   ```

4. **及时注释类型**
   ```python
   # 好的写法
   count: int = 0      # 类型提示
   name: str = "Alice" # 类型提示
   ```

## 常见错误

1. **使用未定义的变量**
   ```python
   print(undefined_var)  # NameError: name 'undefined_var' is not defined
   ```

2. **拼写错误**
   ```python
   user_name = "Alice"
   print(userName)  # NameError: name 'userName' is not defined
   ```

3. **错误的数据类型操作**
   ```python
   number = "100"
   result = number + 50  # TypeError: can only concatenate str to str
   ```

4. **使用关键字作为变量名**
   ```python
   class = "Python101"  # SyntaxError: invalid syntax
   ```
