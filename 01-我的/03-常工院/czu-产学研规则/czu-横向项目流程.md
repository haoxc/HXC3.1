## 流程

- **产学研登记**
	- **登记产学研合同**( #项目负责人) → **院内审核**( 联系 #彭老师) →**校内审核**( `预计1-2天` ⏰ )→ **线下签字**( #闫泓霏 找张忠老师 ) 
- **科技备案**
	- **科技备案**( #项目负责人 `预计9-12天` ⏰)→ **通知打款**( #闫泓霏 联系客户)→**开具发票**( #闫泓霏 )  
- **返款合同**
	- **登记返款合同**( #项目负责人) → **院内审核**( 联系 #彭老师) →**校内审核**( `预计1-2天`  )→  **线下签字**( #闫泓霏 找张忠老师 ) 
- **返款报销**
	- **获取发票**( #闫泓霏) → **院内审核**( 有需要联系 #徐院) →   **线下报销**( #项目负责人 ) 
	- **其他报销**( 按照劳务费报销给客户)
