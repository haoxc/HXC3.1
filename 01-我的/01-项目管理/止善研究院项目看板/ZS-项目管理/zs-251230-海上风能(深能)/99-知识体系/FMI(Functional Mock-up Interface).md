# 📊 FMI

## 📋 基本信息
- **中文全称**：FMI
- **英文全称**：Functional Mock-up Interface
- **缩写/代号**：
- **所属领域**：[[系统工程]] (SystemEngine)

## 🔍 定义与内涵
> 在 系统工程 语境下，该术语的核心定义是：
>**Functional Mock-up Interface (FMI)** 是==由 Modelica 协会维护的一种**开放标准**，用于跨工具交换动态系统仿真模型。它通过定义 C 接口和 XML 架构，解决了不同供应商仿真工具（如 MATLAB, Simulink, Amesim）之间无法直接互操作的问题==。
