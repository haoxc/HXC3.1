## 内容
- **关注点**：**文本、逻辑、前因后果。**
- **视角**：**微观与线性的。** 它是为了让一段话、一个词产生特定含义而存在的背景。
- **侧重点**：侧重于“**语义的确定性**”。
- **例子**：在“我们要建立单一事实来源（SSoT）”这句话中，如果**语境**是“软件开发”，那么 SSoT 指代码库；如果**语境**是“企业管理”，它指权威数据库。

# 📌 **语境：语言的"环境密码"**

**一句话记住**：

> **语境 = 语言使用的"环境密码"**
> _（一句话没语境，可能被误解；有语境，意思才清晰）_

---

## 🔑 **核心定义**

> 语境指语言交际时所处的**具体环境**，包括：
>
> - **语言内部**：上下文、前言后语
> - **外部环境**：时间、地点、场合、文化背景、交际对象等

_例：_
"你怎么回来得这么晚呀？"
- 语境1（责备）：妈妈在等你吃饭，你10点还没回家
- 语境2（关心）：朋友在电话里问你，你刚加班结束
    **同一句话，因语境不同，意思完全不同！**

---

## 🌐 **分类体系：3大类 + 4种维度**（精简版）

### ✅ **1. 语言性语境**（内部语境）

> _语言内部的关联环境_
> - **上下文语境**：前言后语、段落关联
> _例：_ "他昨天去了北京。今天他去了上海。"（"他"指代明确）
> - **文体语境**：书面语/口语的风格差异
> _例：_ "请批示"（公文）vs "啥时候来"（聊天）

### ✅ **2. 非语言性语境**（外部语境）
_语言外部的环境因素_

| 类型         | 核心要素         | 典型场景              | 一句话记                     |     |
| ---------- | ------------ | ----------------- | ------------------------ | --- |
|  **情景语境**  | 时间、地点、场合、身份  | 正式会议 vs 朋友聊天      | "请审议" vs "行不？"           |     |
|  **文化语境**  | 文化背景、风俗习惯、禁忌 | 西方忌问年龄，中国送礼忌送钟    | "祝您生日快乐" vs "别送钟"        |     |
|  **社会语境**  | 社会背景、历史环境    | 90年代 vs 2020年代的用语 | "下岗" vs "灵活就业"           |     |
|  **认知语境**  | 说话人知识背景、心理状态 | 专业术语 vs 普通人解释     | "光速=3×10⁸m/s" vs "光跑得超快" |     |

> 💡 **记忆口诀**：**"语内情文社认"**（语言性、情景、文化、社会、认知）

---

## ✨ **语境的3大核心作用**

1. **限制作用**（限定表达）
    _例：_ "他"在不同上下文中指代不同人
2. **补充作用**（补充隐含意义）
    _例：_ "你真棒！"（语境是表扬，不是讽刺）
3. **消除歧义**（让语言更准确）
    _例：_ "他去了北京"（上下文说明是"他"指代某人）
