人类大脑对社交威胁的反应与对肉体疼痛的反应一致。管理者的言语若触碰以下五个维度，会直接导致下属进入“`防卫模式`”，关闭认知资源。

| **维度**                | **含义**      | **实践逻辑**             |
| --------------------- | ----------- | -------------------- |
| **Status (地位)**       | 相对他人的重要程度   | `给予公开肯定`，减少说教感。      |
| **Certainty (确定性)**   | 对未来的预测能力    | 明确`目标和预期`，减少信息黑盒。    |
| **Autonomy (自主性)**    | 对事件的掌控感     | 给出`选择题`而非填空题，允许自主决策。 |
| **Relatedness (归属感)** | 是否属于“`自己人`” | 建立情感连接，消除团队内的孤岛感。    |
| **Fairness (公平性)**    | 交换过程是否透明公平  | 决策`逻辑公开化`，奖惩标准一致。    |
