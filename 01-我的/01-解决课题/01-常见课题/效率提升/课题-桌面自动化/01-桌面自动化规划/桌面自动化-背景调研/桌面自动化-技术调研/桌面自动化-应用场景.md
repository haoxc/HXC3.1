## 分类
- **数据采集与录入:** 将 Excel 表单数据自动填写到网页或 ERP 系统中。
- **报表自动生成:** 定期打开多个软件汇总数据，并自动发送邮件。
- **批量文件处理:** 自动对成千上万个文件进行重命名、格式转换或分类归档。
- **环境初始化:** 一键启动开发环境、打开所需的办公软件及特定网页。

- **报销与票据录入：** 自动从邮箱下载电子发票，通过 OCR（文字识别）提取金额、日期，并自动填入公司的 ERP 系统或 Excel 报销单。
- **跨系统数据同步：** 在没有 API 接口的旧系统（如旧版财务软件）与现代云端工具（如飞书、钉钉）之间，自动搬运和校对订单、库存数据。
- **薪资与报表生成：** 每月底自动汇总多个部门的考勤表，计算工资，并生成 PDF 格式的工资条发送给每位员工。

## 建议工具选择：

- **初学者/办公族：** 首选 Microsoft Power Automate Desktop，界面友好且与 Windows 深度集成。
- **专业开发者：** 使用 PyAutoGUI 或 Playwright 进行高度自定义开发。
- **电商/业务人员：** 推荐 影刀 RPA，针对国内主流软件有大量现成的自动化模板。
