# CLAUDE.md

## 笔记风格

创建或编辑笔记时遵循。

| 要素 | 规则 |
|------|------|
| 标题 | 见名知意，不加"关于"、"如何"等冗余词 |
| 长度 | 一个屏幕看完核心，细节往下放 |
| 句式 | 陈述句，不写"我认为"、"建议" |
| 对比 | 能用表格不用段落 |
| 链接 | 提到相关概念就链，不重复解释 |
| 语气 | 冷静、精确，像给一年后的自己留纸条 |
| 中文为主 | 英文术语用括号保留，如 `智能体(Agent)` |
| 结尾 | 写完了就停，不凑数 |

## 底层逻辑

1. **检索速度 > 阅读体验** — 标题决定搜不搜得到，表格和清单比段落扫描快
2. **写给未来的陌生人** — 每个结论自带理由，不留隐式知识
3. **一棵树一张嘴** — 每个概念只在一处详细解释，其他地方用链接引用
